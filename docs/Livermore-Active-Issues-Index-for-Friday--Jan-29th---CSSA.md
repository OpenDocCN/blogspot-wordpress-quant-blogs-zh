嗨，杰兹，谢谢你——这是一个长期指数，所以 alpha 需要时间来建立，并且短期内有相当大的波动，但它被设计为非对称的，在积极市场中上涨空间更大，在消极市场中下跌空间更小。它可以作为排名系统使用，但底部的 50 个或 10 个的最常用方式实际上是一个均值回归指数，可以使用 RSI 或 DV2 等进行交易。兰德尔的一个分支是，它特别适用于趋势跟踪方法——由于该指数不包含时机，因此这是非常有益的。

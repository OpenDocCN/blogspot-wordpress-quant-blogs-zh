# Magmasystems 博客：纽约编码营

> 来源：[`magmasystems.blogspot.com/2007/02/new-york-code-camp.html#0001-01-01`](http://magmasystems.blogspot.com/2007/02/new-york-code-camp.html#0001-01-01)

[DonXml](http://www.donxml.com)

正在组织一个

[编码营](http://nyc.codecamp.us)

将于 3 月 3 日星期六在纽约市举行。去年的营地 24 小时内售罄，最多容纳 150 名参与者。今年他们将人数扩大到 400，所以应该有足够的空间。

唐正在寻找演讲者，特别是来自金融行业的。所以，所有来自高盛、摩根、JPMC、花旗、DB、Barcap、雷曼、对冲基金等的人……准备透露一些内部消息！来告诉我们你们在工作中是如何使用.NET 的。

唐还在尝试在新泽西组织一个 Java/.NET 编码营，并正在寻找一些公司捐赠场地。如果你是一个位于北/北中部，可以捐赠一个周六会议设施的组织，请联系唐。

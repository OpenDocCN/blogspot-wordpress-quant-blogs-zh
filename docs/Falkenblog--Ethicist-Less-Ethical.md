# Falkenblog: Ethicist Less Ethical

> 来源：[`falkenblog.blogspot.com/2010/08/ethicist-less-ethical.html#0001-01-01`](http://falkenblog.blogspot.com/2010/08/ethicist-less-ethical.html#0001-01-01)

在

[Bloggingheads.tv](http://bloggingheads.tv/diavlogs/30202)

上，两位哲学家苦苦思索，考虑是否思考道德能使一个人更有道德？还是只是更擅长使用道德来为自己想做的事情找到合理的理由。

[Eric Schwitzgebel](http://schwitzsplintersethicsprofs.blogspot.com/)

举一个例子，例如记录哲学中的伦理学书籍比其他领域的书籍更经常逾期。伦理学家并不比其他人更有伦理道德，只是特别擅长找到为自己想做的事情找到合理的理由。这种便利性的胡扯在经济学和金融领域并不陌生，经济学家可以通过一些已发表的经济学文献来为自己偏好的任何受欢迎的经济政策进行辩护。经济学中的重大问题并没有因为理论而在过去一个世纪变得不受争议，只是苏联的失败以及其他一些数据点使一些论点变得不太令人信服。

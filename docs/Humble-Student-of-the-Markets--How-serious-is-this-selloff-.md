# 谦逊的市场学生：这次抛售有多严重？

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/04/how-serious-is-this-selloff.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/04/how-serious-is-this-selloff.html#0001-01-01)

昨天，全球股市因荷兰和霍兰德的消息而下跌。欧洲股市正处于一个清晰的一个月下跌趋势中。

美国股市下跌至测试一个重要的技术支撑水平。

然而，当我查看外汇和债券市场时，我看不到与股票市场中所存在的相同水平的投资者焦虑。例如，如果欧洲如此糟糕，那么为什么 EURUSD 汇率保持得如此良好？

还应考虑 CADUSD 汇率，它对商品价格和风险偏好有敏感度。加拿大元相对于美元仍然在交易区间内。

债券市场也没有出现恐慌的迹象。在股市经历困难的情况下，高收益或垃圾债券的表现依然相当不错。为什么市场的风险厌恶没有显现出来？

总之，除非我们从外汇和债券市场得到风险偏好明显下降的信号，否则这次股市弱势只是市场横盘整理和波动的一个阶段。

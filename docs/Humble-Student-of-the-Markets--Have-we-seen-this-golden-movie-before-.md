# 市场的谦逊学生：我们看过这部黄金电影吗？

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/06/have-we-seen-this-golden-movie-before.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/06/have-we-seen-this-golden-movie-before.html#0001-01-01)

近几天，博客圈对黄金及其股票可能出现的转机感到兴奋。具体来说，技术分析师指出黄金的上涨（图表顶部面板），黄金股票与黄金比率（GDX/GLD，中部面板）以及高贝塔白银与黄金比率的更好表现。所有这些都指向了贵金属价格的更好时光。

**陪审团仍在审议中**

我在最近的一篇文章中写道，投资时保持政治中立很重要（见

[你是一个好资本家吗](http://humblestudentofthemarkets.blogspot.com/2014/06/are-you-good-capitalist.html)

) 和

[巴里·瑞托兹](http://www.bloombergview.com/articles/2014-06-20/dick-cheney-would-be-an-awful-trader)

近几天也再次强调了这一主题。在此精神下，我想分析一下对黄金及其股票可能出现的技术性突破的兴奋情绪。

首先，考虑上述图表，但以五年时间为期限。长期模式的检查显示，只有 GDX/GLD 比率（中部面板）从相对下跌趋势中上涨，并现在显示出横向筑底模式。

另外两个，即黄金和白银/黄金比率，正在上涨以测试其长期下跌趋势。更乐观的解释可能是，这两个比率已经从相对下跌趋势中上涨（虚线表示），现在它们正在横向整理（以灰色显示）。

珍妮特·耶伦的鸽派信息又如何呢？难道这不能激发像黄金这样的通胀对冲工具更好的表现吗？

是的，但其他通胀对冲工具的图表并未确认黄金及其股票的上行突破。TIP 与 AGG（巴克莱综合债券指数）的相对表现显示，TIPS 在相对债券市场的表现已经开始好转。然而，我们还没有看到任何突破的迹象。

同样，金属和矿业股票（XME）相对于标普 500（SPX）的相对表现也显示出类似的模式，即从相对下跌趋势中出现的上涨，但 XME 仍处于相对横向整理模式。

总之，黄金的短期前景是乐观的。然而，由于黄金已经快速上涨了这么远，现在有点过度延伸。我更愿意等待更多关于上涨突破的决定性证据，然后再加入黄金牛市的行列。

你可以随意吐槽珍妮特·鸽派·耶伦和美联储，如果你想，甚至可以戴上锡箔帽。但我要提醒那些过于教条地看待黄金的读者，关于成为黄金死忠粉的最终真相可以在以下链接找到

[这里](http://incakolanews.blogspot.ca/2009/04/gold-la-plus-ca-change.html)

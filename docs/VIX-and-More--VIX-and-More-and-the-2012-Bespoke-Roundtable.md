# 波动率指数及更多：波动率指数及 2012 年定制圆桌

> 来源：[`vixandmore.blogspot.com/2012/01/vix-and-more-and-2012-bespoke.html#0001-01-01`](http://vixandmore.blogspot.com/2012/01/vix-and-more-and-2012-bespoke.html#0001-01-01)

我很高兴第三年连续被邀请分享我对 2011 年和 2012 年的看法，作为[Bespoke 投资集团](http://www.bespokeinvest.com/)第三届年度圆桌的一部分。

作为一个交易期权的人，仅仅是预测一个股票或指数在一个四周的到期周期内会发生什么，就可以让人感到自惭形秽。由于我很少把大部分思考集中在未来两三个到期周期之后，我总是喜欢接受我认为是定制的全面 37 个问题的“闭卷大作业”的过程。当然，你并不需要对所有问题都有一个观点才能成为一个成功的投资者，但是尽可能多地了解尽可能多的问题，会让你在下一个突发事件发生时更加有准备。正如我在[一个交易者的教育](http://vixandmore.blogspot.com/2010/09/education-of-trader.html)中所指出的，任何一个曾经说过“我们必须知道这个吗？”的人都不适合从事交易，因为在金融市场中，每一天都是一个公开书面小测验。

当回答这 37 个问题时，我试图平衡我认为最可能发生的情景和对一些低概率情景的更具挑衅性的评论，我相信这些情景比其他人更有可能实现。结果通常是一系列乏味和离谱的东西。在 2012 年，我似乎更倾向于乏味的一端，这在一个玛雅日历[可能预测](http://en.wikipedia.org/wiki/2012_phenomenon)世界末日的年份是不难做到的，而在这一年，许多专家都呼吁至少要结束我们知道的欧元区。话虽如此，我确实认为我对从苹果([AAPL](http://vixandmore.blogspot.com/search/label/AAPL))对谷歌([GOOG](http://vixandmore.blogspot.com/search/label/GOOG))到社交网络等各种问题都有一些具有挑衅性的看法。

我 2012 年的全部评论集在[这里](http://bespokepremium.com/roundtable/vixandmore/)。

对于那些好奇我水晶球过去的表现如何的人，我 2010 年的存档预测在[这里](http://bespokepremium.com/roundtable/vixandmore/2010/)。如果我能找到我的（显然不那么令人印象深刻的）2011 年预测的链接，我也会在这里发布。

我还建议阅读其他参与 2012 年定制圆桌的参与者的一些评论。完整的预测页面在[这里](http://bespokepremium.com/roundtable/)。

# 谦逊的市场学生：如何购买“智能”价值

> 来源：[`humblestudentofthemarkets.blogspot.com/2019/06/how-to-buy-smart-value.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2019/06/how-to-buy-smart-value.html#0001-01-01)

在过去十年中，价值投资遭受了重创，因为这种风格严重落后于市场。Callum Thomas 记录了价值折价在过去十年中是如何增长的。折价已经下降到了自纳斯达克泡沫高峰期以来未曾见过的水平，当时与互联网相关的股票暴跌至地面，而价值股票表现出色。

现在是买入价值的时候了吗？这里有一些建议，介绍如何参与价值风格，尽管过去 10 年中该风格面临逆风，但这种方式仍然表现良好。

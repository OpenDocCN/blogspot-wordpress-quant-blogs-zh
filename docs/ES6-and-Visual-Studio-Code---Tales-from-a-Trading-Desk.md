## ES6 与 Visual Studio Code

Visual Studio Code 的七月[更新](https://code.visualstudio.com/Updates)终于在 ES6 的世界中提供了一些意义。在项目的根目录下添加一个 jsconfig.json 文件，并至少在文件中添加以下内容：

```
{
     "compilerOptions": {
          "target": "ES6"
     }
}

```

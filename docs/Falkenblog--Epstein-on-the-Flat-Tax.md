# Falkenblog：关于全额税的埃普斯坦

> 来源：[`falkenblog.blogspot.com/2011/08/epstein-on-flat-tax.html#0001-01-01`](http://falkenblog.blogspot.com/2011/08/epstein-on-flat-tax.html#0001-01-01)

理查德·埃普斯坦有一个

[文章](http://www.hoover.org/publications/defining-ideas/article/86876)

争论一项关于全额税的条款。首先，渐进税将引发浪费的避税行为。如果你曾经与高收入人群的资产管理者交谈，你会了解到最紧迫的问题与选择良好的税前投资无关，因为移动收入跨越时间或某些人为类别有着更大的税后回报。其次，当税收按比例分摊时，有关政府规模的讨论更加理性，因为每个人都会内化成本。目前只有一半的美国工薪阶层纳税，因此不必考虑收入税对他们收入的影响。

这些都是很好的论据，但我想补充一下以下内容。詹姆斯·米勒斯因其关于最佳税收的工作而获得了诺贝尔奖（请参阅他 1971 年的开创性论文，

[关于最佳所得税收理论的探索](http://www.econ.yale.edu/~dirkb/teach/pdf/mirrlees/1971%20optimal%20taxation.pdf)

）。在这些模型中，最佳税率取决于有关收入赚取能力分布、边际收入效用下降速度以及税率对收入产生的影响的假设。

很容易理解渐进税的效用论：富人不像穷人那样重视每一美元，因此他们应该对最后一笔钱多交税，但是还有一个不太明显的强大对消力量。鉴于收入赚取机会呈对数正态分布，最高收入者比中等收入者创造的财富要多得多。因此，在

[大多数参数化](http://www.amazon.com/Optimal-Income-Redistribution-Matti-Tuomala/dp/0198286058)

你会发现

更低

高收入者税率越高越好，因为这些收入者就是创造财富的机器。对最富有的人征收更高的税率首先影响的是这些人，这与你想要的完全相反，你希望对他们的影响最小--就最大化创造的总财富而言。

我觉得这些文献令人信服，因为我是一个自由主义者，实际上就是一个动态效用主义者：我喜欢在人们选择行动以期望获得一定回报的世界中以时间计算幸福。与简单的效用主义者不同，他们假定产出已经确定。而且，我们往往认为财富在科学、艺术和一般的人际友情方面的意外收益是理所当然的；马克思的“农村生活的愚蠢”来自于永远没有时间思考。

不幸的是，我认为这个问题的恶化实际上是由这个事实加剧的，人们更关心的是相对收入而不是绝对收入；我们更嫉妒而不是贪婪。对于大多数人来说，特别是知识分子来说，承认自己从他们富裕的道德下级得到了好处，而这些下级从来没有打算做，而且似乎毫无额外义务地享受着，是一件难事，这些“进步人士”认为渐进税率是一件好事，无论破坏了多少财富，因为阻止一美元流向科克兄弟比一美元少流向一群工人更值得，特别是因为这些工人可能以某种方式与这样的混蛋做生意。

这就是为什么米尔利斯模型现在显得如此古怪。真正关心相对财富的人并不在乎财富与平等之间的权衡，因为对于关心相对财富的人来说，总财富并不重要，而驱使他们的幻想是打倒那些在他们之上的人（见上文）。没人愿意权衡，因为承认这种权衡表明财政乘数不大于一。最好把焦点放在总体国内生产总值上，这假设政府支出和投资是完全替代品，并且就不足的总需求争论乘数，这是一种既可辩护又贫乏的幻想。

嫉妒虽然自然，但是它是一个要超越的恶习，而不是一个在平等和正义名义下庆祝的美德。

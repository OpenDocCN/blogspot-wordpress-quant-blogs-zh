# 可量化边缘：糟糕八月后的九月表现

> 来源：[`quantifiableedges.blogspot.com/2011/09/september-performance-after-bad-augusts.html#0001-01-01`](http://quantifiableedges.blogspot.com/2011/09/september-performance-after-bad-augusts.html#0001-01-01)

九月份以股市最糟糕的月份之一而闻名。昨晚，两者

[伍德谢德](http://ibankcoin.com/woodshedderblog/2011/08/31/september-seasonality-sp-500/)

和

[迈克尔·斯托克斯](http://marketsci.wordpress.com/2011/08/31/september-weakness/)

提供了九月份的良好历史分析。连续第二年，市场经历了一个艰难的八月，标普 500 指数下跌约 5.6%。这让我想知道，在经历了糟糕的八月之后，九月的表现如何。下面我整理了所有在八月份下跌 4%或更多后的九月份列表。

在所有这些九月份中，并没有明显的强劲趋势，但显而易见的是，这些九月份都表现出了较高的波动性。1966 年的情况是市场从高点到低点仅有略高于 5%的幅度。八个实例中有六个在九月份的范围都超过了 9.5%。因此，我不会期待本月市场行情会变得平淡无奇。

# Falkenblog: 斯蒂格利茨称次级抵押贷款问题

> 来源：[`falkenblog.blogspot.com/2008/08/stiglitz-called-subprime-mortgage.html#0001-01-01`](http://falkenblog.blogspot.com/2008/08/stiglitz-called-subprime-mortgage.html#0001-01-01)

乔·斯蒂格利茨将他的名字加入了那些称此次金融危机的人的名单。我猜这就像 1987 年的股市崩盘一样，每个人似乎都在那之后立刻称之为危机。在他

[讲座](http://nobellaureate.feedroom.com/?fr_story=ad40336bfe4768ab29d5e28ea7bfeb558d28e3df&rf=bm&skin=showcase)

在林道诺贝尔奖得主会议上，他说他是告诉市场危机即将发生的几个人之一。

他的批评类似于塔勒布的（他提到了被忽视的重尾部），但更一致，因为几十年来，他已经写下了一些模型。他的许多作品都是基于证明特定的信息不对称如何导致偏离完全市场假设，非帕累托有效的结果（可以使某人变得更好，而不使任何人变得更糟）。

但世界充满着不完美，并且不是帕累托有效的事实，并不能意味着政府监管是一种改进，尤其是因为政府具有更严重的不完美。事实上，政府就在其中，因为政府控制的房利美和房地美正在降低他们的承保标准，以增加家庭所有权，特别是对历史上处于不利地位的社区。当然，像许多批评者一样，他并没有称呼这场危机，他只是在指责市场的非理性和低效，是几十年来一直如此。

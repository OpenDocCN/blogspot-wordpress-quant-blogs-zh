# VIX and More: Open Thread: Do Fibonacci Retracements Work?

> 来源：[`vixandmore.blogspot.com/2009/06/open-thread-do-fibonacci-retracements.html#0001-01-01`](http://vixandmore.blogspot.com/2009/06/open-thread-do-fibonacci-retracements.html#0001-01-01)

在我最近的[Dueling Fibonaccis for the SPX](http://vixandmore.blogspot.com/2009/06/dueling-fibonaccis-for-spx.html)文章发表后，有读者询问我是否了解任何试图评估斐波那契回撤水平有效性的“科学”研究。简单回答是，我不了解任何详细的研究，也无法在[SSRN 数据库](http://vixandmore.blogspot.com/2009/06/dueling-fibonaccis-for-spx.html)中找到任何有意义的内容。

所以我将这个话题交给读者：

+   斐波那契数列是否可以作为交易工具？

+   您是否了解任何已发表的分析，该分析评估了斐波那契回撤水平？

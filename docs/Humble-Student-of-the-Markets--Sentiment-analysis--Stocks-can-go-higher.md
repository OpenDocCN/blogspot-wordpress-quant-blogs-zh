# 谦逊的市场学生：情绪分析：股市还能涨

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/01/sentiment-analysis-stocks-can-go-higher.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/01/sentiment-analysis-stocks-can-go-higher.html#0001-01-01)

库伦·罗奇在

[实用资本主义](http://pragcap.com/individual-investor-stock-allocation-hits-post-crisis-high)

最近强调了最新 AAII 资产配置调查的结果，该调查显示个人投资者的股票配置达到了危机后的最高水平——一个拥挤的多头市场。顺便说一下，我认为 AAII 的资产配置调查比情绪调查更有用，因为这里的关键是观察他们做什么（资产配置），而不仅仅是他们说什么（情绪）。

TD Ameritrade 还发布了一个月度调查，调查客户股票配置，称为

[IMX 指数](https://imx.tdameritrade.com/IMX/index.jsp)

（在下面的图表中以绿色显示）。IMX 指数的历史没有 AAII 调查那么悠久，但它确实证实了个人投资者现在正纷纷涌入股市的现象。

在零售投资者拥挤的多头市场中，现在是更加谨慎对待美国股市的时候了吗？

**机构仍在买入**

还没有。机构投资者还没有完成买入。

最新的美银美林基金经理调查显示，尽管宏观前景看好，但机构投资者还没有集体看多。

[调查结果](http://newsroom.bankofamerica.com/press-releases/economic-and-industry-outlooks/bofa-merrill-lynch-fund-manager-survey-finds-strong-gl)

，mid-December 发布的报告称，机构投资者对全球经济的看法转为乐观：

> 认为全球经济在未来一年将会有所加强的投资者比例已经从 11 月的净 67%上升到净 71%。对比 12 个月前，当时只有净 40%的调查对象预测全球经济将加强，目前对全球经济的信心要强得多。

现金水平仍然很高，这在改善的基本面前景下，对价格上涨是有支持的：

> 平均现金余额保持在投资组合的 4.5%，从历史角度来看，这是一个对股市有积极信号的水平。净 16%的资产配置者表示他们超配现金，高于 11 月的净 9%。更高的现金水平与预期更高利率以及 75%的调查对象的预期相符，即美联储会在 2014 年第一季度引入缩减购债。

下面的图表显示了基金经理的总体资产配置。经理们显著超配欧洲和日本股票，并且超配美国股票，尽管程度并不过高。美国股票的超配仅比历史平均水平高出 0.5 个标准差，这只是噪音。

此外，市场共识正逐渐转向对 2014 年经济积极势头的预期，下图中

来自[BCA Research](http://blog.bcaresearch.com/better-global-economic-momentum)

是众多例证之一。全球宏观前景的改善是基金经理承担更多股票风险的主要原因。

在经济积极势头和基本面改善的推动下，机构投资组合将股票配置比例提高，美国股市在未来几个月里似乎还有进一步的上涨空间。到了年中，机构的买入可能会推动股价进一步走高，并将基金经理的持仓推向更为拥挤的多头位置。如果这些情况发生，我对 2014 年上半年股市表现强劲、下半年转弱的基本情景（参见

我的[2014 年计划](http://humblestudentofthemarkets.blogspot.com/2013/12/my-plan-for-2014.html)

）开始呈现出相当乐观的迹象。

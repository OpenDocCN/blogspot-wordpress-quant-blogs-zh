# 市场的谦逊学生：变得看涨，但首先等待回调

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/08/getting-bullish-but-first-wait-for.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/08/getting-bullish-but-first-wait-for.html#0001-01-01)

我之前曾写道，在全球市场中，除了美国之外，我看到了一些复苏的迹象（参见

[2012 年剩余时间该做什么](http://humblestudentofthemarkets.blogspot.com/2012/08/what-to-do-for-rest-of-2012.html)

）。在最近股票的强劲势头中，我现在终于看到了风险开启/关闭交易看涨解决的迹象。

考虑下面摩根士丹利周期指数的相对表现图，以黑色显示，以及公用事业板块相对于市场的相对表现。最近，周期性股票出现看涨反转，而以防御为导向的公用事业部门在相对基础上出现了看跌反转。

**等待回调**

尽管对风险开启交易有看涨的基础，但今天我还不会过于兴奋。我的许多短期指标已经处于深度超买区域。尽管中期趋势可能是上涨，但我会耐心等待回调，然后再将新的资金投入到股票中。

过去几周，宏观环境相对积极，但我们仍然处于一个起伏不定且波动剧烈的环境。鉴于在欧洲普遍看到积极背景，市场正在为失望做好准备，并且随时可能遭遇空气口袋。可能的触发因素包括

[更多关于希腊退欧的讨论](http://www.marketwatch.com/story/get-ready-for-more-grexit-speculation-2012-08-17)

，在年度杰克逊霍尔会议上可能会因为更多量化宽松政策的预先宣布而感到失望，也可能对 8 月非农就业数据感到失望，

[盖洛普](http://www.gallup.com/poll/156755/Unadjusted-Unemployment-Mid-August.aspx)

他们的失业调查已经显示出恶化。

我内心的投资者准备在接下来的几周内平均买入股票。我内心的交易者想要等待，甚至在这里筹集一些现金，为即将到来的修正期买入做好准备。

# 附加 RSO 回测 | CSSA

> 来源：[`cssanalytics.wordpress.com/2013/10/15/additional-rso-backtests/#0001-01-01`](https://cssanalytics.wordpress.com/2013/10/15/additional-rso-backtests/#0001-01-01)

还有一个值得关注的博客作者是来自[Systematic Edge](http://systematicedge.wordpress.com/)的 Michael Guan。Michael 经常写关于资产配置的不同方法。例如，他的一些文章对于主成分分析（PCA）非常全面，值得一读。最近，他写了一篇[文章](http://systematicedge.wordpress.com/2013/10/14/random-subspace-optimization-max-sharpe/)，展示了对“k”进行了一些附加测试，使用[Random Subspace Optimization](https://cssanalytics.wordpress.com/2013/10/06/random-subspace-optimization-rso/ "Random Subspace Optimization (RSO)") (RSO)在各种宇宙中进行最大夏普优化（仅做多）。结果很有趣，并支持以下结论：1）RSO 是增加收益和降低风险的一个有前途的方法 2）选择“k”对于获得最佳结果是重要的。我将提出一些关于扩展 RSO 的想法。

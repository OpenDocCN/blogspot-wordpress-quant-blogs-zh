# Falkenblog: Landscapes Rule

> 来源：[`falkenblog.blogspot.com/2009/01/landscapes-rule.html#0001-01-01`](http://falkenblog.blogspot.com/2009/01/landscapes-rule.html#0001-01-01)

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEijWm72gEDO1rLV8hwJUDN7_hh7PdeaCPnaiw6rf5jNAa5sTTFV7w9cchWQl_wh7EFHRkrRaWzguqXpM0-DiEmQlHCFpaLnNzKxiPohG8w7w85qczOPeSkVLaTyh8lnj7i8YkB9UA/s1600-h/cole.jpg)

数据表明，来自所有文化的人都倾向于喜欢风景。这些绘画有开阔的空间，有树木的大树，树枝朝着地面，有一条河流，两座山之间的景色，动物，开花结果的植物，最后，还有一条可以跟随的小路。因此，现代艺术家要注意：100 年后，没人会在乎你对人类非人道行为或者平庸的前卫看法，比如一堆罐头汤。有趣的是听到这个人指出，人们对艺术的需求要依赖于文化。

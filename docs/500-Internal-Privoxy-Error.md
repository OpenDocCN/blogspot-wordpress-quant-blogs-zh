# 500 内部 Privoxy 错误

Privoxy 在处理您的请求时遇到了一个错误：

**无法加载模板文件 `no-server-data` 或其包含的组件之一。**

请联系您的代理管理员。

如果您是代理管理员，请将所需的文件放置在 `*(confdir)*/templates` 目录中。 `*(confdir)*` 目录的位置在主 Privoxy `config` 文件中指定。（通常是 Privoxy 安装目录）。

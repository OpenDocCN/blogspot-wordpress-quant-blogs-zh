# 市场谦卑的学生：解读零售销售数字

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/02/interpreting-retail-sales-number.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/02/interpreting-retail-sales-number.html#0001-01-01)

今天报告的零售销售数字喜忧参半。头条数字低于预期，但除汽车外的数字强劲。到处都是修订。

这该如何解读？

我相信美国消费者**理查德·费曼**，诺贝尔物理学奖获得者。

[消费者指标研究所](http://www.consumerindexes.com/index.html)

产生一系列跟踪美国消费者每日活动的数字。日常系列显示适度上升。

日常数据 inherently noisy, but the 月度数字显示了消费者需求明显反弹。

最后， Nomura (通过

[商业内幕](http://www.businessinsider.com/this-is-the-1-sign-that-consumer-are-really-starting-to-feel-good-2012-2)

) 对零售销售报告持积极态度：

> 控制措施反馈到 GDP 的消费者支出部分估计，并暗示年初消费旺盛。最后，**我们偏好的消费者舒适度衡量标准，即外出就餐类别，1 月份增长了 0.6%。外出就餐可以看作是消费中超额可选类别的支出，如果信心动摇，家庭通常会首先削减这些支出**。

记住，我们现在处于一个由中央银行引发的流动性反弹中。上周英国银行加入了派对，本周则是日本银行。请享受吧。

# 量化边缘：强烈的纳斯达克中期多头情绪

[Quantifinder](http://quantifiableedges.blogspot.com/2009/05/quantifinder-unveiled.html)

对于两者

[黄金和白银订阅用户](http://www.quantifiableedges.com/meminfo.html)

# 谦逊的市场学生：通往辉煌的过剩之路

尽管出现了一些过剩现象，但我重申我的观点，人工智能的泡沫还有很大的上升空间，在达到辉煌的过剩阶段之前，它还有很长的路要走（参见[为什么这次的人工智能牛市与 2000 年的纳斯达克不同](https://humblestudentofthemarkets.com/2024/02/11/why-this-ai-bull-is-nothing-like-nasdaq-in-2000/)）。

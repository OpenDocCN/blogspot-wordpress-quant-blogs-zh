# Falkenblog：不要相信专家

> 来源：[`falkenblog.blogspot.com/2010/10/never-trust-experts.html#0001-01-01`](http://falkenblog.blogspot.com/2010/10/never-trust-experts.html#0001-01-01)

科学在某种程度上包含了一些有趣且有用的基本真理，是无知和迷信的一种主要替代。然而，在前沿领域，这主要是一种看见你相信什么的技巧，因此主要是一种诀窍，知道要证明什么，因为聪明的人可以证明任何事情。

在一篇关于套利交易的文章中

[Menkhoff, Sarno, Scheling, and Schrimpf](http://papers.ssrn.com/sol3/papers.cfm?abstract_id=1342968)

说他们通过“风险溢价”解释战略的正回报。这才是他应该的样子，任何可辨认的回报溢价都应该只能通过风险来解释。

所以，这是关于波动性以前工作的一个抛出的小线索：

> 我们研究了另一个风险因素的经验表现：全球外汇波动性创新。这个因素是 ICAPM 理论所提出的市场波动性变化的代理，并且是用于定价股票收益横断面的[Ang, Hodrick, Xing, and Zhang（2006）](http://www.afajof.org/afa/forthcoming/ang1.pdf)所使用的汇总波动风险因素的类似物。我们证明全球外汇波动实际上是外汇超额收益的普遍风险因素。

好的，他们的意思是“这并不是什么难题，一切都可以在标准框架下解释清楚。我们使用风险因素“波动率创新”，这在 2006 年就被证明是一个相关的定价因素。”问题在于所引用的论文显示，高波动率创新与低回报是相关的。此外，除创新外的更高波动性与更低回报相关。事实上，“波动率创新”这个伎俩显然是企图扭曲数据来得出作者想要的结论。在这里，风险溢价实际上是合理的（如果你相信他们的结果，我对他们的结果是否有把握表示怀疑，因为像 2001-2002 年这样好，1997-2000 年这样糟的时候都有收益交易；数据并没有你认为的那么多；请看下面的眼球线，看看是否返回与“好时光”正相关）。如果波动率创新是一种定价风险因素，在其他市场波动率及与其相关的事物通常与回报成反比。

这种对逐渐消逝的风险溢价的搜寻，就像是一个转瞬即逝的幽灵，却无处不在且非常重要，成为了我们这个时代的神话之一。我有更重要的事情要做。

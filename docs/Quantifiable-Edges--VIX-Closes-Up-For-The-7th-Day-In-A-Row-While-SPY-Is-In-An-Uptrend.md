值得注意的是，周三标志着 VIX 连续第七天上升。这是一个非常不寻常的连涨。我决定回顾在 SPY 高于其 200 日均线的情况下，VIX 连续七天上升的其他所有时刻。以下是假设持有期限为 3 天的 SPY 的结果。

虽然这种情况出现的次数不多，但到目前为止，其回报率几乎完全是看涨的。与上涨幅度和平均交易相比，回撤非常小。这似乎值得关注。

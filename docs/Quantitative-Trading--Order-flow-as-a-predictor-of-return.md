# 量化交易：订单流量作为收益预测器

> 来源：[`epchan.blogspot.com/2012/10/order-flow-as-predictor-of-return.html#0001-01-01`](http://epchan.blogspot.com/2012/10/order-flow-as-predictor-of-return.html#0001-01-01)

订单流量是 signed 交易量：如果一个订单在 ask 价格被执行，增量订单流量为 +(订单大小)；如果在 bid 价格被执行，则为 -(订单大小)。在某些市场中，交易者只能从市场制造商那里买入和卖出，但不能相互交易，此时正向订单流量意味着交易者是某证券的净买家。但在所有人都可以在公共订单簿上放置和成交订单的市场中，正向订单流量表明，有信息的交易者（那些愿意积极建仓的交易者）正在急于收购某证券。

订单流量的有趣之处在于，它已被证明是一个好的动量指标。也就是说，正向流量预示着正向未来收益。这可能看起来显而易见，但你要记住，一般来说，正向过去收益绝不代表正向未来收益。FX 订单流量具有这种预测能力，是由 Evans 和 Lyons 在一系列

论文链接：[`www.bis.org/publ/bppdf/bispap02j.pdf`](http://www.bis.org/publ/bppdf/bispap02j.pdf)

中展示的，但这个指标在许多其他市场中很有用，且在不同的时间尺度上也很有效。例如，在一个

论文链接：[`www.people.hbs.edu/estafford/Papers/AFS.pdf`](http://www.people.hbs.edu/estafford/Papers/AFS.pdf)

由 Coval 和 Stafford 表明，如果你能仅通过共同基金交易来揭示某股票的订单流量，你还可以预测其未来收益，比如，一个季度。这篇论文不仅表明订单流量具有预测性，而且还表明，有时特定类型的订单流量（在这个例子中，仅共同基金的）比一般的订单流量更具预测性。在许多情况下，交易者发现，仅计算机构交易者的订单流量，或大订单的订单流量，他们可以更好地预测未来收益。 (难怪机构交易者正竭尽全力将他们的订单拆分成小块，或在暗池中交易！) 我最近还听说，流入行业 ETF 的订单流量可以预测该行业的收益。如果任何读者读过论文，或者有这种行业轮动模型的经验，请留言！

尽管订单流已被证明是有用的，但使用它的零售交易者并不多。原因很简单：它可能难以衡量。特别是在外汇市场上，许多市场不报告交易信息，或者他们报告的信息有足够延迟，以至于信息没有预测价值。即使对于报告即时交易信息的市场，你也需要很好的软件来捕捉每一个买入价、卖出价、交易和交易大小，并将它们存储在数组中，以计算订单流，这对于大多数零售交易软件来说是不可能完成的操作。然而，这个进入门槛可能意味着从这个指标中仍然可以提取出不错的阿尔法。

# Falkenblog: 更多证据表明羡慕战胜了贪婪

> 来源：[`falkenblog.blogspot.com/2012/10/more-evidence-that-envy-trumps-greed.html#0001-01-01`](http://falkenblog.blogspot.com/2012/10/more-evidence-that-envy-trumps-greed.html#0001-01-01)

两位以色列经济学家，哈艾姆·莱维和盖·卡普兰斯基，有一篇新论文，

[带有嫉妒和利他情感的投资选择。](http://papers.ssrn.com/sol3/papers.cfm?abstract_id=2148874)

他们探讨了什么样的效用函数能解释以下偏好。

他们向人们询问了以下情景：

1. 在这一年里，被试的股票投资价值从$25,000 增加到$27,500，而股指也以相同的速度增长。

2. 被试的投资以及股指的增长情况与情景 1 完全一样，而同行群体的投资组合却增长了 50%，从$25,000 增加到$37,500。

3. 在这一年里，被试的投资组合以及股指的价值从$25,000 减少到$22,500。

4. 被试的投资和股指的下降情况与情景 3 完全一样，而同行群体的投资组合却下降了 50%，从$25,000 减少到$12,500。

毫无疑问，他们发现“羡慕概念主导了结果，因为当同行群体的收入增加时，效用基本上会降低，而当同行群体的损失超过被试时，效用就会增加。” 相对财富比绝对财富更重要。 这与我的著作的理论论点一致，

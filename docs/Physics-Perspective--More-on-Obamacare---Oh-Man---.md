# 物理视角：更多关于奥巴马医改...哦天...

> 来源：[`physicsoffinance.blogspot.com/2013/12/more-on-obamacare-oh-man.html#0001-01-01`](http://physicsoffinance.blogspot.com/2013/12/more-on-obamacare-oh-man.html#0001-01-01)

发生了一件有点奇怪的事情。

[我最新的彭博社专栏](http://www.bloomberg.com/news/2013-12-02/does-obamacare-put-too-much-faith-in-markets-.html#disqus_thread)

，上周一发表的文章几乎瞬间消失，原因不明。经过一番调查，似乎 html 中或不知名的其他语言中缺少了一个“代码”。无论如何，现在它已经存在了。

主题是奥巴马医改。简而言之，主题是，我们不应该期望市场轻易地解决医疗问题（想想

[柠檬市场](http://www.google.com/url?sa=t&rct=j&q=&esrc=s&source=web&cd=1&cad=rja&ved=0CCsQFjAA&url=http%3A%2F%2Fen.wikipedia.org%2Fwiki%2FThe_Market_for_Lemons&ei=DR-iUv-8JMWBhAea9YDgAQ&usg=AFQjCNGDXXlaTMuexFzdaQ1YA9CqVSjNRA&sig2=HFNqpVa8benebd_tOCNRVQ)

市场失败的主题（在类似情况下）。我认为，经济学家 mostly 知道这一点；有些事情可以通过政府的组织来做得更好。那么，为什么这个信息不能传达出去呢？或者，经济学家真的相信这一点吗？我分辨不出来。

无论如何，每个人都应该看看比我的小专栏更深入的两件事：

1. 第一件值得阅读的事情是

[一篇很棒的最新文章](http://www.aeaweb.org/articles.php?doi=10.1257/jep.27.4.121)

迈克尔·桑德尔的文章，探讨了市场往往与我们想象的完全不同的价值；为某物创造市场往往会改变我们对其的思考和价值观，这对市场是否有益具有重大影响。他提出的一个重要观点是，是否应该让市场自行其是 NOT 是一个经济学问题；它总是（或几乎总是）涉及到比经济效率更广泛的价值观，因此超出了经济学家的专业领域。

我认为桑德尔是对的。我认为很多人开始意识到这一点。甚至

[教皇](http://www.newyorker.com/online/blogs/johncassidy/2013/12/pope-francis-is-no-marxist-hes-a-marian.html)

!!

2. 第二件值得阅读的事情是一篇迷人的

[书籍](http://www.amazon.com/Social-Limits-Growth-Fred-Hirsch/dp/0415119588)

这是 Fred Hirsch 在 1976 年写的《社会成长的限制》。在阅读桑德尔的文章之前，我从未听说过这本书，这让我有点尴尬，因为我相信每个经济学研究生都在他们的常规培训中读过这本书。我还有很多要学习的。这是一本迷人的书，一本真正的经典，它暗示了我们的某些基本心理和社会行为对我们的经济福祉必须有长期影响，而市场的一般理论完全忽略了这一点。当你这样说的时候，这是显而易见的，但这是经济学……人们已经努力否认显而易见的事实……Hirsch 努力不去……

关于 Hirsch 的简要优秀入门，可以看

[这个链接](http://www.umanitoba.ca/philosophy/ethics/media/social_limits_to_growth.pdf)

这是由曼尼托巴大学哲学系的一位人士，或者与该系有联系的人，或者该系某人的狗写的……我完全不知道是谁。但文章写得很清楚，我对此表示敬佩。

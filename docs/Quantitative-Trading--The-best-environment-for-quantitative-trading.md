# 量化交易：量化交易的最佳环境

> 来源：[`epchan.blogspot.com/2009/10/best-environment-for-quantitative.html#0001-01-01`](http://epchan.blogspot.com/2009/10/best-environment-for-quantitative.html#0001-01-01)

让我来谈谈一个比通常的高深理论讨论更平凡，但对长期利润有着同样重要影响的话题：什么样的办公环境最适合进行量化交易策略的研究和执行？

我以前在不同的办公环境中工作过，所以我觉得有资格提供一个明智的意见。

在摩根士丹利，我蜷缩在一个与办公室其他地方隔开的半隔间的桌子上：除非我或他们站起来，否则没有人能看到或打扰我。在瑞士信贷，我和其他两位专有交易同事共用一个办公室，其中一位喜欢与办公室同事自由地分享他对各种时事的看法。（另一方面，他抱怨我吃午餐时咬苹果的声音对他来说太大了。）在新泽西州的枫树基金会，我和大约其他 100 位同事在交易大厅共用一个办公室，其中许多人都容易陷入同样的观点分享诱惑。

在我的公司，我独自一人坐在地下室办公室里（除了我的猫），我心爱的

[古典 FM](http://www.classicfm.co.uk/on-air/ways-of-listening/)

通过互联网实时传输，我的台式电脑与我们芝加哥办公室的合作伙伴电子连接，我们的交易服务器在亚马逊和其他地方，以及世界各地的其他客户和合作伙伴，但在整个一天内不受任何干扰，除非我自愿接听电话、回复邮件或即时消息。

你能猜到哪种环境是我认为最具生产力的吗？哪一个压力最小？哪一个对我的雇主/合作伙伴/客户的底线贡献最大？

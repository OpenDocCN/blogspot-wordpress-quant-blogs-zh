# Quantifiable Edges: 当在大涨之后出现大跌

> 来源：[`quantifiableedges.blogspot.com/2011/09/when-big-drop-come-after-bigger-gains.html#0001-01-01`](http://quantifiableedges.blogspot.com/2011/09/when-big-drop-come-after-bigger-gains.html#0001-01-01)

当市场经历像昨天那样的强烈回调时，它第二天往往会有所反弹。尽管这种倾向存在，但差异很大，并且取决于许多因素。要考虑的一个因素是市场前一天的走势。让我们看看几条股指曲线来证明我的观点。第一条曲线显示了自 2003 年以来，在收盘时购买所有 1%的跌幅，并在第二天收盘时卖出后的结果。

我将这个策略回溯到 2003 年，因为大约是从那时起，这种股指曲线开始显示出一致的上涨倾向。现在让我们看看所有情况下，大跌未能使标普 500 指数收盘价低于两天前的收盘价。

有不同的方法来看待这个问题，但这里有一个简单的看法。如果今天的大跌没有抹去昨天的涨幅，那么它可能还有更长的路要走。

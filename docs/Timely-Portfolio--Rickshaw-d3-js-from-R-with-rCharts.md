# Timely Portfolio: 使用 rCharts 和 Rickshaw 的 R 语言 d3.js

> 来源：[`timelyportfolio.blogspot.com/2013/06/rickshaw-d3js-from-r-with-rcharts.html#0001-01-01`](http://timelyportfolio.blogspot.com/2013/06/rickshaw-d3js-from-r-with-rcharts.html#0001-01-01)

Shutterstock 开源的 [Rickshaw](http://code.shutterstock.com/rickshaw) 提供了一个非常棒的教学文档([链接](http://code.shutterstock.com/rickshaw/tutorial/introduction.html))，帮助你开始用 Rickshaw 构建交互式时间序列 [d3.js](http://d3js.org) 图表。我忍不住想要重建这个教程，但这次是**完全使用** [R](http://r-project.org) 语言，并借助于功能强大的包 [slidify](http://slidify.org) 和 [rCharts](http://github.com/ramnathv/rcharts)。我将在下面嵌入教程，但如果你直接访问 [Github 页面版本](http://timelyportfolio.github.io/rCharts_rickshaw_gettingstarted/)，它看起来可能会更佳。

slidify 和 rCharts 的作者 Ramnath Vaidyanathan 做了出色的工作，并继续完善这些优秀的工具。它们完全改变了我的工作流程和输出，让我能够结合动态和交互式组件，创建出精美的文档。务必去参加他在 [2013 年 6 月 17 日](http://www.meetup.com/nyhackr/events/122192822/) 的 NYHackr 聚会的演讲。

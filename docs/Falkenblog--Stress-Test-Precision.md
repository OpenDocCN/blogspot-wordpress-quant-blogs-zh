# Falkenblog：压力测试精确度

> 来源：[`falkenblog.blogspot.com/2009/04/stress-test-precision.html#0001-01-01`](http://falkenblog.blogspot.com/2009/04/stress-test-precision.html#0001-01-01)

设定现实的目标是很重要的。因此，听到白宫新闻发言人的话是令人不快的

[Robert Gibbs](http://www.bloomberg.com/apps/news?pid=20601103&sid=aTJal.ypxqE4&refer=us)

> 压力测试旨在确保政府“会对问题有一个准确的诊断”

精确吗？因为对于未交易的银行资产价值存在相当大的不确定性，评估他们的

改变

在给定失业率上升至 13%或房屋资产下跌 15%的情况下的价值是什么，就好比问如果洋基队失去他们的先发投手会发生什么。当然，他们会更糟。更糟多少？人们会有不同的意见。事实上，了解更多的人，他们的意见范围就会更大。无知的人可能只是简单地比较投手的胜利次数和整个球队的平均水平，然后求出差值。更复杂的萨伯统计学方法会考虑到投手最近的表现，特别是在春季赛上，以及这与去年的比较，以及他最好的一年。

预测精确的诊断结果表明白宫是一知半解的，希望能将此结果似乎清晰地支持正在推行的任何政策（因为诊断结果是

精确

，这意味着

明显

）。他们唯一的问题是说服人们他们的压力测试是精确的，我认为他们会发现这是不可能的。指出“模型”和复杂的推理并不足够。向我们展示假设、模拟、按债务人和机构划分的风险说明，你无法不拒绝这个假设——你所产生的数字可能偏离了三倍，这将带来完全不同的政策建议。

完美是良好的敌人，如果你表示期望诊断结果是完美的，那么你就是在确保失败。因为如果你相信银行资产及其敏感性没有很大的标准误差，你会采取不同的政策，而如果它们确实存在很大的标准误差。

考虑犯罪。一个现实的目标是努力

最小化

暴力，并最小化无辜者被定罪的可能性。理想主义者会试图

消除

犯罪和错误定罪。实际的方法在接受自己的局限性方面会更成功，因为它能接受类型 1 和类型 2 的错误。

我正在看

[Bloggingheads](http://bloggingheads.tv/diavlogs/19020)

在两位政治科学家之间，他们正在讨论问题的“复杂性”，因为你不能让船只携带小武器，也不能射杀所有海盗，而海盗只是为了赚钱，通常不会伤害受害者（下面我剪掉了 1 分钟他们抱怨的理由）。从根本上讲，没有完美的解决方案，最后他们指出这确实是非常烦人的。在他们的学术圈里，这听起来像是合理的辩论，但实际上是面对一个不完美解决方案所引发的懦弱无法决定。这让我想起了教父第一部中的一幕，唐·科里昂打了哭泣的约翰尼·方丹，说道“你得像个男人！”。

[你得像个男人!](http://josephmcbee.wordpress.com/2009/02/27/you-can-act-like-a-man/)

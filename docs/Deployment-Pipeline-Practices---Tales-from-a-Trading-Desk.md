# 部署流水线实践 | 交易台的故事

> 来源：[`mdavey.wordpress.com/2014/05/12/deployment-pipeline-practices-2/#0001-01-01`](https://mdavey.wordpress.com/2014/05/12/deployment-pipeline-practices-2/#0001-01-01)

## 部署流水线实践

正如估算并不是免费的（请参阅上一篇博客文章），软件部署[流水线](http://my.safaribooksonline.com/book/software-engineering-and-development/software-testing/9780321670250/anatomy-of-the-deployment-pipeline/ch05lev1sec3)也不是免费的。要实现按顺序排列的待办事项中所预测的故事[完成](http://thinkinginsoftware.blogspot.com/2014/05/is-tdd-dead.html)，人们希望验收标准覆盖了故事达到生产（或克隆生产）环境，部署和性能测试。这导致了与产品负责人的第一个问题，第一个故事的投资回报率可能不等于第一个故事的成本。没有免费的午餐。

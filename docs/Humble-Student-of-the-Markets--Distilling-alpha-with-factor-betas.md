# 市场谦逊的学生：用因子贝塔提炼 alpha

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/01/distilling-alpha-with-factor-betas.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/01/distilling-alpha-with-factor-betas.html#0001-01-01)

任何投资组合经理最难回答的问题之一是：“你的策略可能会出现什么问题？”如果他不知道，那就说明他没有完全考虑清楚他的投资方法的全部细微差别。

**如何利用因子贝塔解开 alpha**

这是我曾经在和对冲基金合作时评估投资策略的一个例子。

几个月前，一位经验丰富的债券组合经理带着一个针对加拿大市场的债券策略想法找到了我们。这个想法包括在加拿大债券组合的基础上，叠加美国国债市场的看涨期权写策略。由此产生的组合具有以下好处：

+   与基准相同的久期敞口，即没有显著的利率方向性赌注；

+   相比基准，具有更高的信用质量

+   与基准相比，实现收益显著更高

我采访了投资组合经理，发现该策略最显著的风险（不分先后）包括：

+   美国和加拿大收益率曲线之间的基础风险

+   货币风险

+   与期权定价相关的波动性风险

**利用因子贝塔分解 alpha**

他已经实时运行了一个“实盘”纸质组合一年多。回测结果良好，在这一时期内增加了超过 3%的 alpha。鉴于我对他的风险敞口的评估，我想看看超额收益是否是由于有利的因子贝塔敞口所致，即策略赌对了某种类型的敞口并且奏效了，或者 alpha 相对独立于因子贝塔。

我对不同情景下策略的周收益 alpha 进行了平均。在第一行，我问道：“当美国长期债券价格上升、下降或相对平稳时，平均每周的 alpha 是多少？”在这种情况下，当长期债券价格上涨时，策略表现不佳，主要是因为在收益率曲线的长期端卖出看涨期权，而期权溢价不足以抵消债券价格的上涨，而在长期债券价格持平或下跌时表现优于基准。

同样，我进行了其他形式的情景分析。当加拿大和美国曲线出现分歧时，alpha 会发生什么？当隐含的股票波动性（我没有一个很好的债券波动性代理）上升或下降时会发生什么？

**对因子贝塔敞口进行压力测试**

我使用了情景分析来预测年度 alpha。平均案例分析假设一个高斯分布，其中 50%的时间暴露于该因子贝塔是中性的，25%的时间是有利的，25%的时间是不利的。以长期债券价格因素为例，我计算了预期 alpha，假设 25%的时间债券价格下跌，25%的时间上涨，50%的时间保持中性。在这种情况下，预期 alpha 为每年 3.90%。

我想进一步对策略进行压力测试。如果市场之神不在我们这边怎么办？

在第二列标记为“不利情况”的部分，我分配了 50%的中性权重，33%的不利权重和 16%的有利权重。以长期债券价格因素为例，预期的 alpha 为每年 0.80%。

如果发生灾难性情景会如何？在第三列标记为“最坏情况”的部分，我分配了 2/3 的中性权重，1/3 的不利权重和 0 的有利权重。（请记住，这些是每周的 alpha 值，很难相信在长期债券价格的情况下，会在整个年度中有一周下跌，而在 52 周中会有 17 周上涨。）在长期债券价格因素的最坏情况分析中，预期的 alpha 为-2.13%。

**结论**

把所有这些放在一起，策略看起来相当不错。我们可以预期每年的 alpha 在 3.0-3.6%之间——这对于一个债券组合来说是一个惊人的数字。在典型的不利年份，我们仍然可以预期超额表现 0.8%至 1.8%。如果情况最糟，一切都不顺利，alpha 下降至-2.1%至 0.2%之间。

这是一个使用情景分析和因子贝塔评估任何投资策略的简单方法的例子。这是另一种提问方式：“投资策略可能会出现什么问题，情况可能会糟糕到什么程度？”

最终，我们没有实施该策略，并非出于投资原因，而是出于商业原因。

+   收益曲线的短端

+   BBB-国债信用利差

+   上市公司股权回报率超过 libor

**为什么高风险产生低溢价：**

赢家诅咒：对于高波动股票的过剩需求产生了低于平均水平的回报

为什么额外的需求？

+   过度自信

+   关于高风险公司的信息成本较低

+   高回报有高风险，因此风险意味着更高的回报谬论

+   有些人风险偏好

+   有些人喜欢正偏态

+   阿尔法发现

+   更容易向客户销售（易接受故事）

+   对于基金经理的回报

+   有限理性（认为 SML 是正斜的）

+   代理问题（利用基金来源的选择权）

+   买进股票的人认为股票会上涨，如果是这种情况，高贝塔更好

**为什么平的/向下的 SML 不套利？**

+   传统（60-40 的股票/股权比是一种约束）

+   非理性（其他人没有注意到 SML 是平的/负的）

+   *相对风险（我的最爱，下面有其他一致的数据）*

+   伊斯特林悖论表明幸福是相对的

+   没有人卖低风险、低于平均回报的股票，因为在一个相对风险的世界里，只有在回报高于平均水平时才会承担风险

+   国内偏好是因为想要超越同行，而不是陌生人

+   相对定向在进化上是坚固的，与恒定相对风险厌恶效用相比（这将是一个奇怪的巧合）

+   如皮质醇等胆固醇水平与地位相关，而不是财富

+   模仿通常比自己摸索更占优势（例如，火、微积分），导致以他人为导向的大脑

+   fMRI 识别出共情、社交信息的神经机制

+   社会地位是人类的普遍现象，贪婪则不是

+   人类普遍存在颠倒的支配等级（即，地位比财富更重要）

+   政治更关乎重新分配而非效率

**反驳：如果风险没有溢价，为什么要承担风险？**

+   所有男性中有 40%会繁衍后代，而女性中则有 80%。

+   男性拥有无价值期权，需要承担风险

+   为什么不承担无限风险？凡事适度。

+   生活是一个复杂、非线性、动态的游戏，其中每个参数都有局部最大值。辐射、维生素 A、氧气、耐受性、风险承担，都可能过度或不足。

## Silverlight Tools: Build for OS X!

在 Big Spaceship 的 Jamie 表达了大多数设计师所共鸣的观点 —— Expression Blend 需要一个 OS X 版本。VMWare 和 Parallels 可以工作，但迫使所有设计师安装 VM 仅仅是为了在他们在 OS X 上使用 Adobe 套件时运行 Blend，这是疯狂的。我知道微软想卖 Windows，但另一方面他们也需要让设计师容易地使用 Expression Blend。

“Silverlight 是 Flash 的一个值得竞争的对手。在其中的构建非常有趣。我推荐它。我认为在某些情况下，使用 Silverlight 构建某些东西会比使用 Flash 更快，反之亦然，而关键在于学习每个技术的优点和缺点。”

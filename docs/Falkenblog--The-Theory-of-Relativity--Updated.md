# Falkenblog: 相对论的理论，更新版

> 来源：[`falkenblog.blogspot.com/2009/11/theory-of-relativity-updated.html#0001-01-01`](http://falkenblog.blogspot.com/2009/11/theory-of-relativity-updated.html#0001-01-01)

观看该视频在 YouTube 上...该女人有这个有趣的理论。她开始问，你知道 H2O 是什么吗？你听说过爱因斯坦。基本上她指出

1) E=mc²

2) 由于质子、中子的直径约为 1E-15m，而原子的直径为 1E-11m。因此，星系中的所有质量都可以装进一个小球中。

3) 因此，质量很小，

我们可以忽略它

, 所以 E=c²。

这符合了她的理论，即我们都只是能量的存在，这不知何故就意味着顺势疗法是有效的。

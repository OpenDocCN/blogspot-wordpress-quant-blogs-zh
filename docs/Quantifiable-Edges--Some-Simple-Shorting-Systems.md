# Quantifiable Edges: 一些简单的做空系统

> 来源：[`quantifiableedges.blogspot.com/2009/04/some-simple-shorting-systems.html#0001-01-01`](http://quantifiableedges.blogspot.com/2009/04/some-simple-shorting-systems.html#0001-01-01)

当市场创新高时，往往会激发媒体的兴奋。标普周四收盘价为 2 月初以来最高水平。这样的消息由主播报道时听起来可能是积极的。但自 2002 年以来，当市场交易在其 200 日移动平均线以下时，创新高后强烈倾向于回调。以下是一些简单系统的结果。这些系统在标普 500 指数触及 X 日新高且低于 200 日均线时建议做空。当标普 500 指数下次收盘低于其 Y 日移动平均线时平仓。

上述所有系统在短期表现都非常出色。正如你所见，创新高并不是多头应该过于兴奋的事情。

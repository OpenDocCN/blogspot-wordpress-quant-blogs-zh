#### 不要逗留在派对中太久

两周前，我为投资者设置了一些悲观触发警报（参见

[市场还能走多远？](https://humblestudentofthemarkets.com/2023/11/19/how-far-can-this-rally-run-5/)

). 两个信号已被触发。

纽约证券交易所（NYSE）的麦克莱伦总和指数（McClellan Summation Index）的周随机指标（底部面板）已触及超买区域。未在图中显示的纳斯达克（NASDAQ）麦克莱伦总和指数也达到了类似的读数。

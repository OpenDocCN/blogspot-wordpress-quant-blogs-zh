# 物理视角：受伤情感的经济学家人

> 来源：[`physicsoffinance.blogspot.com/2014/02/bad-ideas-in-macroeconomics-non.html#0001-01-01`](http://physicsoffinance.blogspot.com/2014/02/bad-ideas-in-macroeconomics-non.html#0001-01-01)

卡特基·阿特雷亚是一位在里士满联邦储备银行工作的经济学家。他是现代宏观经济学方法的忠实信徒，并撰写了一本新书，书名为

[宏观经济学中的大思想：非技术性视角](http://www.amazon.com/Big-Ideas-Macroeconomics-Nontechnical-View/dp/0262019736)

我刚刚在谷歌图书上部分阅读了前 50 页或 so。

阿特雷亚对宏观经济学家自金融危机爆发以来所承受的所有批评感到非常恼火。他著名地

表达了他的恼火](http://market-ticker.denninger.net/uploads/2010/Jun/33655771-Economics-is-Hard.pdf)

几年前，他要求博客作者和非经济学家一般闭嘴，并将经济讨论留给他和其他领域的专家。看来新书的目的就是要纠正所有这些混乱，并向所有人解释为什么现代宏观经济学如此美妙、正确，且不受批评。我认为它不会成功。任何阅读他关于瓦尔拉斯清算所——一个解释经济如何达到瓦尔拉斯均衡的想象机制——部分的人都会很快得出结论，阿特雷亚所钟爱的经济运作图景基本上是数学幻想。

但实际上，我认为这本书可能会以这种方式提供很大的服务。任何认真阅读它的人都能亲自看出现代理性预期宏观经济学方法的脆弱和人为。我会订购这本书，因为从我所看到的，有很多东西可以学习，尽管可能不是关于真实经济是如何运作的。读者将体验到与主流宏观经济学对抗现实的态度和傲慢的近距离接触。从如此高度评价这一方法的作者那里，读者可以相信自己不会被某个无知的博客作者所欺骗，后者建立了稻草人论点（这是受伤的宏观经济学家常见的回应）。

大卫·格拉瑟已经阅读了这本书，并提供了

[一篇评论文章](http://uneasymoney.com/2014/02/03/big-ideas-in-macroeconomics-a-review/)

有一节值得强调，关于阿特雷亚倾向于简单地因为它们不适合他的首选建模方法而将整个真实经济现象领域斥为无关的：

> 正如阿特雷亚在第五章中所承认的，某些较老的宏观经济传统（包括凯恩斯主义和奥地利经济学派等）之间的一个重要区别是，宏观经济功能失调是协调失败的表现。瓦尔拉斯一般均衡的一个属性（即，帕累托最优）是，它实现了不同、自私的、竞争性个体代理之间的完美协调，以一种可能由全知和仁慈的中央计划者实现的方式完全调和他们的计划。瓦尔拉斯一般均衡完全解决了协调问题。在现代宏观经济学的重要结果依赖于假设现实生活中的经济可以合理地被描述为瓦尔拉斯均衡的情况下，现代宏观经济学假设协调失败与宏观经济学无关。只有在排除了宏观经济学中协调失败的范畴之后，使用代表性代理模型在宏观经济学中才变得合法，而在代表性代理模型中，协调失败等同于代表性代理的一种非理性形式。阿特雷亚将关于聚合水平的选择描述为现实主义和可处理性之间的权衡，但对我来说，现代宏观经济学并不是在现实主义和可处理性之间做出权衡，而是先验地决定协调问题不是宏观经济学的相关问题。
>

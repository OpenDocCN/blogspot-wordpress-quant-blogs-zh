# 市场学生：银行不会帮助多头

> 来源：[`humblestudentofthemarkets.blogspot.com/2011/09/no-help-from-banks-for-bulls.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2011/09/no-help-from-banks-for-bulls.html#0001-01-01)

我一直在写关于 KBW 银行指数（BKX）相对于市场的相对表现的文章，自

[五月](http://humblestudentofthemarkets.blogspot.com/2011/05/on-fence-watching-for-apocalypse.html)

我指出，BKX 相对于市场的相对突破是一个系统性风险上升的迹象，与俄罗斯危机或雷曼危机一致。自那时以来，确实发生了相对突破，而且情况变得更糟，糟糕得多。

尽管伯克希尔/BAC 交易半心半意地反弹，但银行相对于市场的相对趋势仍然向下。

直到最近，地区银行的业绩相对表现良好。尽管 BKX（严重权重系统性重要银行）在相对基础上已突破，但地区银行指数保持稳定。现在地区银行已经突破，并且难以在相对阻力之上反弹。

FT Alphaville 广泛报道了欧元区银行因资金市场基金从欧元区银行市场撤资而面临的短期融资问题（例如

[在这里](http://ftalphaville.ft.com/blog/2011/09/01/666451/peering-into-a-us-money-market-fund/)

)，现在看来，

[法国银行正面临麻烦](http://ftalphaville.ft.com/blog/2011/09/01/666396/un-petit-french-funding-probleme/)

由于它们严重依赖批发融资。

看看这张欧元 Stoxx 银行与欧元 Stoxx 指数的比率图表。欧洲银行处于相对下行趋势，并已突破了一条重要的相对支撑线。现在的相对比率创下了历史新低——这是一个麻烦的信号。

系统性风险的迹象正在上升。看看这张各种银行融资成本的图表。哎呀！

**你能避开多少颗子弹？**

综合所有这些情况，我解释这些条件如下：

+   **美国银行系统的系统性风险**：美国系统性重要银行正在发出系统性风险上升的信号。

+   **美国经济衰退的高风险**：美国地区银行的相对弱势预示着美国经济衰退的高风险。

+   **欧洲的雷曼/俄罗斯危机警告**：欧元区银行持续的短期融资问题以及该行业在欧洲的相对突破，都在预示着一个非常严重的问题。难怪 IMF Managing Director [Christine Lagarde](http://www.imf.org/external/np/speeches/2011/082711.htm)在杰克逊霍尔警告说，欧洲银行需要“紧急资本重组”。

也许美联储、欧洲央行、奥巴马政府、国会和欧盟可以团结起来，再次将问题踢到路上。

然而，现实中，你真的能躲过多少颗子弹呢？面对这些即将到来的风险，

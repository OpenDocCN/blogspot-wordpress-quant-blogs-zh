# Falkenblog：国有联邦！

> 来源：[`falkenblog.blogspot.com/2011/11/nationalize-fed.html#0001-01-01`](http://falkenblog.blogspot.com/2011/11/nationalize-fed.html#0001-01-01)

保罗·克鲁格曼的妻子，罗宾·韦尔斯，公布了一

[文章](http://ineteconomics.org/blog/inet/robin-wells-we-are-greg-mankiw%E2%80%A6-or-not)

争论占领华尔街的问题是一些经济学家应该解决的问题。我纳闷她是否认为生物学家都应该教授有关创世论的东西，因为有许多学生相信生命中有某种超越的火花。也许不会。我认为应该有不同教授主持的自由市场与 Keynesian 之间的课程辩论，因为这个问题始终存在，但我不认为 OWS 使这个问题更加紧迫。

也就是说，在研究生阶段，我们从未有任何教授对 Keynes 派/供给派为何正确进行过正式的演讲，尽管最终这是每个宏观经济学家都想知道的问题。就像他们认为他们可以超越这样的实际问题，专注于像汤姆·萨金特那样的技术问题一样。这是天真的，回避房间里的大象。这样的辩论或演讲不会产生好的问题集，他们通常会互相对话，但是对 24 岁的人来说把这样的问题吹到这个年纪这个问题还是需要很多人们去了解在行间阅读自己。此外，它会突出显示出演讲者需要回答的问题类型，最令人信服的论点，给对这个问题有所思考的年轻人们，这个阶段他们只是想支持看起来赢得竞选一方。

我觉得占领华尔街运动非常像青春期前的，就像当我 4 岁的女儿提出不一致或不可能的要求一样（的确，麦莉·赛勒斯有一个支持他们的视频）。例如，他们通常为没有明确目标而自豪，是草根而且有根本的不理解决定永远不会、不能反映每个人的偏好。一些灵魂确实创造了一个

[信仰维基](http://freenetworkmovement.org/commons/index.php?title=Liberty_Square_Blueprint)

,[这个运动]是我们能解决的最好的事情，其中包括'平衡'和'保护人权'。想要这样的事情很容易，并且此类权利是在

[1936 年苏联宪法](http://www.departments.bucknell.edu/russian/const/77cons02.html)

就在对乌克兰人进行种族灭绝之后，并在终结剩余的库拉克之前（遗憾的是权利冲突，而你总是会有优先权）。他们表现得好像卢梭的社会意志存在，每个人都与一些超人的灵魂共享对鼓圈和有机豆类的偏好。

我对 OWS 的一些要求项目感到兴奋：

+   将国际货币基金组织和世界银行转变为透明的、公有的和公开运营的实体

这些机构是民主政府创建的，主要由左派人士（我认识的选择这条路的朋友都是传统自由主义者）构成。它们被认为缺乏透明度和问责制，突显了现实永远无法创造他们所寻求的社会愿景，即在极其复杂的事务上达成某种透明的共识。你看，诺姆·乔姆斯基也是一样，他总是鼓励社会主义，但是每当社会主义出现时，比如在苏联、东欧集团或者柬埔寨，这些国家就不被认为是“真正”的社会主义国家（要是托洛茨基赢了多好！）。他们的幼稚信念行不通，因为它们是不可行的，同样的道理也适用于大银行可以成为一大笔经过协商的投资的饼干罐，而不是由专业人士管理。

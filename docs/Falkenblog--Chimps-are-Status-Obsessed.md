# Falkenblog: 黑猩猩痴迷于地位

> 来源：[`falkenblog.blogspot.com/2013/04/chimps-are-status-obsessed.html#0001-01-01`](http://falkenblog.blogspot.com/2013/04/chimps-are-status-obsessed.html#0001-01-01)

在这个

[TED 演讲](http://www.youtube.com/watch?v=uKByBgqxOw4&playnext=1&list=PL454A0274C1223C90&feature=results_main)

由 Colin Camerer 所著，他指出黑猩猩是人类最近的堂兄。他们也痴迷于地位。相比之下，财富的概念是完全

[对黑猩猩来说是陌生的](http://www.pri.kyoto-u.ac.jp/ai/intra_data/MatsuzawaT/The_chimpanzee_mind.pdf)

，因为财产权不是他们真正能处理的事情。

[`www.youtube.com/embed/uKByBgqxOw4?list=PL454A0274C1223C90`](http://www.youtube.com/embed/uKByBgqxOw4?list=PL454A0274C1223C90)

在我的书中，

[缺失的风险溢价](http://www.amazon.com/Missing-Risk-Premium-Eric-Falkenstein/dp/1470110970/ref=sr_1_1?s=books&ie=UTF8&qid=1345223074&sr=1-1)

，我认为根本性地位取向是为什么一般情况下没有风险溢价，以及为什么对低波动性资产的倒错需求没有被套利掉（基准风险阻止了这一点）。大多数（全部？）学者甚至没有承认这种解释来解释低波动性异常之类的事情。

经济学家假设主要动机是最好定义为贪婪的东西，而我认为嫉妒更重要。我不认为应该这样，那是一个

*规范*

问题，我只是说这是人们的行为方式。的确，我同意经济学家对自私的理解会是一个重大改进：短视的自私比长远的自私更糟糕，但它并不像嫉妒那样具有破坏性。遗憾的是，经济学家存在着以为每个人都像他们一样思考的标准偏见。此外，他们有一种偏见，即由于认为人们不那么嫉妒而更愿意理性自私，所以这一点必须是真的。

人类基因在大约 50000 年前就保持不变了，直到过去 350 年我们才真正改变了普通人的生活。当哥伦布到达这里时，美国大平原印第安人对财产权没有概念，这突显了社会的早期阶段，更像是我们的黑猩猩堂兄和他们赋予我们的本能。认为狩猎采集者更爱好和平，更与环境协调，更真实等等是一种美好的想法，而事实上他们是杀人狂，教条主义者，且痴迷于地位；他们缺乏贪婪并不是一种特征，而是一种错误。

告诉你的孩子，他们应该立志成为优秀的人，而不是伟大的人，因为伟大的人往往缺乏同情心，这是一个很好的做法。然而，我们不应该假设人们更贪婪而不是嫉妒，这很容易做到，因为大多数善于表达的黑猩猩把他们的嫉妒隐藏在利他主义之后，攻击百分之一的人被认为是想通过公平减少不公正。对于某些人来说，这是真的，但我怀疑对于绝大多数人来说，这并不是真的，他们只是想把那些高高在上的人打倒到与自己更接近的水平。

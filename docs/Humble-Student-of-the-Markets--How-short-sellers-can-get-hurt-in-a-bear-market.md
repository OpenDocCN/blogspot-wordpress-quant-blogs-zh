# 谦逊的市场学生：熊市中空头如何受伤

这是一个关于设定回报目标和风险控制的警示故事。常读者知道，虽然我的交易模型并非完美，但它对于摆动交易来说已经相当不错了。

这引发了我对设定回报目标和风险控制的看法。

# Humble Student of the Markets: 是时候骑上商品多头了

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/01/time-to-ride-commodity-bull.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/01/time-to-ride-commodity-bull.html#0001-01-01)

长期读者知道我是一个长期的商品多头。此外，我一直 write on the theme of

[全球复苏](http://humblestudentofthemarkets.blogspot.com/2012/01/global-healing.html)

几周以来。尽管上周美国 GDP 报告令人失望，但我看到迹象表明可能是时候让商品多头骑行了。情绪和动量指标都表明商品价格将再次上涨。

首先，情绪指标显示商品价格处于积累水平。这张来自美银/美林的 Mary Ann Bartels 的图表（注意：牛市和熊市的描绘是我的，不是她的）显示，大型投机者，主要是对冲基金，已经从拥挤的商品价格的多头位置撤离。图表是通过汇总 CRB 指数中所有交易所交易的商品的承诺数据报告生成的。

我对牛市和熊市的描绘显示，在熊市阶段，中性读数是做空的好时机。另一方面，在牛市阶段，中性读数是积累头寸的好机会。牛市和熊市阶段最好由黄金价格的先行指标图表来说明，该图表在 2002 年与其他商品复合体一起触底。

仅仅因为从这种中性信号来看是积累头寸的好时机，并不意味着商品价格没有更多的下跌空间。要看到一些短期上涨，你需要一个催化剂。

**CAT 的看涨指导是看涨催化剂？**

我提供了一个是观察市场方向的的关键指标之一是观察

[公司在 1 季度财报季节的管理层指导和肢体语言](http://humblestudentofthemarkets.blogspot.com/2012/01/global-healing-2012-vs-2009.html)

. 上周出现了一个看涨的催化剂，当时 Caterpillar 发表了一份看涨的指导报告，该公司是一家全球业务的周期性公司，

[报告并给出了非常乐观的指导](http://www.caterpillar.com/cda/files/3276340/7/Caterpillar+Inc.+4Q2011+Final.pdf)

:

> 我们预计世界经济增长的改善将增加对商品的需求。我们的前景假设大多数商品价格将在 2012 年略有上涨，并继续在鼓励投资的水平上。我们预计铜平均价格将超过每磅 4 美元，中部阿巴拉契亚煤炭价格将约为每吨 75 美元，西德克萨斯州中级原油价格将约为每桶 100 美元。

特别是，采矿业将在 2012 年成为增长来源，增长将如此之高，以至于供应将难以满足需求：

> 我们预计全球采矿业将持续强劲，我们的采矿设备订单积压很大。我们预计 2012 年销售将增长，并且正在为许多采矿产品增加生产能力。然而，我们预计 2012 年销售将受到产能的制约。

[WSJ](http://online.wsj.com/article/SB10001424052970204661604577186652609297254.html)

显示尽管上周的 GDP 报告令人失望，但美国经济仍在增长：

CAT 对美国住房的前景持乐观态度：

> 我们预计美国的总建筑支出，扣除通胀后自 2004 年以来一直在下降，将在 2012 年终于开始复苏。我们预测基础设施相关的建筑支出将增长 1.5%，非住宅建筑支出将增长 5%。我们预计 2012 年的住房开工至少为 70 万户，高于 2011 年的 60.7 万户。

他们对欧洲感到乐观，因为 ECB 支持欧元区：

> 欧元区的公共债务危机一直是一个持续的负面因素，但它不太可能触发全球性衰退。欧元区很可能至少有两个季度的经济增长疲软，可能为负增长，但应在 2012 年下半年开始改善。对于 2012 年，我们的展望假设欧元区的经济增长接近零，而整个欧洲的增长约为半个百分点。
> 
> 我们对 2012 年下半年欧洲经济增长的改善寄予厚望，这一期望建立在对欧洲中央银行（ECB）持续宽松政策的预期之上。最近，ECB 已经降低了利率，并且可能在 2012 年进一步降息。

CAT 还看到中国有足够的发展来支持建筑业的需求和商品增长：

> 中国在 2011 年底采取了第一次宽松政策，我们预计进一步的宽松政策可能即将出台。我们预计，2012 年中国经济将增长 8.5%，这足以支持建筑业增长和增加商品需求。

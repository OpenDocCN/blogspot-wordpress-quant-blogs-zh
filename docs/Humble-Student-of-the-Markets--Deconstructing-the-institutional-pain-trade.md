**机构和痛苦交易**

美银美林基金经理调查（FMS）是最有趣的调查之一，因为其频率固定（每月），范围广泛，并且历史悠久。对于不熟悉该调查的读者来说，它主要反映了具有全球投资授权的基金经理的观点。

在最新 FMS 结果的字里行间，我发现机构经理们为晚期周期通胀激增做好了准备，但他们开始减少风险，以应对增长放缓的预期。概括来说，机构经理们认为：

+   增长动力正在放缓，但是

+   通胀预期正在上升，但是

+   最终审判日，无论是衰退还是收益率曲线倒挂，都还遥遥无期。

基金经理们已经调整了他们的投资组合：

+   在大宗商品方面，我将其解读为为晚期周期通胀激增做准备，但是

+   他们通过出售股票来减少风险，

+   出售新兴市场（EM）的头寸，

+   购买债券，但是

+   投资组合的风险偏好仍然高于平均水平。

共识投资组合的亮点在于做多能源，做空美元，以及做空债券。这种分析非常投机，但如果我是市场之神，我想给市场参与者带来最大程度的痛苦，以下是我会做的事情。

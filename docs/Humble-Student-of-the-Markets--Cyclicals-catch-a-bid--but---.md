# 市场的谦逊学生：周期性行业获得买入机会，但是...

**周中市场更新**

：在过去的一周里，周期性行业出现了买入机会。这并不奇怪，因为它们相对于市场的表现非常糟糕。交通运输股表现出了令人印象深刻的强势，因为它们重新获得了相对支撑转为阻力水平。然而，所有其他行业的相对表现要么低于相对支撑，要么处于相对下跌趋势（零售商）。对周期性敏感的铜/黄金比率也有所反弹，但这是在广泛的横盘整理模式（底部面板）的背景下发生的。

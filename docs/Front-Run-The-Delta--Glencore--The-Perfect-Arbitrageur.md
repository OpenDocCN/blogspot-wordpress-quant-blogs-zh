# 领先对冲 Delta：嘉能可：完美套利者

> 来源：[`frontrunthedelta.blogspot.com/2011/06/glencore-perfect-arbitraguer.html#0001-01-01`](https://frontrunthedelta.blogspot.com/2011/06/glencore-perfect-arbitraguer.html#0001-01-01)
> 
> > > > > > > > > 嘉能可有限公司多年来一直令我着迷。我最初是通过科佩塔斯的《[金属人](http://www.amazon.com/gp/product/B000H2N5YY/ref=as_li_qf_sp_asin_tl?ie=UTF8&tag=frorunthedel-20&linkCode=as2&camp=217145&creative=399373&creativeASIN=B000H2N5YY)》了解了他们的存在，并自那时起一直试图在媒体和行业内关注他们的动态。在我眼中，他们以某种教派般的地位被视为最顶尖的人物。国际大宗商品交易中的 Maverick 和 Goose。虽然我认为他们的首次公开募股对他们来说是不幸的事件，但我很高兴能够获得他们的招股书。这是一本[厚厚的 1637 页巨著](http://www.scribd.com/doc/54605519/Glencore-IPO-Prospectus)，他们的招股书包含了大量关于策略、风险和风险测量，以及世界历史上最好的套利公司的内部运作信息。是的，世界历史上最好的套利公司。嘉能可的企业鼻祖包括荷兰东印度公司（[VOC](http://en.wikipedia.org/wiki/Dutch_East_India_Company)）和英国东印度公司（[EIC](http://en.wikipedia.org/wiki/East_India_Company)）。嘉能可是套利技术的典型案例：
> > > > > > > > > 
> > > > > > > > > > "嘉能可专注于最大程度地从整个供应链获得回报，考虑到其庞大的全球第三方供应基础、其物流、风险管理和营运资金融资能力、广泛的市场洞察力、业务多样性、其广泛的客户群、强劲的市场地位和规模经济。"
> > > > > > > > > > 
> > > > > > > > > > "嘉能可营销活动从第三方供应商和嘉能可拥有全部或部分股权的工业资产中获得多种不同的实物商品。这些商品通常附带增值服务，如运费、保险、融资和/或储存，并销售给广泛的消费者和工业产品最终用户，其中许多与嘉能可享有长期商业关系。"
> > > > > > > > > > 
> > > > > > > > > > **套利策略类型**
> > > > > > > > > > 
> > > > > > > > > > "Glencore 经营的许多大宗商品市场都是零散的，并且时而波动。因此，通常在不同形式、地理位置或时间段内，以考虑众多相关定价因素，包括货运和产品质量，在购买或销售大宗商品时会产生价格上的差异。*这些价格差异可以为 Glencore 提供套利机会，通过采购、运输、混合、*存储或以其他方式加工相关商品来获取利润*。尽管 Glencore 的各业务部门用于产生这种利润的策略因商品而异，但主要的套利策略通常可以描述为*地理、产品和时间相关*。”[斜体为作者所加]
> > > > > > > > > > 
> > > > > > > > > +   “地理：Glencore 利用其关系和生产、加工和物流能力，从一个地点采购实物商品并将其运送到另一个地点，那里可以获得更高的价格（减去运输和/或其他交易成本）；
> > > > > > > > > +   
> > > > > > > > > +   产品相关：利用所营销的特定商品的混合或多用特性，例如各种原油产品、煤或浓缩物，以供应比其基本成分价格更高的产品，或利用现有和/或预期价格差异；和
> > > > > > > > > +   
> > > > > > > > > +   时间相关：利用商品未来交付价格与即时交付价格之间的差异，当可用的存储、融资和其他相关成本直至未来日期低于远期定价差异时。”
> > > > > > > > > +   
> > > > > > > > > **营销活动类型**
> > > > > > > > > 
> > > > > > > > > > "Glencore 的营销活动可以按照重点进行分类，如下所示：
> > > > > > > > > > 
> > > > > > > > > > 基础供应链活动：Glencore 的主要营销活动是在其全球大宗商品采购和配送的日常业务中执行的，包括在全供应链上提供多种增值服务，并在经常出现的套利机会（如上所述）中寻求利润。
> > > > > > > > > > 
> > > > > > > > > > 事件驱动活动：Glencore 也寻求围绕事件或预期事件进行优化，例如突如其来的良好或恶劣的天气条件、交通瓶颈或故障（例如，火车脱轨）或劳动力或生产问题，其全球网络提醒其可能发生这种事件或这种事件发生的影响。这些类型的事件经常导致全球和/或地区性的‘市场紧缩’（即，市场供应不足以满足特定市场的需求），Glencore 的营销业务通常能够从这种市场条件中获得增强回报。当然，这些情况是不可预测的，但通常在任何特定年份都相当频繁地发生。
> > > > > > > > > > 
> > > > > > > > > > 价格风险活动：Glencore 还在其营销活动中进行一些持仓操作，利用其深厚的市场情报和分析寻求从特定大宗商品价格和/或价差变动中获利，例如，通过准确预期大宗商品价格的变动并以比以后更低的价格采购该大宗商品，或者以比以后更高的价格出售该大宗商品。Glencore 仅在符合其风险政策和限制的情况下进行价格风险活动，价格风险活动占 Glencore 整体营销业务的相当大一部分。
> > > > > > > > > > 
> > > > > > > > > > **物流**
> > > > > > > > > > 
> > > > > > > > > > "Glencore 利用其全球业务和在多个交易所和地理市场上交易的能力，利用各种交易所之间锌、铜、铅、银和金交易之间的套利机会。这在 2009 年和 2010 年特别重要，当时亚洲市场带动了大宗商品价格的复苏。"
> > > > > > > > > > 
> > > > > > > > > > "套利机会通常由全球和地区性价格波动和各种大宗商品市场的不稳定性增强，导致市场不平衡。Glencore 认为，在特定大宗商品或一组大宗商品的价格组成显示相对波动性和，因此，不确定性的时期，这些大宗商品的套利机会数量往往增加，特别是在市场紧缩期间（例如，当库存水平较低时）。这些市场条件提供了利用由此产生的保险费和折扣结构脱钩的额外机会。相反，在大宗商品价格波动性减少的时期，Glencore 的营销业务往往会有更少的盈利套利机会。相对平稳的营销环境通常会减少定价脱节，从而减少套利机会。相反，市场交投活跃度的减少会增加市场中保险费和折扣结构脱节，减少套利机会的数量。*（斜体为自己添加）*
> > > > > > > > > > 
> > > > > > > > > > "Glencore 的营销和工业投资活动得到了分布在 40 余个国家的全球 50 多个办事处的支持，包括欧洲、北美、中美、南美、独联体、亚洲、澳大利亚、非洲和中东地区。 Glencore 的主要办事处位于瑞士巴尔、康涅狄克州斯坦福、伦敦、鹿特丹、北京、莫斯科和新加坡。 这个网络为 Glencore 提供了重要的全球采购和分销能力。Glencore 的石油货运部门拥有 203 艘船舶的庞大和多样化船队，截至 2010 年 12 月 31 日，这些船舶是根据各种短期和长期租船和商业管理安排运营的，平均剩余的固定租期约为 3 年。总共，Glencore 在 41 艘船舶中拥有股权，它们已交付或目前正在建造，并预计将在 2012 年 3 月之前逐步交付。Glencore 的货运部门有助于其业务部门的市场营销成果，为其提供了即时获得船舶的能力， 这有助于在利用短期营销机会方面获得重大时间优势。*拥有自己的货运部门还使 Glencore 能更好地了解特定时刻的全球贸易流量。*由于内部需求和董事会对 Glencore 能够实现高于行业平均水平的船舶利用率的信心， Glencore 通常在船队租期长头寸，从而对现货运费率波动产生重大影响。然而，鉴于货运市场的性质和对避免所有货运价格风险的机会有限，Glencore 通常在未来的某种程度上暴露于长期租船的运费价格波动，否则将不受对冲，重新租给第三方或满足自己的市场需求。* [本文斜体为作者注]

> > > > > > > > > > **竞争对手**
> > > > > > > > > > 
> > > > > > > > > > "Glencore 的主要竞争对手是[Vitol Group](http://www.vitol.com/)、[Trafigura Group](http://www.trafigura.com/#kCcTViA57nuG)、[Mercuria Energy](http://www.mercuria.com/)和[Gunvor](http://www.gunvorgroup.com/)，它们基本上都是资产轻型（几乎没有上游生产）的商业模式。 Glencore 还面临来自银行（例如[Morgan Stanley](http://www.morganstanley.com/institutional/sales/fixed_income.html#comm)和[Goldman Sachs](http://info.goldavenue.com/info_site/in_who/in_who_aron.html)）的市场竞争，它们具有一些基础设施和没有当前的石油生产，尽管它们的绝大部分业务活动涉及衍生品，而不是石油的实际采购和分销。像 BP 和 Shell 这样的石油巨头所占据的市场份额也直接与 Glencore 的市场份额竞争，尽管它们的参与增加了市场的总量和流动性。"

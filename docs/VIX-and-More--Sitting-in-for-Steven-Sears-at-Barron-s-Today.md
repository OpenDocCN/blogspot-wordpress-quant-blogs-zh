# VIX 和更多：今天为 Steven Sears 在巴伦周刊写稿

> 来源：[`vixandmore.blogspot.com/2010/09/sitting-in-for-steven-sears-at-barrons.html#0001-01-01`](http://vixandmore.blogspot.com/2010/09/sitting-in-for-steven-sears-at-barrons.html#0001-01-01)

今天我是*The Striking Price*的嘉宾专栏作家，在[巴伦周刊](http://online.barrons.com/)上为 Steven Sears 代笔。

在[市场波动性是否会回归危机水平？](http://online.barrons.com/article/SB50001424052970204297404575493863568455090.html)一文中，我借机扩展了一些近期主题，特别是[VIX 期货](http://vixandmore.blogspot.com/search/label/VIX%20futures)的[期限结构](http://vixandmore.blogspot.com/search/label/term%20structure)创纪录的[正向市场](http://vixandmore.blogspot.com/search/label/contango)等，这由[VIX 期货正向指数](http://vixandmore.blogspot.com/search/label/VIX%20Futures%20Contango%20Index)的新高可以看出。

巴伦周刊的文章讨论了 VIX 升高的一些数学含义，将一些重要的数字放在历史背景下，并提供了对交易方法的启示。

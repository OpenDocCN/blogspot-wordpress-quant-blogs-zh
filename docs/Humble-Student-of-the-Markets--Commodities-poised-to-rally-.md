# 谦卑的市场学生：大宗商品准备上涨？

> 来源：[`humblestudentofthemarkets.blogspot.com/2013/05/commodities-poised-to-rally.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2013/05/commodities-poised-to-rally.html#0001-01-01)

美元上周经历的上行技术突破备受关注，但当我分析美元走强的因素时，发现突破很大程度上是由于日元的走弱。实际上，美元的其他组成部分都在测试支撑位，这表明欧元和大宗商品可能在这里上涨。

加拿大元也面临同样的情况。

实际上，如果你将 EURAUD 交叉汇率作为衡量欧洲强弱的指标，欧元正在对抗与大宗商品相关的澳元，因为它已经突破了一个阻力位，实现了向上的突破。从这个角度来看，欧元是否真的很弱，还是只是相对于美元而言显得弱？

正如我所展示的，除了日元之外，美元指数的其他大部分组成部分都处于技术支撑位。我还提到，要关注黄金近期底部的重新测试，作为市场方向的线索（见

所有这些条件都表明大宗商品即将反弹。欧元、与大宗商品相关的货币以及黄金都处于关键的技术支撑位。在我写下这些话的时候，贵金属价格大幅下跌。关注市场稳定或更好的反转迹象。如果出现这种情况，预计周期性部门的轮动将延续，股市将继续缓慢上涨。

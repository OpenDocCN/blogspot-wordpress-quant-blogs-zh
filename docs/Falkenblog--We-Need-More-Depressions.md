# Falkenblog: 我们需要更多的萧条

> 来源：[`falkenblog.blogspot.com/2009/03/we-need-more-depressions.html#0001-01-01`](http://falkenblog.blogspot.com/2009/03/we-need-more-depressions.html#0001-01-01)

这场危机的一个问题是，经济学基本上只有一个大萧条主导其数据。因此，我们就像是看着一个生病的病人的医生，只有一次与一个非常生病的人的经验。现在，如果那个人得了肺炎，而新患者得了其他疾病，你的治疗办法就没有用了。跨国研究是好的，但即使在那里，我们也只能得到少数可比性，而且所有都带有明显的不同之处，无法推广。正如 Russ Roberts 所说，‘这只不过是一个事件！’

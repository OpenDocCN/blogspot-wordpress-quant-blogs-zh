# Falkenblog：Bloggingheads 缩减规模

> 来源：[`falkenblog.blogspot.com/2011/11/bloggingheads-to-downsize.html#0001-01-01`](http://falkenblog.blogspot.com/2011/11/bloggingheads-to-downsize.html#0001-01-01)

Robert Wright 今天宣布他正在缩减规模

[Bloggingheads.tv](http://bloggingheads.tv/diavlogs/40107)

该网站部分原因是因为作为一个商业实体，它显然没有赚钱的潜力。他发现的基本问题是，他的网站试图让双方的人讨论问题，而不是更受欢迎的，即让同意的人彼此交谈（MSNBC，CNN，Fox）：

很遗憾，因为我是个大粉丝。更倾向于听取你同意的人的意见是合理的，因为你认为他们的说法比其他人更正确，所以我不认为这是一种恶意的均衡状态。但另一方面，我更喜欢在广播节目中听我不同意的人说话，因为不可避免地，主持人，尤其是来电者，会为他们的立场提出可怕的论据。当我的智力对手在讲话时，我更享受听到这个，而不是当有人为我同意的事情提出愚蠢的论点时。也就是说，许多（大多数？）我在税收，堕胎，平权行动，战争等方面同意的人，因为我认为他们同意我的原因不正确或不相关。

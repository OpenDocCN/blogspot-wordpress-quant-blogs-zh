# 市场学生：波动性突破？

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/05/volatility-breakout.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/05/volatility-breakout.html#0001-01-01)

本周我一直在看到期权市场的一些奇怪和矛盾的读数。一个尖叫“恐慌”，而另一个尖叫“自满”。

一方面，我周日晚上强调的股指看跌看涨比率（参见

[关注情绪反弹](http://humblestudentofthemarkets.blogspot.com/2014/05/watch-for-sentimental-rally.html)

)，保持高位，显示出高水平的恐慌。这种状况对股市来说是逆向看涨的。下图显示了股指看跌看涨比率的 5 天指数移动平均（蓝色）和 21 天移动平均（红色）。与它们自己的近期历史相比，两者都处于较高水平。

相比之下，VIX 及其期限结构传达的信息完全相反。下图顶部的图表显示了 VIX 期限结构，作为 VIX 与 VXV 的比率，或三个月预期波动性。当比率高于 1 时，意味着期限结构是向下倾斜的，期权市场显示出高水平的恐慌。相比之下，如果比率低，表明期限结构是急剧上升的，这是自满的迹象。

这是一个谜：当股指看跌看涨比率显示出高水平的恐慌时，为什么 VIX 结构处于自满区域？

**底部面板显示了 VIX 指数。如果市场如此恐慌，那么为什么这个所谓的“恐慌指数”如此之低？**

**体制变化？**

我曾写过关于可能的波动性体制变化（参见

[解读可能的波动性体制变化](http://humblestudentofthemarkets.blogspot.com/2014/04/interpreting-possible-volatility-regime.html)

)。在我之前的博客文章中，我展示了一张股指看跌看涨比率（顶部面板）和 VIX（底部面板）的图表。我展示了股指看跌看涨比率的趋势变化可能是更高波动性环境的先兆：

这种看跌看涨比率和 VIX 定价的二分法可能是那个过程的一部分？在这种情况下，相对于其近期历史的看跌看涨比率上升可能是波动性上升的信号，而不仅仅是恐慌情绪的上升。

**紧张的情绪**

确实，最新的 AAII 情绪读数（通过

[Bespoke](http://www.bespokeinvest.com/thinkbig/2014/5/15/bullish-sentiment-increases.html)

)可能是波动性体制变化的线索。尽管主要的大型股指数创下了新高，但本周 AAII 多头从 28.3%升至 33.1%，但与绝对水平相比，它们仍然很低，这通常会被解释为看涨。

然而，AAII 调查读数

[上周的](http://www.bespokeinvest.com/thinkbig/2014/5/8/fewer-bulls-than-bears.html)

也显示了多头和空头都非常低。

我认为这些条件表明市场非常波动。这些都是市场即将进入更高波动性环境的线索。

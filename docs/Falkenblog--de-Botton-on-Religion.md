# Falkenblog：德·波顿论宗教

> 来源：[`falkenblog.blogspot.com/2012/02/de-botton-on-religion.html#0001-01-01`](http://falkenblog.blogspot.com/2012/02/de-botton-on-religion.html#0001-01-01)

我对道金斯和希钦斯坚决的无神论感到困惑，原因是

[说得好](http://www.guardian.co.uk/culture/2012/jan/20/alain-de-botton-life-in-writing)

由英国作家阿兰·德·波顿撰写：

> 相反，他将他父亲的激进无神论与他认为是道金斯和克里斯托弗·希钦斯在他们畅销的攻击宗教中变得刻薄的困扰联系在一起。"我对此有一个世代理论。特别是如果你是 55 岁以上的男人，也许你在宗教手中遇到了不幸——你遇到了一个腐败的牧师，你在学校感到无聊，你的父母强迫你。年轻一代中很少有人有这种感觉。当我出生时——我今年 42 岁——宗教就成了一个笑话。

我的父母从未真正认真对待过宗教，我的许多朋友在我成长过程中也没有。所以，我对西方宗教的压迫如何并不感到激动，因为总是太容易选择退出。尽管如此，希钦斯最后还是给了道金斯一个采访

[据说更加强硬，](http://richarddawkins.net/articles/644473-be-strident-hitchens-tells-dawkins-in-his-final-interview)

而且让他的同僚团结一致。与此同时，道金斯在他去世之前与史蒂芬·杰伊·古尔德合谋，不与进化论的批评者辩论。这种防御性我根本无法同情。

# Falkenblog: Fannie Regulators Exacerbated the Bubble

> 来源：[`falkenblog.blogspot.com/2010/04/fannie-regulators-exacerbated-bubble.html#0001-01-01`](http://falkenblog.blogspot.com/2010/04/fannie-regulators-exacerbated-bubble.html#0001-01-01)

金融执行人员今天的自罪包含了房利美/房地美灾难中被过度支付的傀儡（1270 亿美元并且还在增加）。在 2005 年，房利美和房地美的主要监管机构正在解决与奖金相关的会计小丑闹剧。他们基本上是为了获得数百万美元的奖金而搬来搬去收益。由于房利美和房地美充斥着政治酬金（Jamie Gorelick, Bill Syron, Franklin Raines），这些位置是高级政府官员的主要闲职，国会感谢这提供了他们手下的吸引力。前国会议员已知会得到他们的丰厚咨询合同。而所有这一切都是由美国纳税人资助的。

提供给房利美和房地美的政府支持应该用于扩大低收入美国人的住房所有权，好像我们的明智官员们能够通过赚钱来完成这个任务一样。他们以为银行家更讨厌少数族裔，而不是喜欢赚钱吗？我相信这样的银行家可能存在，尽管我必须说我从未见过这样的银行家，并且很有信心他们和 Ku Klux Klan 一样代表了银行家（我相信极左派中的很多人会说--确实如此！原汁原味！）。

如果你阅读本文档，你将找不到任何与 Alt-A 抵押贷款、贷款标准、信用基础的基本内容有关。没有人在乎。正如法尔克所说，一切都很稳健。

更糟糕的是，他明确表明他的手是通过强调他的目标是服从于他的“使命”，而清除这一使命中的“不必要的干扰”是至关重要的。使命是什么？使住房所有权“更加负担得起”。中间目标是“购买符合条件的负担得起的住房

抵押贷款”。目标是什么？在历史上被忽视的社区中增加住房所有权。这可以通过什么来实现？降低信用标准。

这个计划中的错误前提是，历史上被忽视的少数群体之所以没有类似的住房所有权率，是出于任意的--也就是，种族主义的--原因，所以这一举措是有正面预期利润的。这是对事实假设的错误，是因为很多人认为制度性种族主义是非常真实和重要的。这种错误有着真实的后果。

这些就是监管机构。他们本应该关注这些政府支持企业的作为持续经营的可行性，但他们真正关心的是打击过去的火灾，这样他们就可以回到实现其慈善假象的目的，即创造一个创建顺从官僚，帮助他们维持权力的官僚资本垄断的借口。这个圈子圆满了。看起来，如果私营部门完全由政府批准的垄断组成，生活会很美好。不幸的是，垄断的本质是把人拒之门外，所以不是每个人都能得到高薪的闲职（比如，看看希腊）。

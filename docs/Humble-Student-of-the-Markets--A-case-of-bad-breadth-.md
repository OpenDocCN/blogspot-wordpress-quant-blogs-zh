# 《市场的谦卑学生：不良广度的案例？》

> 来源：[`humblestucentofthemarkets.blogspot.com/2014/05/a-case-of-bad-breadth.html#0001-01-01`](https://humblestucentofthemarkets.blogspot.com/2014/05/a-case-of-bad-breadth.html#0001-01-01)

在与技术分析师的众多讨论中，我听到很多人说，因为先进-衰退线继续创下新高，牛市仍然存在，股票价格可能会继续上涨。

正如下图所示，A-D 线（紫色）在修正之前并未总是给出了市场广度恶化的警告。特别是在 2011 年的修正中，A-D 线并没有发出任何警告。在 2012 年第一次修正期间，虽然 SPX 上涨，A-D 线也是横向移动，但在 2012 年第二次修正期间并未发出任何警告。

**衡量广度的另一种方式**

还有另一种考虑广度的方式。因为广度旨在衡量是将军（大盘股）领导市场，还是士兵（更广泛的市场）领导市场，因此一种简单的方法是分析等权重指数相对于市值加权指数的相对表现。如果等权重指数表现较好并且市场上涨，那么士兵正在领导进攻，这是牛市的；另一方面，如果市值加权指数表现较好，那么这是狭窄领导层的迹象，可能是市场广度恶化的警告信号。

考虑一下等权重 SPX 与市值加权 SPX 的这张图（显示为紫色）。在 2011 年和 2012 年的修正之前，等权重与市值加权比率恶化的时期被阴影区域表示。这些领先时间持续了四至九周不等。可以肯定的是，在 2012 年的第二个修正期间，这种广度指标并未发出麻烦的警告，事实上它正在上升。

现在我们看到等权重和市值加权的 SPX 之间出现了负的分歧。应该视为一个警告吗？

现在考虑一下等权重与市值加权的纳斯达克 100 指数的这张图。这个模式和等权重与市值加权的 SPX 图表是一样的。我们在 2011 年和 2012 年看到了相同类型的广度恶化，正好在市场疲软之前。现在即使在纳斯达克 100 指数上涨的情况下，这个广度指标也在下降，这是一个负的分歧。

简而言之，尽管 A-D 线表现良好，其他广度指标也发出警告信号。不要说你没有被警告过。

# 量化交易：量化交易员

> 来源：[`epchan.blogspot.com/2010/05/quants.html#0001-01-01`](http://epchan.blogspot.com/2010/05/quants.html#0001-01-01)

偶尔，一本为大众撰写的关于交易的书里，甚至对专业人士也有一些有用的精华。

[财富公式](http://www.amazon.com/dp/0809046377?tag=quantitativet-20&camp=14573&creative=327641&linkCode=as1&creativeASIN=0809046377&adid=08GKKTBWBX5F88C13T5D&)

是其中之一。它向我介绍了

[凯利公式](http://epchan.blogspot.com/2006/10/how-much-leverage-should-you-use.html)

，

[通用投资组合](http://epchan.blogspot.com/2007/01/universal-portfolios.html)

，以及复合增长率的最大化。

[量化交易员](http://www.amazon.com/dp/0307453375?tag=quantitativet-20&camp=14573&creative=327641&linkCode=as1&creativeASIN=0307453375&adid=1RH2ND78BHGVY4QMQ3ZJ&)

，由《华尔街日报》记者斯科特·帕特森撰写，是另一本。 （感谢我的合作伙伴史蒂夫告诉我这本书。）

在《量化交易员》中最重要的收获是什么？不，并不是你应该在希望取得巨大成功之前先学会成为大师级扑克或国际象棋选手，尽管考虑到 Patterson 对顶级量化交易员所玩的扑克游戏的详尽描述，你可能会这样认为。在我自己的专业熟人中，交易员兼扑克玩家仍然是少数。

最重要的收获是前员工对雷丁纳森技术公司所说的：“基金的成功没有秘密公式，不是几十年前由天才发现的神奇代码……相反，马达利恩[基金]的 90 名左右的博士团队正在不断改进基金系统……”

换句话说，尽管你可能没有 90 位博士供你差遣，你仍然可以不断改进/完善你的策略，改善你的交易环境的工程设计，增加你的策略多样性。尽管你可能仍然不能每年实现 60-70%的年化回报率，你仍然可以享受逐年稳定的回报。

顺便说一下，很高兴看到我的前同事拉利特·巴 ahl、文森特和斯蒂芬·德拉·皮特拉在书中提到，他们都在很多年前离开 IBM 加入雷丁纳森，他们都非常友好和亲切，与华尔街的常态形成鲜明对比。

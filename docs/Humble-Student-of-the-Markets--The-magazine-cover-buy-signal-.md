# 谦逊的市场学生：杂志封面买入信号？

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/05/magazine-cover-buy-signal.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/05/magazine-cover-buy-signal.html#0001-01-01)

我最近看到了战术交易底部的迹象（参见我上周的帖子

[在这里](http://humblestudentofthemarkets.blogspot.ca/2012/05/another-sign-of-short-term-bottom.html)

以及

[在这里](http://humblestudentofthemarkets.blogspot.ca/2012/05/merde-rally.html)

）。现在我们可能得到了杂志封面逆向买入信号，基于这样一个想法，当主流媒体开始撰写某个故事时，投资者从趋势中受益已经太晚了。很可能，一份大型新闻杂志封面吹嘘某种趋势，是这种趋势即将逆转的信号（例如，著名的

[股票之死](http://www.businessweek.com/investor/content/mar2009/pi20090310_263462.htm)

1979 年《时代》杂志的封面故事

[家，甜蜜的家](http://www.time.com/time/covers/0,16641,20050613,00.html)

2005 年的封面故事）。

现在德国《明镜》周刊，这家德国最负盛名的报纸之一，已经这么做了：

你甚至不需要懂德语就能理解封面的主旨。德国人已经接受了希腊退出欧元区的想法。这篇故事的英文标题是

[希腊退出欧元将是“令人遗憾的，但不是致命的"](http://www.spiegel.de/international/europe/european-leaders-warn-greece-may-have-to-leave-euro-zone-a-832464.html)

。

不多说了。

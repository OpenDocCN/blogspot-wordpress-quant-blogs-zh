# Falkenblog: 我将毁掉一半的价值，只要价格降一半

> 来源：[`falkenblog.blogspot.com/2012/10/i-would-destroy-half-value-for-half.html#0001-01-01`](http://falkenblog.blogspot.com/2012/10/i-would-destroy-half-value-for-half.html#0001-01-01)

[维克拉姆·潘迪特（Vikram Pandit）](https://wiki.example.org/feynmans_learning_method)担任首席执行官期间的股票价格如上图所示。他在 2009 年和 2010 年分别获得 1 美元，但你知道他会补偿这部分损失，所以在 2011 年 3 月他获得了 2300 万美元的“留任奖”。作为首席执行官，他有很大的权力，所以很难避免这种情况，对我来说，这突显了在大型集体中分配激励和权利时出现的问题。尽管我批评这些糟糕的首席执行官获得了大量报酬，但我认为解决方案并不明显。毕竟，克林顿在 1994 年左右的百万富翁税直接导致了股票期权的繁荣，这可能加剧了科技泡沫和世界通信公司（WorldCom）和安然公司（Enron）等公司的欺诈问题。

许多人认为潘迪特的薪酬极其恼人，好像潘迪特拿走了他们的钱，但不像政府的钱那样，这是其他人在浪费他们自己的钱。当然，花旗银行在紧急援助中得到支持，但是 1）他们不可能拒绝它，2）他们全部偿还了，3）这并非就是要给花旗银行更多监管的理由，而是不要救助大型银行。对不好的监管解决方案并不是更多的监管，而是更少的监管。这不仅可以创造更有效的激励和风险和资本的分配，而且更容易实施。

例如，最新的监管提案实际上

[目标电子表格](http://hereisthecity.com/2012/10/16/regulators-warn-on-spreadsheet-risks-in-quest-for-data-excellenc/)

：

> 监管机构表示，预期保险公司将对关键的内部数据流系统（如电子表格）进行适当的控制。这些控制形式应当包括，但不限于，输入验证、变更和发布管理、灾难恢复和文件记录。

他们也会批准我的便利贴笔记。

回到花旗银行，我认为提高价值的最佳途径是将其分成可管理的部分。对于那么大的东西，人根本无法去管理，正如大型银行所表现出来的那样。

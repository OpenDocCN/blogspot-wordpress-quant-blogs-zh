# 谦逊的市场学生：新脆弱五国出现更多裂痕

> 来源：[`humblestudentofthemarkets.blogspot.com/2018/10/more-cracks-appear-in-new-fragile-five.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2018/10/more-cracks-appear-in-new-fragile-five.html#0001-01-01)

经济衰退有助于解开过去扩张周期的过度行为。尽管目前美国经济衰退的可能性仍然相对较低（参见

[2020 年经济衰退？](https://humblestudentofthemarkets.com/2018/10/07/a-recession-in-2020/)

),并且经济中没有太多的过度行为，问题出现在美国国界之外。这次，大部分过度的私人债务积累发生在中国和加拿大。

我去年三月写过关于新脆弱五国的文章。

[洛米斯·赛勒斯](http://blog.loomissayles.com/cracks-appearing-in-five-highly-leveraged-economies)

为这些国家成为新脆弱五国（包括加拿大）提出了论据，基于不可持续的房地产泡沫：

> 五个高度杠杆化的经济体（加拿大、澳大利亚、挪威、瑞典和新西兰）开始出现裂痕。在全球金融危机之后的几年里，这五个国家都有一个共同的主题——多年的房地产繁荣，由低利率推动，导致家庭债务水平非常高。
> 
> 这种繁荣在所有五个市场中都开始消散。房价基本上已经改变了方向，无论是减缓增值还是直接下降。此外，在利率保持在或接近历史低点，劳动力市场继续强劲的情况下，这种情况也在发生。重要的是，这是一个许多人认为在這些国家的 otherwise 强劲经济增长背景下不可能发生的调整。但我们对房价持长期观点，并且几年前就开始强调这些市场的可负担性问题。

迹象显示那些房地产泡沫正在破灭。

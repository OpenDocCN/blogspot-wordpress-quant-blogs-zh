# Magmasystems Blog: Short Sell Any Time

> 来源：[`magmasystems.blogspot.com/2007/06/short-sell-any-time.html#0001-01-01`](http://magmasystems.blogspot.com/2007/06/short-sell-any-time.html#0001-01-01)

（感谢 Wall Street Folly）

证券交易委员会（SEC）周三以全票通过了最终废除烦人的、荒谬过时的“tick test”（点测试）。该测试只允许在最后成交价高于前一个价格时进行卖空。这意味着如果卖空前的股票价格下跌，交易员就不能进行卖空交易。

想知道这对市场上的所有模型意味着什么吗？（我的意思是 HFT 和算法模型，而不是吉赛尔·邦辰类型的模特...但，我相信海蒂·克鲁姆也很高兴！）

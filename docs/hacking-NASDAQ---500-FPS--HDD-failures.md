# 以 500 FPS 的速度黑客攻击纳斯达克：硬盘故障

> 来源：[`hackingnasdaq.blogspot.com/2014/01/hdd-failures.html#0001-01-01`](http://hackingnasdaq.blogspot.com/2014/01/hdd-failures.html#0001-01-01)

最新发布了一些关于真实世界硬盘故障率的精彩数据。作为一个消费者和储存大量 TB 数据的人来说，这种数据是无价的。完整的帖子可以在

[这里](http://blog.backblaze.com/2014/01/21/what-hard-drive-should-i-buy/)

我正在使用 WD 绿盘，配合 SSD 在 ZFS 上，因此用大容量的廉价硬盘和快速缓存/工作集（SSD）能得到物超所值。到目前为止，只损失了 1 个硬盘，幸运的是它是在烧录过程中失败的。

记住这些数据是针对普通用户级别的硬盘，而不是你典型的昂贵的 HP/DELL/IBM 服务器。

简而言之：希捷 barracuda 硬盘很差劲 - 小心。

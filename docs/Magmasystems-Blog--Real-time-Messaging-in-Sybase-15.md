# Magmasystems Blog: Real-time Messaging in Sybase 15

> 来源：[`magmasystems.blogspot.com/2009/02/real-time-messaging-in-sybase-15.html#0001-01-01`](http://magmasystems.blogspot.com/2009/02/real-time-messaging-in-sybase-15.html#0001-01-01)

如果你的组织正在从 Sybase ASE 12.5.4 迁移到 Sybase 15，那么你可能想看看

这些

幻灯片。

一张幻灯片引起了我的注意。这是一张关于

[eSpeed](http://www.bgcpartners.com/)

。eSpeed 在他们将 Sybase 迁移到其中一项工作是：

用集成实时消息传递（RTDS 4.0）替换 Sybase ASE 15 中的“轮询-广播”遗留解决方案

我需要了解更多关于这个信息。Sybase 有一些幻灯片

在这里](http://www.sybase.com/content/1030415/RTDS-SybaseTemplate.pdf)

。

许多使用 Sybase 的金融公司实际上每'n'分钟进行一次数据库轮询，以找出表中哪些行发生了变化。如果 eSpeed 做了我认为他们做的事情，那么这为自己迁移到 Sybase 15 提供了一个有力的理由。（当然，这假设你最终不会迁移到 SQL Server 2008/2010。）

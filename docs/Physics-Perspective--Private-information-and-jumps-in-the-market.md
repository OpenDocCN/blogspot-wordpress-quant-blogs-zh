# 物理视角：市场波动与私有信息

> 来源：[`physicsoffinance.blogspot.com/2011/10/private-information-and-jumps-in-market.html#0001-01-01`](http://physicsoffinance.blogspot.com/2011/10/private-information-and-jumps-in-market.html#0001-01-01)

跟随我的

[第二篇近期文章](http://physicsoffinance.blogspot.com/2011/10/what-moves-markets-part-ii.html)

关于是什么推动市场，两个读者发表了有趣且值得注意的评论，我想稍微探讨一下。我在帖子中提出了证据，许多大的市场波动似乎并不是与新闻形式的公共信息的突然到来有关。两个评论都指出，这可能忽略了另一种信息来源——通过交易者的行动带入市场的私有信息：

> 以及...
> 
> > 这两条评论都注意到，市场上发生的每一笔交易（至少很多交易）可能都在揭示交易者所持有的私有信息的一部分。原则上，可能是这样的私有信息冲击市场，导致了大的波动（如
> > 
> > [the work](http://arxiv.org/PS_cache/arxiv/pdf/0803/0803.1769v1.pdf)
> > 
> > of Joulin and colleagues).
> > 
> > 我认为在这方面有几个需要注意的事情。首先，虽然这是一个合理且可信的想法，但不应该过度延伸。显然，如果你仅仅
> > 
> > *assume*
> > 
> > 认为所有交易都携带关于基本面的信息，那么有效市场假说——在“价格对基本面的新信息作出反应”的意义上解释——本质上就成为正确的定义。毕竟，大家都同意交易推动市场。如果假设所有交易都揭示信息，那么我们只是简单地假设了有效市场假说的真实性。这是一个同义反复。
> > 
> > 更实际的是，将这个想法视为一个需要进一步检查的假设。当然，有些交易确实揭示了私有信息，比如一个对冲基金突然买入 X 并卖出 Y，这反映了基于研究的一种信念，即 Y 相对于 X 暂时被高估。同样，有些交易（如第一个评论中提到的）可能没有揭示任何信息，仅仅是由于与基础股票价值无关的原因而进行。由于我不知道有没有独立的方法来确定一个交易是否揭示了新的信息，我们被困在一个无法测试的假设中。
> > 
> > 但一些研究试图从另一个角度检查这个问题。再次，考虑大的价格波动——那些位于回报分布的肥尾部分。一种认为私有信息是原因的提议认为，大的价格波动主要是由像对冲基金、共同基金等大型投资者的较大成交量引起的。一些这样的交易可能揭示了新的信息，一些可能没有，但让我们假设现在大多数是这样的。在
> > 
> > [一篇论文](http://polymer.bu.edu/hes/articles/ggps03.pdf)
> > 
> > 在
> > 
> > *Nature*
> > 
> > 2003 年，Xavier Gabaix 和同事认为，你可以从显示共同基金的大小分布遵循类似幂律，指数为 1.05 的数据，来解释市场回报分布的幂律尾部的精确形式——它的指数非常接近 3。他们分析中的一个关键假设是，由交易量 V 产生的价格影响
> > 
> > Δp
> > 
> > 大致等于
> > 
> > Δp = kV^(1/2)
> > 
> > 。
> > 
> > 这种观点似乎支持这样的想法，即新私人信息的出现，通过大宗交易表达，可能会解释 Jouvin 研究中提到的无新闻的大幅跳跃。（这种透露的信息似乎不太可能导致像 1987 年的崩盘，或 2008 年的整体崩盘这样剧烈的市场动荡）。但从这个角度看，这些论点至少似乎与 EMH 观点一致，即许多大型市场运动反映基本面的变化。但再次，这假设所有或至少大多数大宗交易都是由基本面的私人信息驱动的，这可能并非如此。这项研究的研究人员自己并没有对大宗交易是否真的反映基本信息做出任何断言。相反，他们指出，...
> > 
> > > 这种理论，其中大型个体参与者推动市场，与股票市场运动难以用基本价值的变化来解释的证据一致...
> > > 
> > 但更近期的研究（
> > 
> > [这里](http://www.santafe.edu/media/workingpapers/04-02-006.pdf)
> > 
> > 和
> > 
> > [这里](http://arxiv.org/PS_cache/arxiv/pdf/0803/0803.1769v1.pdf)
> > 
> > ，例如）表明这种解释并不太吻合，因为大回报与大宗交易之间假设的关系是不正确的。这项分析相当技术化，但基于对纳斯达克每分钟交易的研宄，表明如果你只考虑极端回报或极端成交量，回报与成交量之间没有普遍的相关性。早期研究中的相关性假设可能在平均上大致正确，但在极端事件上并不成立。"作者总结说，"大跳跃"并非由大宗交易量引起。
> > 
> > 实际上，正如这些后期研究作者所指出的，拥有宝贵私人信息的人不希望这些信息立即以一大块的形式透露出来，因为这将带来市场的负面影响（迫使价格与他们相反移动）。1985 年阿尔伯特·凯 yle 所写的一篇广为人知的论文显示，一个拥有宝贵私人信息、交易最优化的知情交易员如何将他的交易隐藏在噪声大、无信息的交易背后，假设这种交易存在。这可能在实际中有点过于难以置信，但大宗交易确实通常会被分解成许多小交易，以此最小化影响。
> > 
> > 总的来说，那么，似乎我们得出的结论是，公共或私人新闻确实解释了
> > 
> > *一些*
> > 
> > 大的价格波动无法合理地解释所有情况。还有其他因素。重要的是，再次考虑这对有效市场假说（EMH）的最有意义含义意味着什么，我认为这是指市场价格能够相当准确地反映基本价值（因为它们吸收了所有相关信息并正确处理了这些信息）。证据表明，价格经常在没有新信息的情况下出现大幅波动，并且价格可能因此被推动远离基本价值。
> > 
> > 后者的论文确实提出了另一个机制作为例行大型市场波动的驱动因素。这是一个更加机械的过程，围绕订单簿中订单的自然动态。我会在另外的时间详细探讨这个。现在，只是简单地介绍一下
> > 
> > [这篇论文](http://arxiv.org/PS_cache/arxiv/pdf/0803/0803.1769v1.pdf)
> > 
> > ，描述了关键思想：
> > 
> > > 那么，还有什么需要解释的看似自发的大的价格跳跃呢？我们认为解释来自于市场即使在“流动性”的情况下，也处于流动性消失的领域，因此处于自组织临界状态[31]。在电子市场上，订单簿中可用的总成交量在任何时刻都是股票市值的一小部分，比如说 10^(-5) - 10^(-4)（例如[15]）。流动性提供者承担被“挑选”的风险，即在大幅上涨之前卖出，或者相反，因此他们会非常谨慎地放置限价订单，并且一旦出现不确定性信号就会取消这些订单。这种信号可能仅仅是由于订单流的自然波动引起的，这可能导致在某些情况下流动性灾难性的下降，从而引起价格跳跃。实际上确实有证据表明，大的价格跳跃是由于局部流动性枯竭引起的。

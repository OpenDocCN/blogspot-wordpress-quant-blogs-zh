# Falkenblog: 宽容也可能是一种侮辱

> 来源：[`falkenblog.blogspot.com/2011/01/tolerance-can-be-insulting.html#0001-01-01`](http://falkenblog.blogspot.com/2011/01/tolerance-can-be-insulting.html#0001-01-01)

在 Bloggingheads.tv 上，心理学家 Joshua Knobe 提出了这样一个观点：如果有人对你在某个重要问题上表现出宽容，那就意味着他们基本上不在乎你。这凸显了美德，就像维生素和氧气一样，可能过多或过少。

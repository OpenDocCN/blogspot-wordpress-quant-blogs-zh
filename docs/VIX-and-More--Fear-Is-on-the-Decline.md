# VIX 和更多：恐慌正在下降

> 来源：[`vixandmore.blogspot.com/2008/10/fear-is-on-decline.html#0001-01-01`](http://vixandmore.blogspot.com/2008/10/fear-is-on-decline.html#0001-01-01)

当前 VIX 为 77.13，这似乎是谈论恐慌开始离开市场的一个奇怪时机。

当我打下这些字的时候，VIX 下跌了 2.00 点，SPX 下跌了大约 15 点。VIX 和 SPX 都在今早下跌，这很不寻常，但也不是闻所未闻。更值得注意的是，这种情况发生在周一，而周一 VIX 通常会“跳涨”以反映“[日历反转](http://vixandmore.blogspot.com/search/label/calendar%20reversion)”效应，即根据日历来定价 VIX，但只在交易日内计算。简而言之，市场制造商倾向于在周五降低期权价格，以预期即将到来的周末，并在周一提高价格。

现在我们已经连续两个周一没有出现让许多投资者感到恐慌的世界末日般的头条新闻了。出于这个原因以及其他原因，市场似乎今天早上显得不那么兴奋，VIX 终于开始感受到重力的作用。

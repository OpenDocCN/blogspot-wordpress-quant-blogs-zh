# New Position | CSSA

> 来源：[`cssanalytics.wordpress.com/2012/09/20/new-position/#0001-01-01`](https://cssanalytics.wordpress.com/2012/09/20/new-position/#0001-01-01)

我最近加入了一家新公司——[Flexible Plan Investments](https://www.flexibleplan.com/public/home.aspx)，担任经济研究与战略发展副总裁。Flexible Plan 是由 Jerry Wagner 创立的，他是 NAAIM 的联合创始人，也是[Wagner Award for Active Management](http://www.naaim.org/wp-content/uploads/2012/07/NAAIM-call-for-papers-13_draft1.pdf)奖项的创造者。Jerry 是行业先驱，他使绝对收益策略广泛适用于单独管理的账户的投资顾问。他还是战略多元化的主要倡导者：结合各种不相关的、 actively managed 策略，以实现跨市场周期的更稳定的回报。我对这个机会感到兴奋，并期待在研究方面做出贡献。今后我将在博客上更活跃，分享许多新的量化研究方向和思路。

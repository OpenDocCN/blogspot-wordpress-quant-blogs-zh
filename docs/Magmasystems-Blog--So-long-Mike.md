# Magmasystems 博客：再见，迈克

> 来源：[`magmasystems.blogspot.com/2006/11/so-long-mike.html#0001-01-01`](http://magmasystems.blogspot.com/2006/11/so-long-mike.html#0001-01-01)

我们的朋友迈克是一家大型咨询公司金融服务部门的 25 年老兵。他刚刚得知自己将被解雇，原因是他没有达到（不切实际的）销售指标。该部门的全新管理层和迈克在事情上意见不一致，而摆脱一个人的简单方法就是设定他失败。

迈克很快就能找到一个新的职位，因为他在金融服务领域享有盛誉并受到尊敬。而且，机遇也往往是在逆境中产生的。

了解当你被设定要失败时的预警信号。迈克一眼就看到了，并且做好了准备。

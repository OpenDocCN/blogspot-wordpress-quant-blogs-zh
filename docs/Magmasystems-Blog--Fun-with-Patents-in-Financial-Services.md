# Magmasystems 博客：在金融服务中的专利乐趣

> 来源：[`magmasystems.blogspot.com/2007/06/fun-with-patents-in-financial-services.html#0001-01-01`](http://magmasystems.blogspot.com/2007/06/fun-with-patents-in-financial-services.html#0001-01-01)

浏览美国专利局很有趣，看看你的同事在金融行业中是如何创造的。看看谁取得了专利更有趣。在搜索页面上，在“受让人”领域键入您最喜欢的公司。即使你我可以创造一项伟大的技术，*也*可能被列为发明家，我们通常也需要将工作分配给我们工作的公司。

# Falkenblog: 我的书明显且错误

> 来源：[`falkenblog.blogspot.com/2009/07/my-book-is-obvious-and-wrong.html#0001-01-01`](http://falkenblog.blogspot.com/2009/07/my-book-is-obvious-and-wrong.html#0001-01-01)

我还没有收到很多公众反馈，

[我的书](http://www.amazon.com/Finding-Alpha-Search-Return-Finance/dp/0470445904/ref=pd_rhf_p_t_1)

，但私下的反馈相当循环论证。一方面，有人说我的发现是错误的。有一位愤怒的经济学家认为我“在说地球是平的”。我的实证发现不够严谨，因为我使用了错误的统计测试，如果我用更强大的测试，我就不会拒绝风险解释回报这一标准理论所指示的假设。实际上，我引用的支持文章中许多都留下了这种可能性的空间。我承认这是可能的。任何事都是可能的。只是不太可能。当前的范式为每个资产类别（货币、横截面股票、债券各有一个）找到局部风险因素，却没有直觉。此外，任何测试的复杂性增加都会降低其统计功效，如果理论有点用，那么“未能拒绝”会给人带来安慰。这个理论糟透了，它主要是符号错了（在任何资产类别中，一般风险较高的资产回报低于平均水平！）

另一方面，我得到的评论是“这很明显”，“太简单了”，或者“大家都知道”。我也认为这很明显且简单，但并非金融界的每个人都明白：1) 风险和回报通常在任何明显的方面都不相关；2) 如果人们有相对地位取向，这是一个均衡结果。说出一些新、真、重要且明显简单的事情，比

既要新颖、真实、重要。但是那些认为这显而易见且真实的人往往是文献中的新人，因此他们没有被训练去看清直接呈现在眼前的东西。专家发展出了特殊的“理论”透镜来观察现实，时间一长，没有它就看不清了。他们就像那些据说无法看到哥伦布的船在地平线上出现的美国土著，因为他们的脑子被硬性连接起来看不到不可能的东西。

但是，新手们看到我提供的数据，这主要是对文献和样本平均值的调查，并认为这是明显的（见 SSRN 证据总结

[这里](http://papers.ssrn.com/sol3/papers.cfm?abstract_id=1420356)

那些在这个领域多年的人认为这不真实，而那些没有将其作为首要关注点的人，则认为这是显而易见且微不足道的。我对我的想法在这一点上的未来感到乐观，因为正如他们所说，科学的进步是一次葬礼接着一次。

但后来我注意到两位亚马逊评论者认为这对普通投资者没有任何影响。让我再试一次，因为显然任何足够聪明能买我的书并阅读的人都不傻，这一切都是我的错。

在书中，我探讨了“寻找 alpha”的策略，这些策略基于一个事实：好的策略往往具有很高的局限性，大多数人在这方面都会失败，而且人们会对自己正在做的事情说谎。但是，从 30,000 英尺的高空来看，实际投资的含义非常明显：相对购买“低风险”股票。这可以通过多种方式实现：在保持总贝塔值为 1.0 的同时，最大化夏普比率，或者最大化信息比率。所有这些相对于标普 500 被动基准的改进，其幅度都要大于从主动投资转向被动股票投资所带来的提升。只要资本资产定价模型（CAPM）不适用，我认为它不适用，因为大多数投资者都在对标基准，这就暗示了这种基本关注低波动性、避免高波动性的明显投资策略。以下是我的讨论内容

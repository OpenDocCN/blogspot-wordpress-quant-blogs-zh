# 金融视角：金融时报数学能力检查

> 来源：[`physicsoffinance.blogspot.com/2011/09/financial-times-numeracy-check.html#0001-01-01`](http://physicsoffinance.blogspot.com/2011/09/financial-times-numeracy-check.html#0001-01-01)

[这篇文章](http://www.ft.com/cms/s/0/19c46aca-e9ad-11e0-adbf-00144feab49a.html)

来自金融时报的视角不幸地是金融媒体的一个典型（是的，不仅仅是

关键词：金融

媒体）。只需思考一下以下段落的合理性，这段话评论了由欧盟委员会总统何塞·曼努埃尔·巴罗佐提出对金融交易征税的提案：

> 巴罗佐先生没有透露他的计划的细节，只是说它能筹集一些 550 亿欧元。然而，委员会进行的一项研究发现，这种税收还可能通过 0.53%到 1.76%的地区国内生产总值对长期经济增长造成影响。

文章没有提到是谁进行了这项研究，也没有给出链接。但还有更糟糕的。如果准确报道，似乎欧盟委员会的经济学家——或者他们请来做提到这项研究的人——实际上认为 0.53 中的“3”和 1.76 中的“6”意味着什么。在经济增长谈论这样的精确度是相当令人印象深刻的。在大不确定的时期。

我敢打赌，他们对结果的信心更准确的表述可能是，比如说，粗略地在 0 到 2%之间，或者甚至可能是-1 到 3%。但这样做就意味着没有人对未来的事情有丝毫的确定性，而这并不是通常的做法。

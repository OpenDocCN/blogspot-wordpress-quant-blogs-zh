# 谦卑的市场学生：希腊是如此……去年的事情

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/02/greece-is-solast-year.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/02/greece-is-solast-year.html#0001-01-01)

当市场先生等待关于希腊是否会实施某项措施的新闻时，他告诉我，担忧希腊是如此……去年的事情。尽管所有关于希腊可能违约的焦虑，但希腊股市在过去几周一直在上涨，并超过了欧元 STOXX 50 的表现。

欧元区的担忧已经转移到葡萄牙，其股票表现一直大幅不佳。

对于多头来说，一切尚未失去。葡萄牙市场正在发起战术性反弹，并测试其下降趋势线。

此外，

[葡萄牙 2 年期收益率](http://www.bloomberg.com/apps/quote?ticker=GSPT2YR:IND)

，曾一度飙升至 20%以上，现在正在回落，或许表明市场先生预期将从欧洲央行的 LTRO2 计划中获得进一步缓解。

这就是为什么我对风险交易相对乐观。如果这就是市场先生所担忧的最糟糕的事情，那么是时候买入并长期持有了。

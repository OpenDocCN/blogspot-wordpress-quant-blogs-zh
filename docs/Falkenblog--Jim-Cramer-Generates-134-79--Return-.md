+   模型投资组合胜过标普 500：134.79%的平均总回报

+   P.S. 我只能给您延长[此优惠](http://secure2.thestreet.com/cap/prm.do?OID=012904&cm_cat=RM&cm_ite=012904&cm_ven=EMAIL)48 小时，所以请不要拖延... 当您享受我们 129.95 美元的优惠时，您绝对不会失去任何东西。

请注意，他的实例股票选择的平均回报远高于任何人的低门槛，回报率为 30%至 134.79%（爱呢）。当人们在网上以 50 美元出售男性增大丸时，它们是愚蠢的，而且不是很多钱，但正如约翰·斯图尔特指出的那样，股市不是一场游戏。这是非常可耻的，CNBC 应该意识到这使他们成为他的骗局的一部分。30%以上的回报根本不可能是代表性的，他们知道这一点，当然也会说他们的意思不是每一次的回报都这么高，但这就像彩票广告说“任何人都有可能赢” - 这是足够准确的，但极其误导。

就股票推荐而言，我指出，与标准资产定价理论相比，所有股票推荐都承诺超过平均水平的回报。理论上，应该有一半的股票被推荐以低于平均水平的回报，因为它们具有良好的“风险调整”回报，但这并未发生。这种情况从未发生，突显了标准资产定价理论对风险承担的严重误解。人们只愿意冒风险去超越，永远不愿意以更大的安全性获得较低的回报（不包括现金类资产持有，如 AAA 债券或短期票据）。

吉姆·克雷默（Jim Cramer）现在是互联网的行骗者。他对公司细节的博学令人印象深刻，他确实在过去十年里经营了一个成功的对冲基金，尽管作为一个对冲基金，他的业绩声称没有经过审计，并且不可信（

[十年来有 24%的年回报和 1 个季度下跌](http://falkenblog.blogspot.com/2008/12/when-investors-dont-ask-questions.html)

但出于某种奇怪的原因，像许多著名的投资者一样，他很快意识到销售建议比投资更有利可图。如果他有像他所声称的边际好处（超额收益 5％？），那么出售这些建议而不是投资，尤其是当你有资金来投资时，就根本毫无意义。

所以现在他用来在上个世纪 90 年代进行高抛低吸的同一公司（CNBC）现在用来播出他的非常受欢迎的节目《疯狂金钱》（Mad Money），说明最好的推销员造就了对体验免疫和乐意上当的受害人。请注意，克雷默

# 鹰部落博客：我住在伊甸园

> 来源：[`falkenblog.blogspot.com/2010/07/i-live-in-eden.html#0001-01-01`](http://falkenblog.blogspot.com/2010/07/i-live-in-eden.html#0001-01-01)

![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgk6uj7B9BbT0mybQyLSCphKXCUKWJRxIAsvje7iJR9hXdOzWdtBZJ93ssC_s_de0rf3JKhZbfyuhT7ZhxQbnlQE6ym2xSmlNe9oti7O94sB05DJzwSJXiOBurMzPX4TmYIRmbYYw/s1600/gar_eden.jpg)[伊甸园](http://maps.google.com/maps?q=eden+prairie+Minnesota&um=1&ie=UTF-8&hq=&hnear=Eden+Prairie,+MN&gl=us&ei=mbY7TPbMDML_lgfyrKX7BQ&sa=X&oi=geocode_result&ct=image&resnum=5&ved=0CEkQ8gEwBA)

明尼苏达州，而且这是美国整个地方最好的居住地

[根据《财经》杂志](http://www.huffingtonpost.com/2010/07/12/money-magazine-best-place_n_643484.html)

当然，我所在地区还有更好的社区，但其中一项标准是不能太贵，以至于普通《财经》杂志读者无法住在那里（他们称之为‘过于昂贵的’住房）。

这是一个不错的郊区，提供了很多选择，但我住过其他一样好的郊区（新泽西州西菲尔德，俄亥俄州索伦）。

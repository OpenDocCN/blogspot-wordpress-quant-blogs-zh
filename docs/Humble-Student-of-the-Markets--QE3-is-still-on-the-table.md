# 谦逊的市场学生：量化宽松 3 仍在桌面上

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/02/qe3-is-still-on-table.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/02/qe3-is-still-on-table.html#0001-01-01)

在周五非农就业(NFP)发布之后，有些噪音认为这会降低美联储实施量化宽松 3 的可能性。我不这么认为。

根据其

2012 年 1 月会议后的[声明](http://www.federalreserve.gov/newsevents/press/monetary/20120125c.htm)

美联储有 2%的通胀目标，但没有就业目标[强调添加]:

> 长期通胀率主要由货币政策决定，因此委员会有能力设定长期的通胀目标。***委员会认为，以个人消费支出价格指数的年变化率衡量的 2%的通胀率，与美联储的法定任务最为一致。*** 将这一通胀目标明确地传达给公众，有助于保持长期通胀预期稳定，从而促进价格稳定和适度长期利率，增强委员会在面对重大经济干扰时促进最大就业的能力。
> 
> 就业的最大水平主要由影响劳动力市场结构和动态的非货币因素决定。这些因素可能会随时间变化，并且可能无法直接衡量。因此，***指定一个固定的就业目标是不合适的***；相反，委员会的政策决策必须由对就业最大水平的评估来指导，认识到这样的评估必然是不确定和可修订的。委员会在做出这些评估时考虑了广泛的指标。关于委员会成员对长期产出增长和失业的正常率的估计，每年在 FOMC 的经济预测摘要中公布四次。

[查尔斯·施瓦布](http://www.schwab.com/public/schwab/resource_center/expert_insight/investing_strategies/bonds/schwab_bond_insights_the_transparent_fed.html)

最近发布的一份研究笔记表明，美联储正在关注核心 PCE 衡量的通胀，因为它比 CPI 权重更低[强调添加]:

> 美联储还历史上首次宣布了明确的 2%通胀目标。这一明确的通胀目标也有助于减少关于政策长期的不确定性。美联储将使用基于个人消费支出（PCE）的变化作为衡量标准的 2%年目标。在最新数据中，PCE 的年同比增幅为 1.8%。所以它们仍然略低于这些目标。在会议后的新闻发布会上，伯南克被问到：“为什么是 PCE 而不是消费者价格指数？”**一个原因是，在 CPI 中，住房的权重更大。在房地产泡沫期间，它似乎低估了通胀，而现在由于租房比买房更受欢迎，它可能高估了通胀。PCE 还可以更灵活地调整以适应不断变化的消费模式。**美联储的批评者也可能认为，PCE 的年增幅通常也低于 CPI 的变化。

记住，在美联储的新透明度计划下，我们不再需要猜测美联储将要做什么，他们已经揭示了他们的方法。考虑这些来自

[达拉斯联邦储备银行](http://www.dallasfed.org/data/pce/index.html)

对于 PCE、核心 PCE 和修剪均值 PCE，这是排除通胀率更波动成分的另一种技术。

无论从哪个角度看，核心 PCE 和修剪均值 PCE 都固执地保持在低位，并低于 2%的目标。这让我认为，美联储认为它还有更多的刺激空间，而不会引发通胀螺旋。关注这一系列数据，以获得 QE3 可能会从桌面上移开的线索，不要为就业报告等经济发布的周期性反弹迹象感到焦虑。

伯南克对策（Bernanke Put）至少现在还存在。

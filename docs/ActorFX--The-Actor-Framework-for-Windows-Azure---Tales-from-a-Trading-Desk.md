## ActorFX: The Actor Framework for Windows Azure

很高兴看到来自[MS Open Tech](http://www.codeplex.com/site/users/view/MSOpenTech)的一个用于帮助编写 Azure 云应用程序的 Actor [framework](http://blogs.msdn.com/b/interoperability/archive/2012/12/05/ms-open-tech-releases-a-new-project-actor-framework-how-to-lower-the-barrier-to-entry-for-cloud-computing-act-i.aspx)。 代码存放在[CodePlex](https://actorfx.codeplex.com/)。 今年早些时候 Channel9 有一个[视频](http://channel9.msdn.com/Shows/Going+Deep/Hewitt-Meijer-and-Szyperski-The-Actor-Model-everything-you-wanted-to-know-but-were-afraid-to-ask)值得一看的关于 Actors 的节目 - Hewitt, Meijer and Szyperski: The Actor Model (everything you wanted to know, but were afraid to ask)。

很有趣看到 ActorFx 随着时间的推移如何演变，以及它是否能赶上 Java VM 堆栈中的[akka](http://akka.io/)。

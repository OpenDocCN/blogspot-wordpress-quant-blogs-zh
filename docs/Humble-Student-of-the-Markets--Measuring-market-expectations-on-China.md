# 《市场的谦虚学生：测量中国市场预期》

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/05/measuring-market-expectations-on-china.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/05/measuring-market-expectations-on-china.html#0001-01-01)

中国的消息仍然是喜忧参半的。一方面，

[汇丰中国制造业采购经理人指数](http://www.businessinsider.com/may-hsbc-china-flash-pmi-2014-5)

以 49.7 的数据超出预期，为五个月来的最高水平，表明形势正在稳定，但仍低于 50，表明经济仍在萎缩中。

另一方面，

[汤姆·奥利克](https://twitter.com/TomOrlik/status/469464652003696640/photo/1)

指出，李克强指数持续下降，这表明经济正在放缓。

**测量市场预期**

而不是试图自己预测，我想应该问问大市场，他对中国发生的事情有什么看法。对于长期视角，我测量了几个股票市场指数相对于 MSCI 全球所有国家指数（ACWI）的相对表现。所有回报均为总回报，并以美元计量，因此我们在一个货币单位中进行了苹果对苹果的比较。我测量的国家和地区有：

+   美国（SPY）

+   欧元区（FEZ）

+   资源市场，因其对世界增长和中国需求都很敏感：加拿大（EWC）、澳大利亚（EWA）和南非（EZA）以等权重计算。

+   大中华区，定义为中国（FXI）、香港（EWH）、台湾（EWT）和韩国（EWY），这样可以更好地衡量中国，而不仅仅是分析 FXI，这是一个相对狭窄的指数。

正如图表所示，自 2009 年中期以来，美国市场一直表现优异，超越了 ACWI，而欧元区股票一直在下降，但在 2012 年中期扭转了局面。大中华地区自 2009 年中期以来一直处于相对下降趋势，表明动力正在丧失。资源市场对全球增长和中国需求都很敏感，自 2012 年以来一直处于相对下降趋势。

如果我们进一步放大，下一个图表显示了资源篮子相对于大中华篮子和旧中国（FXI）相对于新中国（FXI）的相对表现，后者在金融领域具有较大的权重，而新中国（PGJ）则在以消费者为导向的互联网和技术领域具有较大的权重（参见我之前的帖子

[新中国与旧中国的配对交易](http://humblestudentofthemarkets.blogspot.com/2014/03/a-new-china-vs-old-china-pair-trade.html)

）。

当线条上升时，中国的增长预期由信贷驱动的基础设施增长引领，这是增长经济的老套路。当线条，特别是蓝色的新旧中国线，下降时，增长预期由新政策引领，即重新平衡增长，朝着中国消费者和家庭部门的方向。

根据最后两张图表，我可以得出以下关于市场预期的普遍水平的结论：

+   中国的增长已经放缓了好几年，并且继续放缓；但

+   最新的上升预计将由另一轮由宏观基础设施支出引领的刺激措施所引领，尽管所有官方言论都对此表示反对。

事实上，

[中国国务院总理李克强](http://www.reuters.com/article/2014/05/23/us-china-economy-policy-idUSBREA4M08E20140523)

在上周表示，面对“下行压力”，经济需要进行一些“微调”：

> “目前，经济总体稳定，我们看到了积极的结构性变化，但下行压力仍然很大，我们不能自满，”李克强在访问内蒙古北部地区时说。
> 
> 他说：“我们应该及时、适当地使用适当的政策工具和预防性微调，以帮助解决实体经济的融资压力，特别是小型企业的融资困难和高的借款成本。”
> 
> 这种政策微调应该有助于保持货币供应和银行信贷的“合理增长”，他说。

如果市场预期正确，中国不会崩溃，但那条路上隐藏着失去的十年（参见

[中国会变成日本吗？](http://humblestudentofthemarkets.blogspot.com/2014/05/china-turns-japanese.html)

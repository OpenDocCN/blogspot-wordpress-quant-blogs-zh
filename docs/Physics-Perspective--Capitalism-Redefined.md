# 物理视角：重新定义资本主义

> 来源：[`physicsoffinance.blogspot.com/2014/03/capitalism-redefined.html#0001-01-01`](http://physicsoffinance.blogspot.com/2014/03/capitalism-redefined.html#0001-01-01)

[一个绝佳的必读文章](http://www.democracyjournal.org/31/capitalism-redefined.php?page=all)

由尼克·哈瑙尔（Nick Hanauer）和埃里克·贝因霍克（Eric Beinhocker）撰写。

> 对于除了最富有的 1%的收入者之外的所有人来说，美国经济是破碎的。自上世纪 80 年代以来，普通美国人的生活与我们的繁荣正在增长的统计数据之间存在着日益加剧的脱节。尽管遭遇了大萧条的挫折，但在过去的三十年里，美国经济规模翻了一番，而中产阶级的收入和购买力却停滞不前。许多婴儿潮一代人失去了他们的退休储蓄，而伟大的财富却被创造了出来。企业利润达到了创纪录的高度，而社会流动性却达到了创纪录的低点，落后于其他发达国家。对于太多的家庭来说，美国梦正在变成更多的历史记忆而不是可以实现的现实。
> 
> 这些事实不仅突显了不平等问题和富豪统治力量不断增长的问题，还应该迫使我们提出更深层次的问题，即我们的经济如何运作，关键是我们如何评估和衡量经济进步的概念。
> 
> 怎么可能呢，华尔街用像信用违约掉期这样的产品创造了巨额财富，摧毁了普通美国人的财富，然而我们却把这种活动算作是增长？同样，制造让美国人变得更加肥胖、更加生病和寿命更短的食品产品也创造了财富。然而，我们也把这算作是增长，包括了巨额的医疗费用。全球变暖造成了更频繁的飓风，摧毁了城市和生活。然而，修复损害所需的经济活动最终也被算作是增长。
> 
> 我们的经济政策讨论几乎总是集中在让我们变得更富有以及实现经济增长来实现这一目标上。关于是提高还是降低利率，增加还是减少监管，我们进行了激烈的辩论，而我们的政治体系也因预算的激烈意识形态斗争而陷入瘫痪。但是我们对这一切是为了什么的讨论太少了。几乎没有人会问：我们想要什么样的增长？财富意味着什么？它对我们的生活有什么影响？

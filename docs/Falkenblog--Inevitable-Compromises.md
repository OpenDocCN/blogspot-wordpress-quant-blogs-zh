# Falkenblog：不可避免的妥协

> 来源：[`falkenblog.blogspot.com/2008/06/inevitable-compromises.html#0001-01-01`](http://falkenblog.blogspot.com/2008/06/inevitable-compromises.html#0001-01-01)

华尔街日报注意到纯粹的危险，民主党全国大会上政治正确专家试图计划今年夏天的全国大会：

> 考虑一下臀包。
> 
> 丹佛民主党人希望通过可生物降解的气球和有机小吃来举办“绿色环保最优惠的会议”。
> 
> 民主党全国大会的主办委员会希望为志愿者提供 15,000 个臀包。但它们必须由有机棉制成。由美国的工会劳工生产。

他们有一位“绿色化主任”，他有以下目标：

* 用帆布或基于玉米的生物塑料制成的横幅

* 为会议参与者提供自行车

* 由回收胶合板制成，使用环保油漆的舞台

* 所有垃圾都将分开回收利用

* 气球应该是可生物降解的。

* 不鼓励油炸食品，提倡有机食品。每餐至少 50%是水果/蔬菜。

* 节能灯泡，节水水龙头

* 媒体亭的太阳能

我认为实践你所说的是伟大的。国会就是如此。

[大多数法律豁免](http://www.time.com/time/magazine/article/0,9171,967427,00.html?promoid=googlep)

它也适用于其他，比如平等机会和积极行动的法律，灭火器规定，最低工资，反对游说的法律。如果民主党人想要多付 30％并且受到尝试看哪件物品应该放在哪个垃圾桶的烦扰，那就让他们有更多的权力。

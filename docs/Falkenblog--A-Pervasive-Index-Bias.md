# Falkenblog: 一种普遍的指数偏见

> 来源：[`falkenblog.blogspot.com/2008/11/pervasive-index-bias.html#0001-01-01`](http://falkenblog.blogspot.com/2008/11/pervasive-index-bias.html#0001-01-01)

我在这个博客头部视频中感到有趣的是，这些家伙指出艺术指数存在巨大的生存偏见。它们都有。

大多数资产指数都是由对相关资产有强烈利益关系的团体提供的，而这种情况在资产流动性越低时更加明显。

所有数据提供者都存在明显的偏见，会夸大其呈现的指数的收益。例如，Malkiel 和 Saha（2005 年）发现，没有任何对冲基金数据库提供商在其数据集中包含了长期资本管理公司的最后一年，也就是-92%的回报，因此这些指数排除了让对冲基金投资者感到紧张的风险。Malkiel 和 Saha 估计，这种偏见大约会使年化收益增加约 6%。谁创造了这些指数？通常是与正在研究的产品有关联的团体。例如，监控对冲基金收益的瑞士信贷/Tremont 指数由以下机构维护：

> 这个合资企业，瑞士信贷 Tremont 指数有限责任公司，结合了瑞士信贷这一全球领先的投资银行公司和 Tremont Capital Management, Inc.这一全方位对冲基金投资管理公司的数据研究组的相当专业的知识。

在这个领域，最有知识并且能够获得最好数据的人将成为该领域的倡导者，这是一种不可避免的利益冲突；一个人不会在自认为是无关、低效或欺诈的领域投入大量知识。指数提供者和投资组合经理之间通常有一定的距离，但是他们不可能保持中立：如果市场不兴旺，看起来没有好机会，他们的服务就无法长久存在。毕竟，直到 20 世纪 80 年代，也就是几代人，人们普遍认为主动投资组合经理优于被动指数管理——因为证据通常是由主动经理及其行业团体持有和提供的。这基于这样一种直觉：管理投资组合的人明显比仅仅购买一揽子权益证券要好。所以在使用收盘价格计算总收益之类的问题上，在非常流动的市场中，这是一个良好的假设，在不流动的市场中，这是一个重大错误。

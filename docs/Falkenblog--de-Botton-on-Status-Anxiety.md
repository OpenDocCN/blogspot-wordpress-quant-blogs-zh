# Falkenblog：波顿论地位焦虑

> 来源：[`falkenblog.blogspot.com/2013/09/de-botton-on-status-anxiety.html#0001-01-01`](http://falkenblog.blogspot.com/2013/09/de-botton-on-status-anxiety.html#0001-01-01)

我觉得阿兰·德波顿对哲学的态度相当清新，因为人们感觉到他真诚的不确定性，并且在他的作品中发现了发现的价值。他对日常改善应用美德和寻求意义很感兴趣，这是我生活中非常重要的两个目标。有趣的是他曾在一个

[NYT review](http://www.nytimes.com/2013/08/16/books/the-art-of-sleeping-alone-sophie-fontanels-memoir.html?partner=rss&emc=rss&utm_source=dlvr.it&utm_medium=twitter&_r=0)

索菲·芬塔内尔（Sophie Fontanel）的自我放纵书籍描述了她自愿禁欲的情况，这凸显了波顿广度和深度（波顿的俏皮话基本上是“性是混乱的，适应它吧”）。

无论如何，以下是关于地位焦虑的波顿观点。他认为，地位焦虑比以往任何时候都更糟，因为现在我们相信我们的出生限制较少，对自己的命运负有更多责任。

[保罗·克鲁格曼同意](http://www.slate.com/articles/business/the_dismal_science/1996/12/the_cpi_and_the_rat_race.html)

与这种对生活的看法相符，但像大多数经济学家一样，无法将其推向极致的推论是，这导致零风险溢价，当与各种吸引人的股票相结合时，高风险资产的回报低于平均水平（请参阅我的书

[缺失的风险溢价](http://www.amazon.com/Missing-Risk-Premium-Eric-Falkenstein/dp/1470110970/ref=sr_1_1?s=books&ie=UTF8&qid=1345223074&sr=1-1)

)。

> `.NET 编译平台`（Roslyn）提供了一套编译器 API 和工作区 API，这些 API 能提供关于你的源代码的丰富信息，并且与 C# 和 Visual Basic 语言完全一致。将编译器作为一个平台来使用大大降低了创建代码相关工具和应用程序的门槛。它为元编程、代码生成和转换、C# 和 VB 语言的交互式使用以及将 C# 和 VB 嵌入到特定领域的语言中创造了许多创新机会。

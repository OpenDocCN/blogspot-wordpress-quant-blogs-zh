The

**趋势模型**

是一种资产配置模型，根据全球股票和大宗商品价格的输入应用趋势跟踪原则。该模型的时间跨度较短，通常每年交易大约 4-6 次。实质上，它试图回答一个问题：“全球经济的趋势是扩张（看涨）还是收缩（看跌）？”

我的市场交易员运用

**交易组件**

趋势模型的更换，以寻找主趋势模型信号方向的变化。一个变得不那么看涨的看涨趋势模型信号是一个交易 "卖出" 信号。相反，一个变得不那么看跌的看跌趋势模型信号是一个交易 "买入" 信号。交易模型的实际外样本（非回测）信号历史如下图中的箭头所示。从历史交易模型交易来看，其周转率通常约为每月 200%。

**这么多问题，答案却很少**

常读者都知道，我一直在预测 2017 年周期性市场顶部（参见

[2017 年市场顶部路线图](https://humblestudentofthemarkets.com/2016/08/27/the-roadmap-to-a-2017-market-top/)

). 除了战争或动乱带来的永久损失风险外，严重的熊市主要与经济衰退相关。我的分析基于美国经济的可能路径，以及下一次经济衰退的时间。

我的基础情况，我们将其称为“漏顶”场景，大致如下：

+   经济处于扩张的后期阶段，开始过热。

+   投资者和交易员对增长充满热情，将股票价格提高到不切实际的水平（因此“漏顶”）

+   美联储的回应是加息以降低经济热度，但发现已经跟不上趋势…

+   导致了经济衰退和股权熊市。

如果发生“气泡顶”情景，则 SPX 指数在今年将达到 2500-2600 水平，然后触顶。

我一直在考虑一个替代方案，将其称为“软弱顶”情景，市场可能已经触顶。

+   经济处于扩张的后期阶段，开始过热。

+   投资者和交易商对特朗普税改计划下的减税和放松管制的前景感到振奋，这也是目前的情况。

+   税改计划在国会陷入僵局，一直到 2018 年才得以延迟。

+   特朗普任命鹰派人士进入美联储理事会（目前有三个空缺，丹尼尔·塔鲁洛辞职）。

+   特朗普所任命的美联储理事会成员由坚硬货币倡导者组成，变得更加激进，收紧货币政策，并将经济推向衰退。

+   产生了股权熊市。

“软弱顶”情景基于财政政策不如预期以及美联储政策向更加鹰派的转变。在这种情况下，当前股指价格水平大致达到最好的水平。

这两种情况之间的关键区别在于财政和货币政策的可能路径。这些是我们无法回答的重大问题。本周，我将解释这些风险，并提供一些建议，指导如何观察哪种情况更有可能发生。

# Falkenblog：Plausible Theories

> 来源：[`falkenblog.blogspot.com/2009/01/plausible-theories.html#0001-01-01`](http://falkenblog.blogspot.com/2009/01/plausible-theories.html#0001-01-01)

经常听到一些理论似乎很好，但进一步研究表明它们并不行。有很多反馈循环是很难看到的，所以很多时候理论实际上只是偏导数，而现实中只有全导数。

例如，小型哺乳动物经常玩耍，我无数次听到自然节目的讲述者指出这种戏打斗是小熊/小狮子等长大后的战斗练习。但实际上，科学家们对此进行了

[测试](http://www.sciencedirect.com/science?_ob=ArticleURL&_udi=B6W9W-4FD9T0V-2&_user=10&_rdoc=1&_fmt=&_orig=search&_sort=d&view=c&_acct=C000050221&_version=1&_urlVersion=0&_userid=10&md5=bd19c7048989995ea7a185e543d7af1a)

通过记录松鼠猴和猫鼬的戏打斗频率，记载了成年期的战斗成功率，得出一个结论：幼年期的戏打斗次数和幼年期的战斗成功率与成年期的战斗技能之间没有相关性。他们还发现，戏打斗并没有涉及到真实战斗中的关键战术。幼崽会通过战斗学习，但主要学到如何协调各项活动、如何应对意外和如何社交。因此，从足够高的抽象层次上看，这个理论似乎是合理的，但仔细观察发现，它其实是错误的。动物的戏打斗并不是成年战斗的练习，除非在某种非常抽象的意义上。

同样，政府的乘数效应、进口配额、产业联合以及各种其他伟大的自上而下的想法也会以某种非常模糊的抽象方式起作用。但纳入了所有的机会成本、激励效应、将成本分摊给众多人来帮助少数人的成本后，它们都不如束手无为。不幸的是，这些理论与其他目标（即：再分配）非常一致，因此并没有好好检查。

# 市场学生：观察反弹

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/11/watching-for-bounce.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/11/watching-for-bounce.html#0001-01-01)

在我上周一早上发表的最后一篇文章中（参见

[波动的市场即将到来](http://humblestudentofthemarkets.blogspot.com/2012/11/choppy-markets-ahead.html)

），我写道市场可能会在本周反弹：

> *可能急于看空还为时过早。市场超卖，本周随着期权到期周通常具有积极倾向以及 SPX 测试其 200 日移动平均线支撑，市场正准备反弹。

我显然是错误的。然而，随着股票市场继续贬值，它们变得更加超卖。AAII 看涨情绪（通过

[Bespoke](http://www.bespokeinvest.com/thinkbig/2012/11/15/bullish-sentiment-drops-most-since-may.html)

）正在迅速下降。

**一个好奇的看涨背离**

尽管空头控制着市场，但我想重申在这些水平上做空的风险。一个有趣的看涨背离正在货币市场中出现。尽管股票继续它们的看跌模式，即在早上保持强势并在其余时间逐渐减弱，但风险偏好的货币，如欧元（显示为红色）和加拿大元（蓝色）却在显示出强势。

同样，商品价格，这是另一种风险偏好的资产，已经开始稳定。

尽管目前看来空头似乎控制着市场，但我还想重申我对当前水平做空的谨慎态度。如果你想要看空，至少要等待反弹后进入你的位置。

# 可量化边缘：就业报告会导致大幅波动吗？

> 来源：[`quantifiableedges.blogspot.com/2008/04/will-employment-report-cause-large.html#0001-01-01`](http://quantifiableedges.blogspot.com/2008/04/will-employment-report-cause-large.html#0001-01-01)

与最近的波动相比，过去两天一直非常温和。有一种说法是，交易员在周五的就业报告发布之前不想下大注。期望似乎是就业报告将导致周五的大幅波动。

为了测试这一点，我将 2 天的平均真实范围与 20 天的平均真实范围进行了比较。截至周四 SPY 收盘时，该比率约为 0.53。我运行了一个测试，看看当这个比率在报告发布前跌至 0.55 或更低时会发生什么。基本预期是，在消息发布后波动性的缩减将会逆转，并导致波动性的爆炸。

这并没有成立。下面的表格显示了结果。

第三列显示了发布当天的真实范围与昨天的 20 天平均真实范围之比。9 个实例的平均值为 0.94——意味着报告后的真实范围甚至没有达到平均大小（1）。在最后一列中，我展示了 SPY 第二天开盘时的百分比缺口。

你可能听了很多关于周五早上数字重要性的炒作。如果结果只是炒作，不要感到太惊讶。历史上在这样的大幅缩减之后，并没有导致你可能会预期的波动性爆炸。

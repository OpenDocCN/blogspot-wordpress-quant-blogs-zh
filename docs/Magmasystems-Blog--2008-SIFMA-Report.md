SIFMA

晚上结束时，在 Coral8 派对进行的同时，新泽西州遭遇了严重的风暴，风速达到 75 英里/小时。大量的树木倒伏和风暴损害，新泽西州交通局取消了所有列车。我试图找到回家的路，度过了一段有趣的时光。下次，我不会犹豫。

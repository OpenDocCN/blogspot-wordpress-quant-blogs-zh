# Falkenblog: 无民主的精英管理制度

> 来源：[`falkenblog.blogspot.com/2011/04/meritocracy-undemocratic.html#0001-01-01`](http://falkenblog.blogspot.com/2011/04/meritocracy-undemocratic.html#0001-01-01)

《无人问津》似乎也获得了同样的

[接受](http://reason.com/blog/2011/04/11/some-notable-atlas-shrugged-mo)

1492 年哥伦布发现美洲 500 周年之际正是 1992 年。时机很关键，现在平等主义在思想世界里占据主导地位，而不是个人主义。

[表达清楚](http://bloggingheads.tv/diavlogs/35672)

相信精英管理制度是可怕的，因为它是反民主的。米尔曼似乎认为，只因为优点不能绝对衡量，就应该被忽视。此外，他似乎认为社会目前是由一些星际大法官根据某种优点评判标准选出的精英“运作”的；我倾向于认为许多领导者仅仅是受欢迎的人，他们学会如何在反支配等级中行使职权，并且这些领导者分布得相当广泛——很多层级的政治家，相互竞争的商人。

亚里士多德在他的

[政治学第三篇](http://www.constitution.org/ari/polit_03.htm)

正义在某种意义上是简单的，平等。使正义在政治上变得困难的是，平等的是什么？我更喜欢一个不完美的精英管理制度，而不是一个认为优点是无意义的社会。

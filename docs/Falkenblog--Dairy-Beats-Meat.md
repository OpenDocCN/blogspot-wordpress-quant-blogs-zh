# Falkenblog：奶制品战胜肉类

> 来源：[`falkenblog.blogspot.com/2009/02/dairy-beats-meat.html#0001-01-01`](http://falkenblog.blogspot.com/2009/02/dairy-beats-meat.html#0001-01-01)

Cochran 和 Hapending 的一个有趣的事实

[《10000 年的爆炸》](http://www.amazon.com/gp/product/0465002218?ie=UTF8&tag=bloggingheadstv-20&linkCode=xm2&camp=1789&creativeASIN=0465002218)

，是指养牛产奶，而不是养牛产肉，在一定面积的土地上产生了 5 倍的热量。因此，养牛产奶的人能够繁衍后代，并最终压倒那些文化不养牛产奶的人。有趣的一点是，这是一种奇怪的种族灭绝形式，因为一群乳糖不耐受的人不能简单地采用奶业，因为缺乏乳糖耐受基因，它将是无用的（而且不舒服）。最终，群体争斗，乳糖耐受者的数量简单地压倒了乳糖不耐受者。文化和基因是相互关联的，因为具有乳糖耐受基因的人不能拥有乳制品文化，而没有乳制品文化，就不能选择乳糖耐受。

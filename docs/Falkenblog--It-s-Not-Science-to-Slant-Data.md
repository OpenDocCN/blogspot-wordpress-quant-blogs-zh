# Falkenblog: It's Not Science to Slant Data

> 来源：[`falkenblog.blogspot.com/2009/12/its-not-science-to-slant-data.html#0001-01-01`](http://falkenblog.blogspot.com/2009/12/its-not-science-to-slant-data.html#0001-01-01)

在最新的 Science Saturday 上，George Johnson 试图通过说“每个人都这样做”来贬低 climategate 电子邮件的相关性。就像，那只是更衣室里的科学谈话一样。

没有任何借口试图在统计上隐藏中世纪的暖期，或者贬低批评者的信誉。这并不是简单的谈论，就像白人可以随意使用“N”字一样，但那毫无意义：每个人都知道说话的人在种族问题上没有可信度。

谈论联合，和隐藏显著趋势，都是纯粹的偏斜数据，符合个人偏好。如果有人因为 1970 年代的 beta-return 数据与 CAPM 无关而对其进行了阴影处理，那就是故意的欺诈。幸运的是在经济学领域，我们都在看着相同的数据，所以没有人会尝试这样做，但如果他们这样做了，那将是个大问题，是个严重的污点。他们在那些电子邮件中所说的没有任何好的借口。

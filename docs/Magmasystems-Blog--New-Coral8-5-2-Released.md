Coral8 引擎 5.2 版本现已发布

Coral8 很高兴地宣布 Coral8 引擎 5.2 版本已正式发布。它即可

【下载】

。Coral8 引擎

5.2 版本包含了许多新功能和改进，包括：

+   可查询（公开）窗口

+   在 CCL 模式查询中自动跟踪事件因果关系

+   单次匹配模式

+   “桶”窗口

+   在工作室中编辑和调试透视图

+   PAM 用户认证插件

+   Coral8 门户

+   自定义图表 API

+   用于管理仪表板和用户的简洁 UI

+   Java 1.5 SDK（预览）

+   消息层改进

+   访问控制文件更改

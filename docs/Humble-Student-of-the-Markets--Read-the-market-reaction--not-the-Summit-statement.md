# 谦卑的市场学生：关注市场反应，而不是峰会声明

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/06/read-market-reaction-not-summit.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/06/read-market-reaction-not-summit.html#0001-01-01)

当我们接近又一个至关重要的欧洲峰会时，讨价还价变得更加激烈。 虽然

[默克尔语录](http://www.telegraph.co.uk/finance/financialcrisis/9357592/Angela-Merkel-rules-out-eurobonds-for-as-long-as-I-live.html)

"只要我活着，我就不认为有什么总债务责任"引起了头条新闻，毫无疑问，幕后有很多交易。

[各种提案](http://www.dw.de/dw/article/0,,16051421,00.html)

正在考虑包括“银行联盟、财政联盟和进一步迈向政治联盟”的一揽子方案，并向德国施加强大压力，要求其让步。

预测五花八门。 尽管共识是德国最终将不得不在欧元债券问题上让步，但不一定在本次峰会上，像雷·达里奥（通过

[零对冲](http://www.zerohedge.com/news/ray-dalio-dont-assume-germany-will-bail-europe-out-consider-fat-tail-significant-possibility)

) 说：“别指望了！”

虽然我不指望德国人会改变他们国歌的歌词

***欧罗巴，欧罗巴，高于一切***

，我也不指望本次峰会会承认失败。 最有可能的是，我们将看到典型的欧洲包容，或者宣布“我们有一个秘密计划来制定计划”。

虽然我对峰会声明有些兴趣，但我对市场反应更感兴趣。 如果我是对的，并且我们得到了一种“我们有一个计划来制定计划”的声明，市场会上涨还是下跌？ 根据市场对新闻的反应，我得到了更重要的线索，可以判断牛市还是熊市控制着这个市场。

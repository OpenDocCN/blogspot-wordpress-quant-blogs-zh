# 弗尔肯博客：彼得·席夫，先知？

> 来源：[`falkenblog.blogspot.com/2008/12/peter-schiff-prophet.html#0001-01-01`](http://falkenblog.blogspot.com/2008/12/peter-schiff-prophet.html#0001-01-01)

我并没有读过彼得·席夫的太多作品，但是看看他在 2006 年 8 月就预言熊市的片段吧！这可能是选择性报道，但与其他永久悲观者不同，他提到了具体的抵押市场问题，指出了贷款标准的下降和试用期利率。下次我去巴诺书店一定要买他的书。

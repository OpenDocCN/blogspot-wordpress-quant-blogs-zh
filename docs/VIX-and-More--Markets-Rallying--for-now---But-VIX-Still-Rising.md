# VIX 和更多：市场反弹（目前），但 VIX 仍在上升

看到市场从底部反弹，而 VIX 仍然在攀升，确实不常见，但在过去几分钟内，VIX 已经升至 80.35 点，这种分歧通常以看跌的方式解决。

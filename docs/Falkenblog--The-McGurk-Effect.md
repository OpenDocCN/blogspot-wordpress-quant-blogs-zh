# Falkenblog: The McGurk Effect

> 来源：[`falkenblog.blogspot.com/2010/11/mcgurk-effect.html#0001-01-01`](http://falkenblog.blogspot.com/2010/11/mcgurk-effect.html#0001-01-01)

我正在阅读和观看一些

[乔纳·莱尔的作品](http://www.jonahlehrer.com/articles)

，并发现他是一位聪明，富有魅力的演讲者，类似于马尔科姆·格拉德威尔。他提出的研究趣闻好像

[如下博客](http://falkenblog.blogspot.com/2010/11/redistributions-assumptions.html)

，然后建议这对某些东西有深远的影响。从某种意义上说，这就是《猜想特立独行》的方法：找到由天才教授研究的“新奇”结果（您知道史蒂夫·利维特被广泛认为是最杰出的年轻经济学家吗？），并指出它完全超出了传统范式。

例如：大多数人认为自己是高水平的司机。他们不可能都是。但他们仍然都这样认为（我也是！）。这是非理性的。建立在理性预期上的模型是错误的。这证明了导致抵押危机的布莱克-斯科尔斯模型是错的。 证毕。

我喜欢这些故事，就像其他人一样，但请注意，当它们进入我的领域时，它们变得不再有趣了 100 倍。神经科学，暗物质，动物心理学似乎都不错。当专家或专业知识似乎在您熟悉的主题上无处不在时，您应该持怀疑态度。

例如，这就是麦克格尔克效应，它相当迷人。基本上，如果您看到某人说“fa”但发音为“ba”，您将听到“fa”。您的视觉影响您的听觉。

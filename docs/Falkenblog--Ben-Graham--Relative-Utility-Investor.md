# Falkenblog：本杰明·格雷厄姆：相对效用投资者

> 来源：[`falkenblog.blogspot.com/2012/12/ben-graham-relative-utility-investor.html#0001-01-01`](http://falkenblog.blogspot.com/2012/12/ben-graham-relative-utility-investor.html#0001-01-01)

我的一个主题是

[失落的风险溢价](http://www.amazon.com/Missing-Risk-Premium-Eric-Falkenstein/dp/1470110970/ref=sr_1_1?s=books&ie=UTF8&qid=1345223074&sr=1-1)

是人们人们正在进行基准测试——也就是相对效用投资者——那么他们只有在觉得自己有优势时才会偏离大家都在做的基准。没有其他理由这样做。由于很多人错误地认为自己在高风险（反脆弱？）的股票上有优势，这导致这些证券在债券、股票、期权、马匹和彩票中的回报低于平均水平。

重要的是要认识到这是人们进行集中投注的原因。《华尔街日报》有

[一篇关于一位老同事的文章](http://online.wsj.com/article/SB10001424127887324731304578193323337104806.html)

经典价值投资者

[本杰明·格雷厄姆](http://en.wikipedia.org/wiki/Benjamin_Graham)

，并指出这位 107 岁老人的指导原则：

> 他告诉我，他永远的目标是“对我买入的股票了解比卖出股票的人更多”。

这是一个很好的规则，格雷厄姆经常提到。从整体上来说，这显然是不可能的，但我认为这不仅仅是描述性的，而且是很好的规范性建议。如果你知道这是你正在玩的游戏，那么应该更加谨慎，更加冷静地承担风险。另一种选择是你可以承担巨大的风险，这些风险通过它们的风险溢价产生回报溢价，但不幸的是风险溢价通常是负的，所以这个理论似乎被常规经验所证伪。

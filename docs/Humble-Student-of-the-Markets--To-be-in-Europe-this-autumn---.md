# 谦卑的市场学生：今年秋天在欧洲...

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/09/to-be-europe-this-autumn.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/09/to-be-europe-this-autumn.html#0001-01-01)

金融博客圈的大部分讨论都集中在美联储的行动上。由于本·伯南克宣布美联储可能会无限制地扩大其资产负债表，上周股价飙升。

让我们退后一步，考虑以下两个相对回报图表，它们都是相对于 ACWI，即 MSCI 全球国家指数 ETF。我故意去掉了标签。你更愿意拥有哪一个，这张？

我不会让你悬念。第一张图表是 FEZ，代表欧元 STOXX 50 的 ETF 与 ACWI 的比率，第二张图表是 SPY，代表美国股市的比率与 ACWI。（所有这些都是以美元计价的美国上市 ETF，所以没有汇率影响。）

从技术上讲，过去几周的上涨行情是不寻常的。尽管所有关于美联储行动的喧嚣，但它一直是欧洲股市开始触底并开始成为领导力量。美国股市相对于全球股市开始疲软并开始落后。更不寻常的是，通常被认为是高贝塔值的新兴市场股市表现不佳。

美国投资者不应该只关注他们自己的市场，并在大西洋彼岸寻找新兴的领导力量。当然，像住宅建筑商这样的板块仍然是领导者，如下面的相对图表所示。但这个在美国市场中的权重相对较小，这个行业不可能带动整个美国市场。

当我查看重量级的周期性行业，如工业（XLI）对 ACWI 时，它们似乎已经突破了相对支撑位 - 这不是一个健康市场的迹象。

对于消费品非必需品股票（XLY）对 ACWI 也是如此。XLY 形成了与 SPY 类似的模式，即突破相对上升趋势线。

将所有信息综合起来，这些图表表明，全球投资者对财政悬崖的担忧远超过对欧元区任何金融危机的担忧。考虑到美联储和欧洲央行正在举行一场盛大的派对，似乎人们在欧洲的时光比在美国要愉快得多。

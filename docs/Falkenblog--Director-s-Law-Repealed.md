# 福尔肯博客：董事法被废除

> 来源：[`falkenblog.blogspot.com/2010/08/directors-law-repealed.html#0001-01-01`](http://falkenblog.blogspot.com/2010/08/directors-law-repealed.html#0001-01-01)

一个有趣的问题是，为什么民主国家中底层的 51%不每隔几年就剥夺富人的财产。宪法受到相当有倾向性的解释（

[第十修正案](http://www.usconstitution.net/const.html#Am10)

（隐私权）。在这段视频中，弗里德曼指出，认为政府从富人那里拿钱给穷人的想法是一个普遍的误解。他提到

[董事法](http://www.jstor.org/pss/724835)

（实际上是美国大约 1960 年的一种经验趋势），即富人和穷人支付给中产阶级的项目。

底层的 51%联盟并不有效，因为他们不擅长政治活动：需要有人撰写演讲稿、撰写初始支票，而这些人的经济地位通常高于平均水平。顶层的 51%会是一个好的联盟，但由于顶层的 10%赚了这么多钱，他们很容易被剥夺。因此，弗里德曼论点中最合逻辑的联盟是从中下层到中上层，依赖于非常富有的人和非常贫穷的人。

，尽管弗里德曼的观点可能令人沮丧，但还有比中产阶级的暴政更糟糕的事情：精英与最底层阶级的联盟。我们的立法者给予大众各种闲职、养老金和福利，形成了一个不生产的联盟，他们通过提供更多的福利来增加自己的受欢迎程度和权力。最终结果是破产。

我们已经用自由主义者的法律取代了萨伊定律——供给创造其自身的需求——即需求创造其自身的供给。因此，克鲁格曼轻蔑地

[嘲笑](http://www.jsonline.com/news/opinion/100160239.html)

保罗·瑞安提出的预算案因为冻结了未来的自由裁量支出，仿佛这种适度的财政约束是疯狂的。这怎么能持续下去呢？在私营公司中，如果你亏损，你会削减直到收入与支出相匹配，而在政府中，这通过宏观经济学的诡辩变得不合逻辑。所有好的自由主义者都认为，在失业率高于平均水平的时候，任何支出（总需求的一个组成部分）都是一个好主意。支出什么或如何支出并不重要。债务的产生被认为无关紧要，因为如果市场借给我们钱，那一定意味着这是谨慎的。让政府让路以促进增长的想法被认为是愚蠢和刻薄的。

因此，我们可以增加运输安全管理局的规模，尽管他们整天都在寻找重复犯罪的迹象，而这种犯罪无论如何都不会再次发生，这很像我们的金融监管机构分析金融实践的方式。他们会仔细检查 11 岁男孩口袋里的箭头，就像他们最近对我儿子所做的那样，他被带到一边，因为这一违规行为受到了严肃的审问（他在审讯期间实际上在哭泣）。这些无所作为的人并没有增加我们国家的财富或安全，但他们正在创造永久性的负债和投票工会成员。

如果你阅读罗马历史，你会看到贵族、骑士和平民之间的不断斗争，关于谁被视为公民的问题，以及最终皇帝的神化。从建立共和国的贵族到最终神一样的皇帝统治，得到了被免费粮食和大竞技场活动安抚的大众的支持。这是一种杠铃策略，而且不稳定，总是在扩张。

在上一次扩张中的一个根本性错误是，仅仅因为贷款人愿意向购房者提供资金，就认为这是个好主意。即使在违约之后，你仍然听到有人为那些零首付的购房者辩解，好像他们是受害者（当前的市政债券和国债销售人员请注意）。同样，美国目前正处于和平时期借贷记录的最高水平，而我们问题的解决方案似乎是政府应该花更多的钱。任何一个有财政纪律的人都知道，你不会仅仅因为你买得起就买东西，账单必须支付。

因此，也许每个社会在不同的时间都有不同的最佳联盟，而目前我看不到有什么能阻止它将一切降至少数精英的联盟，这些精英真诚地误以为政府支出的美德，再加上底层庞大的权利群体，所有这些人都实实在在地依赖中产阶级为生。这听起来很像南美洲。

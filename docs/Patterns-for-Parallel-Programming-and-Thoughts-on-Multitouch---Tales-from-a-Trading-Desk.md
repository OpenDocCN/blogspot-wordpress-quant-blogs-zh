## Patterns for Parallel Programming and Thoughts on Multitouch

我还没有下载 P[atterns for Parallel Programming](http://www.microsoft.com/downloads/details.aspx?FamilyID=86b3d32b-ad26-4bb8-a3ae-c1637026c3ee&displaylang=en)：通过 .NET Framework 4 理解和应用并行模式，但怀疑它值得一读[`blogs.msdn.com/pfxteam/archive/2009/10/31/9915569.aspx`](http://blogs.msdn.com/pfxteam/archive/2009/10/31/9915569.aspx)。

DDJ “Multitouch Gains Momentum” 文章提供了一些关于多点触控的思考。文章没有提到 [Silverlight](http://timheuer.com/blog/archive/2009/07/30/silverlight-3-multi-touch-introduction-fundamentals-basics.aspx) 也支持多点触控，这在某种程度上是个遗憾，因为富互联网应用（RIA）在某些情况下是首选的应用程序。如果你现在看看 10/GUI [视频](http://10gui.com/)，你开始可以看到下一代交易桌面的某些有趣的想法。

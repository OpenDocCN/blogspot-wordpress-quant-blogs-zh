# 谦卑的市场学生：多头 2，空头 0

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/12/bulls-2-bears-0.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/12/bulls-2-bears-0.html#0001-01-01)

你可以通过市场对新闻的反应来判断其可能的走势。根据昨天的市场表现来看，多头似乎已经掌握了控制权。

在我上一篇文章中（见

[市场心理的关键测试](http://humblestudentofthemarkets.blogspot.com/2012/12/key-tests-of-market-psychology.html)

)，我提出股市需要进一步显示出强势才能让多头占据上风。具体来说，我在关注：

+   上海综合指数能否突破下降趋势线？

+   欧洲股市将如何对蒙蒂辞职的消息作出反应？

+   美联储在周三将采取什么行动，市场将如何反应？

我们已经知道其中两个问题的答案。上海综合指数在经历了三月的下跌趋势后，成功实现了上涨突破。尽管该指数从这里不一定能直接上涨，但中国股市的技术前景比一周前要乐观得多。给多头记一分。

在欧洲，这一消息

[意大利总理马里奥·蒙蒂提前辞职，前总理西尔维奥·贝卢斯科尼试图重返权力中心](http://www.bloomberg.com/news/2012-12-09/monti-plans-to-resign-as-berlusconi-seeks-return-to-power.html)

市场因一则消息感到恐慌。法国媒体

[解放报](http://journal.liberation.fr/publication/liberation/1108/#!/0_0)

将其描述为“木乃伊归来”：

周一开盘时股市大跌，但随着交易日进行，股市逐渐回升，收盘时接近当日最高点。正如我昨天所写，STOXX 600 已经在技术上突破了阻力位。该指数不仅守住了突破点，而且在当天收盘时还上涨了，这是一个看涨的信号。

周一，意大利 MIB 指数是欧洲股市中表现最差的。尽管如此，它还是回升至接近当日最高点收盘——这是另一个看涨的信号。

这种市场行为表明卖家已经力竭，多头控制了市场走势。再给多头记一分。

到目前为止，多头已经取得了两个胜利（中国和欧洲），而空头没有得分。我将在周三密切关注市场对美联储政策决定的反应。根据我目前所看到的，似乎圣诞老人上涨行情已经启动。

# 谦逊的市场学生：斯姆特-霍利法案 2.0 版？

> 来源：[`humblestudentofthemarkets.blogspot.com/2011/10/smoot-hawley-ver-20.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2011/10/smoot-hawley-ver-20.html#0001-01-01)

[路透社](http://www.reuters.com/article/2011/10/02/usa-china-currency-idUSS1E78T0PQ20111002)

报道美国参议院预计将就一项惩罚中国货币疲软的法案进行投票。

[Patrick Chonavec](http://chovanec.wordpress.com/2011/10/03/economy-on-the-edge-of-a-nervous-breakdown/)

报告说现在人民币存在贬值压力：

> 然而，在最近几天，发生了另外一些可能具有更严重含义的事情。过去十年中，中国在其经常账户和资本账户上均实现了顺差。在贸易和投资方面，中国是外国货币的净进口国，这使得对其货币升值以解决失衡的压力增大。中国通过设定购买美元（和其他外币）的价格并将其作为官方储备囤积，防止人民币升值过快。因为那个价格总是定在市场均衡点之下（人民币被低估），无论设定了多么有限的交易区间，人民币倾向于触及上限。换句话说，人民币是一种单向赌注，总是承受着对美元升值的压力。
> 
> 直到本周，当人民币开始[触及交易区间的下限](http://online.wsj.com/article/SB10001424052970204138204576602292247397816.html)，暗示人民币相对于美元存在贬值愿望。为什么？可能是因为资本账户已经反转，现在投机者正急于将他们的人民币兑换成美元，以将他们的钱从中国撤出。重要的是要澄清这到底意味着什么，以及这并不意味着什么。这并不意味着人民币现在突然会大幅贬值。中国持有 3 万亿美元的货币储备，可以动用这些储备来支持任何它希望的汇率。即使中国受到诱惑要贬值（请注意，这将加剧通货膨胀），美国参议院日益增长的政治压力要求对中国低估的货币采取行动，这将阻碍走这条路。

岩石遭遇坚硬的壁。人民币承受压力是因为对中国硬着陆的担忧。美国政策制定者需要警惕市场力量。否则，他们 risk the path of Smoot-Hawley, which served to exacerbate the downturn during the Great Depression.

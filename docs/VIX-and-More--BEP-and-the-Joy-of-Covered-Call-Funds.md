# VIX 和更多：BEP 与 covered call 基金的乐趣

> 来源：[`vixandmore.blogspot.com/2008/03/bep-and-joy-of-covered-call-funds.html#0001-01-01`](http://vixandmore.blogspot.com/2008/03/bep-and-joy-of-covered-call-funds.html#0001-01-01)

我最喜欢的一个涉及 covered call 策略的基金是

**标普 500 covered call 基金 (BEP)**（http://vixandmore.blogspot.com/search/label/BEP）

，一个

一个**封闭式基金**（http://en.wikipedia.org/wiki/Closed-end_fund）

这是 covered call 基金中交易最活跃的一个。目前，BEP 以净资产价值的 4.9% 的折扣进行交易，它是当前市场捕捉波动溢价的一个极好方式，无需建立自己的 covered call 投资组合。从下面的图表来看，尽管 SPX 年内下跌了大约 11%，但通过对 SPX 编写 covered call，BEP 今年以来的损失仅为大约 1.5%。在下跌和横盘市场中，covered call 基金会几乎总是超越 SPX 的表现，但在牛市中通常具有较小的上涨潜力。

# 谦卑的市场学生：一个引人入胜的配对交易

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/01/an-intriguing-pair-trade.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/01/an-intriguing-pair-trade.html#0001-01-01)

在过去的一周里，我看到了来自不同技术分析师的评论，指出黄金股票似乎已经触底，准备反弹。我进行了观察，并同意这一评估。然而，尽管技术背景积极，我对在这个阶段做多黄金股票感到略微不安，因为这一群体往往与股市略微负相关，而主要股指在非常短的时间内下跌得非常快。

对黄金股票的进一步分析也表明，它们可能已经上涨过快，变得过度买入。那么，让我建议一种风险更可控的方式来参与金属和硬资产反弹主题：买入金属和矿业，做空黄金矿工。

上图展示了这一对（XME vs. GDX）金属和矿业股票相对于黄金股票的情况。除了去年 8 月的短暂时期外，自 2012 年底以来，XME 一直在稳步相对上涨。现在 GDX 已经展开了相对反弹，这种关系已经回到区间底部，这可能是做多 XME/做空 GDX 交易的相对低风险切入点。

不用说，如果您要进行这样的交易，请设置一个紧密的止损，以防黄金股票继续其相对反弹。

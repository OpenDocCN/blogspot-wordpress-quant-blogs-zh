# 谦逊的市场学生：寻找量化α的难以捉摸之处

> 来源：[`humblestudentofthemarkets.blogspot.com/2009/10/elusive-search-for-quant-alpha.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2009/10/elusive-search-for-quant-alpha.html#0001-01-01)

我的帖子[为什么我不是自下而上的股票量化分析师](http://humblestudentofthemarkets.blogspot.com/2009/10/why-i-am-not-bottom-up-equity-quant.html)引发了相当多的反馈，无论是在评论部分还是通过电子邮件。 我与许多人有了一些有趣的交流，尤其是我认为量化α可以找到的地方。

简单概括我在之前帖子中的观点，过去几十年来，自下而上的股票量化分析的准入障碍已经大幅降低。 因此，使用多因子自下而上的量化股票选择技术的竞争优势已经大大削弱。 我们都在使用相同的数据库和工具。 难怪我们最终陷入同样拥挤的交易，自下而上的股票选择α正在变成一场无人能赢的军备竞赛？

**回到第一原则：对人类行为进行建模**

要找到永远难以捉摸的α，重要的是回到第一原则并问：量化分析为什么有效？

除非你能有说服力地回答这个问题，否则你将无法找到持久的α。

量化分析的根源来自于上世纪 70 年代开始的金融学者撰写的异常研究文献。 还记得低市盈率异常吗？ P/B 异常？ 小盘股和忽视效应？ 随后对盈利预期和意外等进行了调查。 投资经理人将其中许多见解以系统化的方式应用于其投资组合中。 因此，量化分析诞生了。

许多量化分析人员从未理解或忘记了为什么购买低市净率股票会带来更好的回报。 通常情况下，具有低市净率的便宜估值股票在基本上有些问题 - 一个“yuck”因素。 购买它们需要投资者忍受从投资组合中闻到“yuck”的气味。 量化分析给了你购买这些股票的纪律性。

那些早期的日子是真实的，现在也是如此。 量化技术的价值在于系统地应用利用人类行为的原则。

许多量化分析人员已经忘记了构建模型中的人类行为建模部分。

**在对人类行为进行建模中仍然存在α**

我可以建议几种建立量化α的方法。 它们都需要努力和对量化分析人员培训和思考方式进行真正的改变。

第一个是极客式的解决方案。

如今，大多数自下而上的多因子模型使用常见因子，如市盈率、市净率、企业价值与 EBITDA 等。 虽然这对从三万英尺高的高度估值股票是一种有用的技术，但为什么不利用计算机的强大功能更接近地面呢？

我们知道行业分析师分析他们公司的方式不同。零售分析师将关注像同店销售（通常每月发布）这样的指标，销售每英尺（销售每英尺的驱动因素是什么？）等。相比之下，能源分析师可能会关注寻找成本、提升成本、精炼利润等。

我们拥有这项技术。为什么不构建专门的行业专家系统来分析行业股票呢？为什么要使用所有行业通用的指标如市盈率或价格对销售额（银行的价格对销售额是什么？），当这些可能与该行业不相关时呢？

专家系统位于人工智能领域，但人工智能研究已经取得了长足的进步，是时候让量化分析师将这种技术应用于投资了。这需要真正的工作吗？是的。这涉及到在技术和开发上的重大投资吗？是的，但您认为竞争优势来自哪里？

将这种方法视为一种使用定量分析的系统纪律来模拟基本投资者行为的方式。

**更加启发式**

另一种方法是在使用量化技术时变得更加经验主义和启发式。我在[前一篇帖子](http://humblestudentofthemarkets.blogspot.com/2009/10/why-i-am-not-bottom-up-equity-quant.html)中概述的逐步转向 top-down 分析的方法就是这种方法的一个例子。

阿夫纳·曼德尔曼还写了一篇[精彩专栏](https://secure.globeadvisor.com/servlet/ArticleNews/story/gam/20091024/RBUYSIDE24ART1907)，讨论了使用启发式技术将定量分析的力量与基本投资者的洞察力相结合。那也是一个很好的解决方案。

各有所好。

**改变公司**

不要误会。这种改变需要真正的工作，以及改变量化分析师的文化和基因倾向。量化分析师将不得不变得更加市场敏感。例如，我曾与金融学者交谈，面试过初级量化分析师候选人，他们甚至不知道如何执行交易，对买卖差价只有一个模糊的概念。

多年前，我面试了一家资产管理公司，该公司管理的资产达数百亿美元。量化分析师被划分为不同的子功能部门。一组负责股票选择的阿尔法，另一组负责行业阿尔法。组合构建是另一个完全不同组的职责，有时这个组还与其他组相隔离。组合实施和交易由另一组完成。明白我的意思吧。像这样的公司往往是由拥有令人印象深刻学术背景的量化分析师组成的。这类公司的核心信仰往往是如果我们能够招募到更聪明的博士，我们就能构建下一代收益惊喜模型（或任何模型），从而获得更好的阿尔法。

那种类型的 compartmentalization 鼓励了一定程度的过度专业化，这在公司中造成了一种功能障碍。人们没有被鼓励去看到整个大局。你当然不需要有市场敏锐度。在这些巨头中爬到顶层的方式是技术上更优秀，玩正确的政治游戏，正如在投资银行爬到顶层的方式是成为一个更好的收入创造者，而无需了解更大的问题。

像那样建立的公司注定要成为恐龙。他们会挖掘越来越低等级的矿石，直到有一天他们醒来意识到矿藏已经全部消失。

量化投资公司若想追求下一代α收益，就必须做出改变。但要想改变，他们必须更加努力且以不同的方式工作。这需要文化上的转变。

以 Jeremy Grantham 为例，说明文化转变。当 Grantham 离开我前东家[Batterymarch](http://www.batterymarch.com/)时，他与 GMO 共同创立了该公司，这两位管理者都以高度量化而知名。两者都已超越了他们纯粹的量化根源。Grantham 最新一季度的信件([Grantham 的最新季度信件](http://www.gmo.com/websitecontent/JGLetter_ALL_3Q09.pdf))涉及了各种主题：

+   **估值仍然过高**（标普 500 的公平价值是 860），**但调整可能会较为温和**。他六个月前写道，“无论基本面如何，都将会有一个急剧的上涨”，因为市场实际上已经过度反应。然而，接近零的利率和其他形式的刺激政策可能会为市场筑底。

+   **他相信一个新兴市场泡沫正在形成**。价值管理者倾向于退出并购买其他更便宜的东西，但不是 Grantham：“在我悲惨的一生中，我愿意参与一次泡沫，哪怕只是泡沫中的一小部分，而不是过早地退出。骑在泡沫上是种有罪的快乐，完全被价值管理者所否认，他们通常会为此付出高昂的代价（投资者纪律之神（Thor?））因为太早地行动。”

这些精选的沉思录听起来并不像是一位单一节奏的量化模型师的教条式断言（我们相信 X，无论 X 是优质公司、低市盈率、市净率等），而是具有[情境意识](http://researchpuzzle.com/blog/2009/10/26/situational-awareness/)。

总之，量化并没有死亡。我[证明](http://humblestudentofthemarkets.blogspot.com/2008/01/are-quants-victims-of-their-own-success.html)了构建不会让你陷入拥挤交易的量化模型是可能的。你必须改变你的思考方式并且更加努力工作。

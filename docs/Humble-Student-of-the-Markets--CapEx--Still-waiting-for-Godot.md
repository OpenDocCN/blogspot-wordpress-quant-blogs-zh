# 谦卑的学生市场：资本支出：仍在等待戈多

> 来源：[`humblestudentofthemarkets.blogspot.com/2014/04/capex-still-waiting-for-godot.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2014/04/capex-still-waiting-for-godot.html#0001-01-01)

在最近的一篇文章中，我写道美国经济正处于周期中期，我们需要看到资本支出加速的迹象，才能使股价从当前水平有意义地上涨（见

[股市多头需要下一阶段的什么](http://humblestudentofthemarkets.blogspot.com/2014/04/what-equity-bull-need-for-next-phase.html)

）。一个线索就是来自像 GE 这样的资本货物集团公司的财报。

GE 在周四报告，而另一家资本货物集团 Honeywell 在同一天报告。我对这两份报告的反应最好的描述是，

"还行吧！" **GE：持平的收入增长** GE 的盈利报告对股东来说是一份扎实的报告，但我并不是从股东的角度来看它，而是用它来寻找全球，或至少是美国，资本支出加速的线索。这份报告可以在[这里](http://www.ge.com/sites/default/files/ge_webcast_presentation_04172014_0.pdf)找到，它让我觉得有些欠缺。特别是，展示订单积压进展的这页有些令人失望。从第四季度到第一季度的积压增长是持平的，从 2360 亿美元增长到 2370 亿美元。石油和天然气部门是资本支出增长的主要来源，但这不足为奇。高盛的大卫·科斯汀按行业划分了资本支出，并显示能源部门在几个季度内一直在增加其在标普 500 中的资本支出份额（通过[商业内幕](http://www.businessinsider.com/capex-spending-by-industry-2014-2)）。尽管 GE 的其他部门有一些亮点，但似乎在非能源基础上并没有广泛的资本支出加速。**Honeywell 的销售增长也很小**

Honeywell 的报告可以在

[这里](http://investor.honeywell.com/phoenix.zhtml?c=94774&p=irol-irhome)

，也有些令人失望。公司超过了华尔街的盈利预期，但在销售上失准（记得我们正在寻找销售加速）。有机基础上，销售仅同比增长了 1%。国防和太空部门的环境存在问题，HON 在欧洲通过涡轮增压器的销售实现了良好的总收入加速。

公司预计下一季度有机销售增长（我的关键指标）为 3%，2014 年下半年为 4%，不包括国防和太空业务。尽管他们提高了盈利预期，但销售指导没有变化。

如果 HON 能代表资本支出的前景，3-4%的数字并不算差，但这还不足以满足华尔街对资本支出加速的预期。同样的评论也适用于 GE。更值得一提的是，比尔·麦克布莱德在

[计算风险](http://www.calculatedriskblog.com/2014/04/the-sluggish-recovery-for-us-heavy.html)

最近突显了美国重型卡车销售缓慢复苏的环境特点：

简而言之，等待资本支出复苏仍然可以比作

[等待戈多](http://en.wikipedia.org/wiki/Waiting_for_Godot)

。

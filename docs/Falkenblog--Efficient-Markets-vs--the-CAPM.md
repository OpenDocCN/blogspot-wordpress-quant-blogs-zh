# Falkenblog: 有效市场和 CAPM

> 来源：[`falkenblog.blogspot.com/2009/07/efficient-markets-vs-capm.html#0001-01-01`](http://falkenblog.blogspot.com/2009/07/efficient-markets-vs-capm.html#0001-01-01)

'理性市场'的'神话'表明在实践中提出了一个无人争辩的论点，但流行的讨论和讨好的作者却无休止地讨论。如果你认为理性意味着完美或市场下跌的缺失，那么你错了，所以不要在这个珍贵的思考上浪费太多时间；这就像发现自然和教养影响智商一样。

重要的是要记住，尽管几乎专门是经济学家相信市场某种形式的有效性，大多数非经济学家并不这样认为。回到 20 世纪 70 年代和 80 年代，当指数基金被引入时，大多数领先杂志痛批这个想法，指出指数基金是多么愚蠢，因为沃伦·巴菲特或皮特·林奇这样的人证明了这是次优选择。因此，约翰·博格尔（万得集团首席）必须在自己担任首席执行官的同时努力说服他的董事会接受指数基金！即使今天，指数基金在股权投资者中只占很小一部分（约 15％，具体取决于如何计算）。

为什么**资本资产定价模型（CAPM）及其衍生物被接受程度比有效市场假说更好呢？两者都依赖于理性投资者，尽管它们是独立的理论（互不蕴含）。我认为有效市场假说更容易被戕描，这很不幸，因为作为一个指导，EMH 的含义比 CAPM（或其衍生物 SDF 或 APT 方法）更好地逼近了现实。也就是说，风险的任何度量作为第一近似都不是线性正相关于回报。相反，大多数投资者在交易愈多时会亏钱，并且他们对市场的偏离平均来说没有超额收益。详情请参见我著作的一则小讨论。

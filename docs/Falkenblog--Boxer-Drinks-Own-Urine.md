# Falkenblog：拳击手喝自己的尿

> 来源：[`falkenblog.blogspot.com/2009/09/boxer-drinks-own-urine.html#0001-01-01`](http://falkenblog.blogspot.com/2009/09/boxer-drinks-own-urine.html#0001-01-01)

对于那些没有看到今晚的比赛的人来说，

[UFC 比赛](http://www.sherdog.com/news/news/update-belfort-franklin-on-weight-for-ufc-103-19811)

，看起来很棒，这

[梅威瑟-马尔克斯](http://www.reuters.com/article/entertainmentNews/idUSTRE58017020090901)

比赛很有趣。一部 HBO 特别节目突出了马尔克斯独特的训练仪式；观看并享受。

看到他更喜欢新鲜的（人们可能会认为最好像白葡萄酒一样，在华氏 40 度）。现在，如果你在救生筏上而且脱水了，你应该喝自己的尿几次，之后矿物质和盐的浓度超过了水合作用。如果你喝自己的尿（而且知道你在哪里），细菌感染的可能性是相当小的，而且它是无菌的（不像粪便，里面有来自大肠的大量细菌）。确实，

[喝尿](http://en.wikipedia.org/wiki/Urophagia)

是一种颇受欢迎的顺势疗法补救方法。

但是，“能”并不意味着“应该”。市面上有很多能够替代必需营养并且口感极佳的能量饮料。

顺便说一句，任何说梅威瑟之外没有更伟大的拳手，从艺术才能和运动天赋方面来看的人，显然是没有见过的。

[安德森·席尔瓦](http://www.youtube.com/watch?v=v3M4VacJ4NU)

战斗。

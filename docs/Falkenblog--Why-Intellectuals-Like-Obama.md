# Falkenblog：为什么知识分子喜欢奥巴马

> 来源：[`falkenblog.blogspot.com/2008/10/why-intellectuals-like-obama.html#0001-01-01`](http://falkenblog.blogspot.com/2008/10/why-intellectuals-like-obama.html#0001-01-01)

知识分子认为他们是专家，政治家应该把他们视为高级顾问（听听《新共和国》的作家诺姆·谢伯对奥巴马对专家的信任的赞美）。因此，只要一个政治家看起来像是欣赏政策狂热者的人，他们就会得到更好的新闻报道。任何在市级以上的政治家的一个好策略就是让你与之交谈的知识分子觉得自己是全世界最聪明的人，因为这是专家的最终幻想。

这是一个幼稚的幻想，认为政治家是‘技术专家’，他们从客观的、领先的专家那里得到最好的理念来制定政策（这些人通常看不起信仰有组织的宗教的人）。专家们在政策影响方面的观点常常截然相反，所以‘听’他们说话只是一个微不足道的练习——你可以选择正确的那些。而且，政治家无法像某个知识分子希望的那样执行一个计划，只不过是说，这个特定的知识分子并不像我一样有说服力和正确，大多数知识分子都是这么想的，所以不一致之处并不是什么大问题。

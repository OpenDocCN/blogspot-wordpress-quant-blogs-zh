# 量化边缘：“圣诞前夜的三天”（更新版）

## 目的

在这篇博客中，我将研究市场行为并量化我的发现。利用情感、广度、价格和成交量指标（包括标准指标和自定义指标），我将尝试揭示可以被市场参与者利用的短期优势。我经常会将这些研究中的观点添加进去，有时可能会发布没有可量化研究支持的看法。

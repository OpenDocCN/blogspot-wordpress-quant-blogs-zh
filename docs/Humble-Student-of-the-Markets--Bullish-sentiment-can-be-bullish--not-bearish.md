# 市场的谦逊学生：乐观情绪可能是看涨，而不是看跌

> 来源：[`humblestudentofthemarkets.blogspot.com/2012/02/bullish-sentiment-can-be-bullish-not.html#0001-01-01`](https://humblestudentofthemarkets.blogspot.com/2012/02/bullish-sentiment-can-be-bullish-not.html#0001-01-01)

在上周五，美国股市遭遇了 2012 年以来最糟糕的一天，因为标普 500 下跌了-0.69%。在市场过度买入且情绪调查显示高度乐观的情况下，这是否标志着中期顶部的到来？

我不这么认为。让我们回到关于情绪模型的基本原则上。当投资者过于乐观时，市场之所以会下跌，因为

***再也没有其他人会买了***

仔细阅读最后那句话，尤其是最后一部分。还有其他人会买入吗？

[道格·卡斯](http://www.thestreet.com/story/11397116/1/kass-a-contrarians-dream.html)

这样认为，因为许多投资者在股票上的投资不足：

> 当前的主导投资者类别——个人投资者、对冲基金和养老基金——已经降低了风险，并且相对不太承诺于股票投资。
> 
> 对股票的重新配置（并从债券中退出）代表了一种被低估且潜在巨大的（隐性）需求，这很容易成为推动标普 500 指数在 2012 年创下历史新高的催化剂。

[梅班·法伯](http://www.mebanefaber.com/2012/01/27/sentiment-update-2/)

展示了 2012 年 1 月 27 日 AAII 资产配置的图表，显示个人投资者相对于历史平均水平而言，对股票的配置不足。（此后 AAII 更新了这些数据，个人投资者的配置已达到平均水平。）这些读数表明，股票配置还有很大的上升空间。

上周，

[巴里·Ritholz](http://www.ritholtz.com/blog/2012/02/where-are-the-bears-2/)

与一位技术分析师交谈，她说她不认识任何看跌的人，甚至鲁比尼也变成了看涨者。Ritholz 然后反问，“看跌的人都去哪了？”以下是

[一条关键评论](http://www.ritholtz.com/blog/2012/02/where-are-the-bears-2/#comment-609193)

针对他指出个人投资者在其投资组合中并未过度乐观的帖子，作出了回应[强调部分已添加]：

> 这很有趣。Street.com 的技术分析师海伦·梅斯勒进行了一项非科学调查，我猜测大部分回应可能来自零售人群，关于他们对近期市场有何期待。***不到五分之一的人表示他们为进一步的收益做好了准备，其余的人则认为市场将回调不到 5%，还有一小部分人预期会有更大跌幅。***
> 
> 我发现这很有趣，因为上次我听说“聪明钱”略微看跌，“愚蠢钱”则显著看涨，尽管不是过度乐观。所以情绪要么已经改变，要么我错了，她的回应来自“聪明钱”群体，或者她的调查样本以某种方式被扭曲了。

**一系列“良好的超买状况”**

对于多头来说好消息是，高乐观情绪和市场低权重组合是引发买入狂潮的配方。

不要忘记，这是一场由中央银行流动性推动的上涨。英格兰银行、美联储和欧洲央行（通过长期再融资操作）都致力于量化宽松。最好的类比是 2010 年下半年看到的 QE2 上涨行情。市场在上涨过程中经历了一系列“良好的超买状况”。

那么我对于股票首次遭遇阻力并经历了 2012 年以来“最糟糕”的单日下跌是否过于担忧？

不。现在是逢低买入的时候。

# Quantifiable Edges: 99% Down Volume Days

> 来源：[`quantifiableedges.blogspot.com/2010/06/99-down-volume-days.html#0001-01-01`](http://quantifiableedges.blogspot.com/2010/06/99-down-volume-days.html#0001-01-01)

纽约证券交易所（NYSE）的上涨成交量百分比在周五低于 1%。换句话说，超过 99%的成交量来自下跌的证券。这种幅度的宽度是显著的。根据我的数据库，自 1970 年以来，只有其他 4 天 NYSE 的上涨成交量百分比低于 1%。我列在下面，以及它们第二天的回报率。

交易员可能还想在图表上查看这些日期。你会注意到，它们都标志着中期低点，或者非常接近一个。

## 下载

### Docker

```
docker pull apachecn0/blogspot-wordpress-quant-blogs-zh
docker run -tid -p <port>:80 apachecn0/blogspot-wordpress-quant-blogs-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g blogspot-wordpress-quant-blogs-zh
blogspot-wordpress-quant-blogs-zh <port>
# 访问 http://localhost:{port} 查看文档
```

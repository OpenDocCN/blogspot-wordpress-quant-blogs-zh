# Xi (Rossi) LUO: New R Package SCIO Posted

> 来源：[http://rossiluo.blogspot.com/2012/04/new-r-package-scio-posted.html#0001-01-01](http://rossiluo.blogspot.com/2012/04/new-r-package-scio-posted.html#0001-01-01)

As promised, the new R package for 

[SCIO](http://arxiv.org/abs/1203.3896)

(and

[post](http://rossiluo.blogspot.com/2012/03/statisticians-can-compute-fast.html)

) has been uploaded to CRAN.  It will be publicly available across all platforms in a few days.

This is a version with the minimal set of functions for estimation.  I will add more functionalities in future releases.

I have also updated our R package

[clime](http://cran.r-project.org/web/packages/clime/index.html)

.  Thanks useRs for all the suggestions.  The JASA paper is

[here](http://amstat.tandfonline.com/doi/abs/10.1198/jasa.2011.tm10155)

, and the preprint is

[here](http://arxiv.org/abs/1102.2233)

.  I have also posted the cancer result using this method on this

[post](http://rossiluo.blogspot.com/2012/03/genetic-profiling-for-breast-cancer.html)

.

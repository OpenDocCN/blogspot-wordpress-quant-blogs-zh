## Massively Parallel Processing (MPP) Cross Platform Optimizer

The mashing of databases and hadoop has begun.  [Microsoft](http://gsl.azurewebsites.net/Projects/Polybase.aspx) is offering [PolyBase](http://www.zdnet.com/microsofts-polybase-mashes-up-sql-server-and-hadoop-7000007424/) (SQL Server 2012 Parallel Data [Warehouse](http://www.microsoft.com/en-us/sqlserver/solutions-technologies/data-warehousing/polybase.aspx)), others (Teradata, Rainstor etc) are on the same track. Mike Ferguson [article](http://intelligentbusiness.biz/wordpress/?p=468), “Big Data – Relational Opens Its Mouth – Is It Going To Consume Hadoop?” offers further thoughts on this subject.  Interesting times.

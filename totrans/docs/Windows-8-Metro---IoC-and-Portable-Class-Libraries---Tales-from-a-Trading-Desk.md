## Windows 8 Metro – IoC and Portable Class Libraries

[CodePlex](http://metroioc.codeplex.com/) offers MetroIoc – a native WinRT Inversion of Control container, for use in Metro style apps. What is probably more interesting if that MEF in part of the [Portable](http://msdn.microsoft.com/en-us/library/gg597391(v=vs.110).aspx) Class [Libraries](http://www.alexthissen.nl/blogs/main/archive/2011/07/07/portable-libraries-for-visual-studio-2010-sp1.aspx). MSDN Magaazine has an [article](http://msdn.microsoft.com/en-us/magazine/hh335063.aspx) from 2011, “Portable Class Libraries: A Primer”, which discusses DI frameworks – MEF and Unity. Finally, the BCL blog offers “Announcing Portable Library Tools” which briefly [touches](http://blogs.msdn.com/b/bclteam/archive/2011/01/19/announcing-portable-library-tools-ctp-justin-van-patten.aspx) on MEF.

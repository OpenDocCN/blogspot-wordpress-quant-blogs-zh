# Falkenblog: Renewable Resources Run Out

> 来源：[http://falkenblog.blogspot.com/2010/10/renewable-resources-run-out.html#0001-01-01](http://falkenblog.blogspot.com/2010/10/renewable-resources-run-out.html#0001-01-01)

Matt Ridley, who wrote The Rational Optimist, was

[interviewed by Russ Roberts](http://www.econtalk.org/archives/2010/10/ridley_on_trade.html)

. This was an interesting insight:

> There is not a single example of a nonrenewable resource that has run out. Nobody ran out of stone in the stone age, iron in the iron age, or bronze in the bronze age. That's not why these ages peter out, it's rather because we move on to something else ..whereas renewable resources tend to run out, like whales, passenger pigeons, and white pines.

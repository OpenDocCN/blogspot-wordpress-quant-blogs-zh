## Microsoft and the Actor Pattern: Orleans

Microsoft is [opening](http://www.zdnet.com/microsoft-opens-early-adopter-program-for-its-orleans-cloud-framework-7000018564/) up Orleans for early [adopters](http://blogs.msdn.com/b/david_gristwood/archive/2013/07/25/orleans-windows-azure-and-halo-4.aspx).

> [Orleans](http://research.microsoft.com/pubs/153347/socc125-print.pdf) is a software framework for building reliable, efficient and scalable cloud applications. Its programming model is based on distributed virtual actors, which are isolated units of state and computation that communicate through asynchronous messages. Orleans has two design goals: to greatly simplify the development of cloud applications, and to ensure that those applications are effortlessly scalable and reliable

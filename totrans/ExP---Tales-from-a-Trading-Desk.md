## ExP

The [Experimentation Platform](http://exp-platform.com/default.aspx) (ExP) [enables](http://blogs.zdnet.com/microsoft/?p=1342) product groups at Microsoft and later on will enable developers using Windows Live to innovate using controlled experiments with live users. The platform enables testing new ideas quickly using the best-known scientific method for establishing causality between a feature and its effects: randomized experimental design. The basic methodology in controlled experiments is to expose a percentage of users to a new treatment, measure the effect on metrics of interest, and run statistical tests to determine whether the differences are statistically significant, thus establishing causality

# Timely Portfolio: R creates d3/javascript charts in Ipython Style Notebook

> 来源：[http://timelyportfolio.blogspot.com/2013/05/r-creates-d3javascript-charts-in.html#0001-01-01](http://timelyportfolio.blogspot.com/2013/05/r-creates-d3javascript-charts-in.html#0001-01-01)

I am not sure I have ever done a post like this, but I was so blown away I had to do this post simply to embed this amazing Youtube video from the author of the R packages [rCharts](http://ramnath.github.io/rCharts/) and [slidify](http://slidify.org).  Watch this screencast as he creates d3/raphael charts from R using rCharts integration with [polycharts](http://www.polychartjs.com/), [nvd3](http://nvd3.org), [xCharts](http://tenxer.github.io/xcharts/), and [morris](http://www.oesmith.co.uk/morris.js/).  To top it off, he does it all using shiny to create an Ipython type working environment.
